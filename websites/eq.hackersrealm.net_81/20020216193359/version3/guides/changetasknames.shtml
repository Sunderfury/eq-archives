How to change task bar names
By: Tux Racer
Print This Guide

 

  1. You need a hex editor, for this guide I used Hex Workshop 3.11. But any hex editor should work just fine.
  2. Open the .exe of the program you want to change the title bar of in the hex editor. In this case we will use W32DSM89.exe.
  3. Make sure you know what the title bar is in the program. In this case the title bar is "URSoft W32Dasm Ver 8.9 Program Disassembler/Debugger" If you can't remember the whole thing don't worry, you just need to know some/most of it.
  4. Now do a search for part of or the whole title bar. Like shown below:

 

  1. You might find it in more then one place, but look for the place that contains "URSoft W32Dasm" and the rest of the title bar.

  1. Now you need to go in and change what is there. I suggest changing the name to the name of a real title bar of another program. I am sure that "Suck this Verant" as a title bar would be a bit suspicious. For this example we will use "Solitaire"

  1. But as you can see, "Solitaire" isn't big enough to fill in the whole title bar. All you need to do now is zero it out. Like this:

  1. Now save the program and open it. You should see something like this:

  1. If the program does not open then you messed up. I hope you made a backup, you just try again. But we are not done yet. You need to rename "W32DSM89.exe" to "sol.exe" or the name of the exe that would open that program. Because they can see the name of the exe that is opened as well.
