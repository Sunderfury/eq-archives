Welcome to the Feekle Log project! Feekle Log is a program that was analyze the everquest logs files to give reports. You can download the full working program here, along with the scource code. Users are encouraged to take the scource code and work on it to make new versions. When you send me a new version, I will display it here. The goal of the Feekle Log project is the create the most complete Everquest logging program on the net!
