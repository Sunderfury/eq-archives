ダンシング ザップ ( Dancing zap ) 標的を追跡する複数の小さな電光の球を放ち、
その球に触れた敵を痛めつけます。		 6
ドロー ニア ( Draw Near ) 魔法をかけたパーティ メンバーに敵を引きつけます。
敵が強い場合、引きつける力が通じないことがあります。		 41
イーグル アイ ( Eagle Eye ) パーティ メンバー 1 人の遠隔戦闘スキルをアップさせます。 55
フラッシュ ( Flash ) 敵を取り巻く空中に閃光を発生させ、標的を痛めつけます。 3
フォーカス マインド ( Focused Mind ) パーティ メンバー 1 人の知性をアップさせます。 24
フリーク ( Freak ) 1 体の敵を魔法詠唱者から強制的に追い払います。
マナの消費量は、魔法をかける相手の力によって変わります。		 5
フリーズ ( Freeze ) 敵を氷で包み、その場でフリーズさせます。
マナの消費量は、魔法をかける相手の力によって変わります。		 69
フリジッド アーマー ( Frigid Armor ) 霜の防護シールドが魔法をかけた相手を覆い、防御力をアップさせます。
さらに、攻撃してくる相手にダメージを与え、
ときにはフリーズさせることもあります。		 50
フル リザレクト ( Full Resurrect ) 死人を蘇らせ、ライフも完全に回復させる力が与えられます。
マナの消費量は、魔法をかける相手の力によって変わります。		 40
ハーモニー ( Harmony ) ライフとマナの共鳴パワーを集め、パーティ全員に平等に分け与えます。 80
ヒーリング ハンド ( Healing Hands ) パーティ メンバー 1 人のライフを回復します。 0
ヒーリング ウィンド ( Healing Wind ) 癒しのそよ風がパーティを包み、ライフを回復させます。 9
ホールド クリーチャー ( Hold Creature ) 1 体の敵をその場に縛りつけます。
マナの消費量は、魔法をかける相手の力によって変わります。		 11
アイス ストーム ( Ice Storm ) 空から鋭利な氷片の嵐を呼び、標的の頭上に降り注がせます。 72
アイスブラスト ( Iceblast ) 無数の氷片を、標的に向かって弧を描くように放ちます。 24
アイスフューリー ( Icefury ) 無数の氷片を高速に浴びせて、標的をずたずたにします。 54
アイスシャード ( Iceshard ) 鋭利な氷の破片を、標的に向かって弧を描くように放ちます。 12
インプロージョン ( Implosion ) 破壊力のあるエネルギー波が、標的の周りに放射されます。 82
インスピレーション ( Inspiration ) 魔法をかけた相手の、魔法使用能力がアップします。
知性、自然魔法スキル、戦闘魔法スキルがアップします。		 45
インヴィンシビリティ ( Invincibility ) 魔法をかけた相手が、ダメージをほとんど受けなくなります。 46
アイアン フィスト ( Iron Fist ) パーティ メンバー 1 人の強靭性をアップさせます。 8
キル ( Kill ) 自然の猛威が、一撃必殺の力となって放たれます。 96
キリング フィスト ( Killing Fist ) 純粋な意志から生まれた強大な力で、標的に打撃を与えます。 102
ライフ バランス ( Life Balance ) 癒しの響きを奏で、パーティ全員のライフを均等にします。 66
ライト レイ ( Light Ray ) 純粋な光を集めた強烈な光線を標的に浴びせます。 48
ライトフット ( Lightfoot ) パーティ全員の敏捷性をアップさせます。 30
ライトニング ( Lightning ) 鋭い電撃を放ち、標的を切り裂きます。 20
ライトニング ブラスト ( Lightning Blast ) 束にした稲妻を、標的に放ちます。 36
ライトニング ボルト ( Lightning Bolt ) 標的に、稲妻を連続して放ちます。
魔法詠唱者が移動するか、攻撃されると、中断します。		 84
ライトニング ストーム ( Lightning Storm ) 稲妻を伴う激しい嵐を呼び寄せ、嵐の中にいる敵を痛めつけます。 90
マジック アーマー ( Magic Armor ) 魔法エネルギーのシェル (殻) が魔法をかけた相手を包み、
防御力をアップさせます。		 3
メジャー ヒール ( Major Heal ) パーティ メンバー 1 人のライフを最大回復します。 32
マナ バランス ( Mana Balance ) 魔法の響きを奏で、パーティ全員のマナを均等にします。
パーティ メンバーは、魔法のかかる範囲にいなければなりません。		 46
マナ チャネル ( Mana Channel ) 魔法詠唱者から魔法をかけた相手へ、マナが徐々に移ります。 12
マナ シールド ( Mana Shield ) マナを消費して、魔法詠唱者が受けるダメージを吸収します。 17
メリック ウィンドフォール ( Merik's Windfall ) パーティ全員の能力を大幅にアップさせます。
強靭性、敏捷性、知性アップされます。		 70
マインド スプリング ( Mind Spring ) パーティ全員の知性をアップさせます。 38
マルチ スパーク ( Multi-Spark ) 敵を追跡する複数の電光の球が放たれます。 30
ミスティック エイド ( Mystic Aid ) パーティ全員の接近戦闘攻撃ヒット率を xx% アップさせます。 84
ノーヴァ ストライク ( Nova Strike ) 太陽の爆発力そのものが、標的に放たれます。 70
ナーチャー ( Nurture ) パーティ全員のライフを 1 秒ごとに回復します。 46
オーバーサイト ( Oversight ) 敵の視界をかすませて、遠隔攻撃の正確さをダウンさせます。
マナの消費量は、魔法をかける相手の力によって変わります。		 22

マナの消費量は、魔法をかける相手の力によって変わります。		 66
リジェネレーション ( Regeneration ) 自然のオーラが魔法をかけた相手を包み、ライフの回復を促進します。 18
リザレクト ( Resurrect ) 死人を蘇らせる力が与えられます。
マナの消費量は、魔法をかける相手の力によって変わります。		 4
