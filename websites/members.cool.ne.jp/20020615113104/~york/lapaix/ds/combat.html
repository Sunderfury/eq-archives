戦闘魔法一覧


魔法名 ( 英語名 ) 効果 必要レベル
アシッド クラウド ( Acid Cloud ) 酸性ガスの雲が魔法をかけた領域を取り巻き、
近くの敵を痛めつけます。		 12
アシッド ガス ( Acid Gas ) ひと吹きの酸性ガスが魔法をかけた領域を取り巻き、
その中に捕らわれた敵を痛めつけます。		 1
バトル ヒーリング ( Battle Healing ) パーティ メンバー 1 人にライフを与えます。 10
ブラスター ( Blaster ) 一群の猛烈な火炎の球を呼び出して、敵に浴びせます。 98
ボンバード ( Bombard ) 数個の爆発する火の球を、敵に向かって強く投げつけます。 100
バーン ボディ ( Burn Body ) 敵 1 体の屍を、灼熱の火炎で焼き尽くします。 1
クラスター ボム ( Cluster Bomb ) 数個の魔法の爆弾に点火して、敵に向かって放ちます。 96
コープス トランスミューテーション ( Corpse Transmutation ) 屍からマナを吸収し、魔法詠唱者に与えます。 44
カース ( Curse ) 敵に呪いをかけ、
戦闘能力とダメージからの回復能力をダウンさせます。
マナの消費量は、魔法をかける相手の力によって変わります。		 26
ファイア サイクロン ( Cyclone Of Fire ) 空から巨大な炎のサイクロンを呼び、
その進路にいる敵を焼き尽くします。		 90
デス ブラスト ( Death Blast ) 殺戮の風を呼び、火炎の中の敵を爆破します。
詠唱者をクリックすると、この魔法が有効になります。		 24
ドラゴン ファイア ( Dragon Fire ) 灼熱の火炎の塊を呼び出し、敵を一掃します。 104
エクスプロード ボディ ( Explode Body ) 屍を粉砕して、周りの敵に衝撃ダメージを与えます。 20
エクスプローシブ パウダー ( Explosive Powder ) 1 個の魔法の球に点火して、標的に向かって放ちます。 3
ファイア リング ( Fire Ring ) 激しい炎の環が魔法詠唱者を取り巻き、
環の中にいる敵を燃え上がらせます。		 26
ファイア ウォール ( Fire Wall ) 魔法詠唱者の前に灼熱の火炎の壁を作り出し、
その壁を通るすべての敵を燃え上がらせます。		 32
ファイアボール ( Fireball ) 大きな火炎の球を投げつけて爆発させます。 20
ファイアボール レイン ( Fireball Rain ) 空から爆発する炎の雨を降らせ、その炎で敵を焼き尽くします。 84
ファイアブラスト ( Fireblast ) 数個の爆発する火の球を、標的に向かって発射します。 90
ファイアボム ( Firebomb ) 1 個の爆発する火の球を、敵に向かって強く投げつけます。 70
ファイアレイン ( Firerain ) 空から炎の雨を呼び、その粒に当たった敵を燃え上がらせます。 48
ファイアショット ( Fireshot ) 標的に 1 個の火の球を放ちます。 0
ファイアショット アーマー ( Fireshot Armor ) 防御力がアップします。
また、敵に攻撃されるとファイアショットで反撃します。		 63
ファイアスカル ( Fireskull ) 炎に包まれたどくろを召喚し、標的に放ちます。 16
ファイアスプレー ( Firespray ) 複数のファイアショットが、魔法詠唱者の手から散開します。 4
フレイム ( Flame ) 近くにいる敵を、炎で燃え上がらせます。 8
フレイム ブレード ( Flame Blades ) 炎の剣を呼び出して、標的に向かって放ちます。 52
フレイム シールド ( Flame Shield ) 火炎の防護シールドを魔法詠唱者の周りに作り出します。
防御力がアップし、さらに攻撃者を燃え上がらせます。		 32
インシネレート ( Incinerate ) 灼熱の火炎が敵を包みます。 88
インフェルノ ( Inferno ) 魔法詠唱者の前で、猛烈な火炎が散開します。 54
リーチ ライフ ( Leech Life ) 敵のライフを吸い取り、魔法詠唱者に与えます。 2
マライン ( Malign ) 敵の防御力を低下させ、衰弱させます。
マナの消費量は、魔法をかける相手の力によって変わります。		 80
マナ チャント ( Mana Chant ) ライフを消費してマナを回復させます。 9
メタル シャード ( Metal Shards ) 無数の鋭利な金属片を、標的に向かって弧を描くように放ちます。 36
メテオ ( Meteor ) 空から 1 個のメテオを呼び、標的に叩きつけます。 66
メテオ ストーム ( Meteor Storm ) 空から複数のメテオを呼び、魔法をかけた領域に叩きつけます。 102
オージャー マイト ( Ogre's Might ) 魔法をかけた相手の物理パワーを増幅し、強靭性をアップさせます。 34
ペイン リフラックス ( Pain Reflux ) 受けたダメージのうち xx% が攻撃者に反射されます。 38
ペスト ( Pestilence ) 魔法をかけた領域に細菌ガスの雲を呼び寄せ、
その中に捕らわれた敵を痛めつけます。		 42
ペスト クラウド ( Pestilence cloud ) 魔法をかけた領域に巨大な細菌ガスの雲を呼び寄せ、
その中に捕らわれた敵を痛めつけます。		 78
ファイア ピラー ( Pillar Of Fire ) 1 個の火柱を作り出して地上を移動させ、
その進路にいる敵を燃え上がらせます。		 72
レイジ ( Rage ) ライフを消費して、接近戦闘最小攻撃力、
遠隔戦闘最大攻撃力をアップさせます。		 3
リコンスティテューション ( Reconstitution ) パーティ全体に、1 秒ごとにライフを与えます。 24
リップ マナ ( Rip Mana ) 敵のマナを奪い取り、魔法詠唱者に与えます。 0
シャーグ リバイブ ( Sharg's Revive ) パーティ メンバーを 1 人蘇らせ、少しのライフを与えます。
マナの消費量は、魔法をかける相手の力によって変わります。		 45
ソウル ランス ( Soul Lance ) 強力な魔法の槍を召喚し、標的に向けて放ちます。 7
スピリット アーマー ( Spirit Armor ) 変化する光のオーラが魔法をかけた相手を包み、
防御力をアップさせます。		 12
