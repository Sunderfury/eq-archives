--New UI 用 XML ファイル--
新しいUI用のカスタムXMLファイルです。

使い方：
新しく作りたいセット用にフォルダを作成します。
これはEverQuestがインストールされたフォルダ(通常は c:\Program Files\EverQuest)にある
uifiles というフォルダの中に作成します。
(defaultというフォルダは消したり変更したりしてはいけません。)
(ゲーム中で、このフォルダの名前を 指定する必要があるので半角英数で)
このフォルダの中にダウンロードしたLZHを解凍して出てきたxmlファイルをコピーします。
EQを立ち上げ、/loadskin フォルダ名 と打ち込みます。
( custom という名前にしていたら /loadskin custom )
( このとき、ウィンドウ配置を変えずにロードしたい時は /loadskin custom 1 とします )
エラーが無ければ切り替わりますが、エラーがあった場合は default が読み込まれます。

注意点：
パッチ当たった時は注意深くファイル名を見てください。
もしカスタムUI用に入れたXMLファイルがあった場合は、カスタムUIのXMLファイルは
削除するほうが良いです。
そして、わたしに連絡してください :p
