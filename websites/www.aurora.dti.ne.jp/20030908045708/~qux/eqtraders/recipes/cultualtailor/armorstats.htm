Velium LeatherTailoring

SoV 裁縫は、全て普通の Sewing Kit ではなく、Coldain Tunner's Kit を使います。
これは Thurgadin の bank内に並ぶ裁縫ベンダーから入手可能です。
Coldain Velium Temper は、Thurgadin のNimren Stonecutter (41, -111) が売っています。
Small brick of Velium は、 SoVのGolemタイプ(icewellやSkyshrine等)や、Crystal CavernのOrcがよく落とします。

それぞれに必要な材料
Velium Boning ... small brick of velium + file + coldain velium temper
Velium Studs .. velium bits x 3 + file + coldain velium temper (２個できます)
Yew Leaf ... WakeningLand で Forage or 地面に落ちています。
Cobalt Cod ... Cobalt Scar で釣り
Drake Egg ... Skyshrine で Forage or SkyshrineのNamed Drakeからrare drop
