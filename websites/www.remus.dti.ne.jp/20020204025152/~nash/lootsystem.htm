Loot System

Lootは DKPに改良をしたものを採用する。

1. DKP(Dragon Kill Point)とは。

Playerは、Dragon/God/その他Big TargetやHard ZoneのRaidに参加することで、
DKPを取得します。
PlayerがアイテムをGetすると、アイテムの価値に等しいDKPを失います。
DKPの値によってLotto Inできる条件が異なるため、DKPを多く持つPlayerは、
アイテムをGetするChanceが増えます。

2. DKP対象

FKでの公式RaidでのMobは全てDKP対象です。
現在は、VtS, Lord-Doljon, DtV, Wuoshi, Vaniki が対象になります。
今後は、Rok Dragonなど、さまざまなMobが追加される予定です。

DKP pointは DKPのページを参照ください。
ここに記載されていないRaidに関しては、Raid後、参加DKPを決定することがあります。
Itemの取得DKPも同様です。

途中参加/途中退場時のRaid参加DKPは以下のルールとします。
- Target Charge前に合流 : DKPあり(Full Point)
- Target Charge中、Kill後に合流 : DKPなし
- 途中退場 : DKPあり(Full Point)

3. Lotto Systemの詳細

Lotto INには、DKPによる優先権があります。

1st Tiler : Itemに必要なDKPを持っていて、1stキャラで使用。
2nd Tiler : Itemに必要なDKPの50%があり、1stキャラで使用。
3rd Tiler : Itemに必要なDKPが50%未満で、1stキャラで使用。

まず1,2,3 Tilerは参加キャラでの使用を前提としています。

1,2,3 Tilerまでで希望者がいなかった場合には、

・ND品の場合、2ndキャラでの使用でLotto INできます。
この場合も、Getしたら1stキャラのDKPを消費します。

・CD品の場合、その場はbanker預かりとします。
そしてRaid終了後、Lotto BBSでAuctionを行います。

このLotto BBSによるAuctionにも、以下の優先順位があります。

4th Tiler : 1stキャラで使用。DKPでの購入。
5th Tiler : 2ndキャラで使用。DKPでの購入。
6th Tiler : 1st/2ndキャラ問わず、EQ moneyでの購入。

最も重要なことは、全体の戦力UPを考えて、Lotto INすることです。
譲り合う気持ちを大切にしましょう。

3. Loot System補足

・Item GetによりDKPがマイナスになることがあります。
マイナスになると、1st Tiler, 2nd TilerにはLotto INできなくなりますが、
3rd Tiler以降は Lotto INできます。

・DKPを消費しGetしたItemは、Lotto BBSで売却可能です。
その際のLotto INは、上記 4th/5th/6th Tilerに相当します。
4th/5th Tilerで購入先が見つかった場合、売却したキャラのDKPは、時価のDKPが還元され、
購入したキャラのDKPは消費となります。
6th Tilerで購入先が見つかった場合、売却したキャラのDKPは、時価の1/2のDKPが還元されます。

・FKに入会した時点のDKP初期値は0です。

・Raid参加DKPは、そのRaidの中で使用可能です。

・DKPは個人管理です。DKPのページで各自でご自分のDKPを必ず更新してください。
DKPが更新していない場合に、Lotto INできなくなる場合があります。
またDKP担当により定期的にDKPの更新がチェックされます。

4. お土産itemについて

・移動中にTarget以外から出るItemは、お土産itemとしてlotto BBSでlottoします。
その際には、DKP消費はありません。
但し、1st(参加キャラ)での使用を優先とします。
2nd使用でのlotto時には、その旨を宣言してください。(1stキャラが誰も振らなければ取得可能)

・Sov Armor Partsに関しては、1st(参加キャラ)使用、必要Factionありをlotto条件とします。
またGemありを優先とします。(lotto時に「Gemあり」と宣言お願いします)

・Keep Itemは、Raid終了時に bankerへお渡しください。
但し、ND品や、Lore品、またbankerが持ちきれない場合は、そのままKeepし、Raid BBSへ
Keep Itemの報告をお願いします。(3日以内)

・Faction UP用のQuest Item(GiantのHelm/Toesや、VLのPaw)もlotto対象とします。

・Keep Itemの報告期間後、lotto BBSでのお土産lottoが開始されます。
lotto期間は、7日間 とします。

・お土産で取得したitemが、後に不要になった場合は、できるだけ、 Guildに返還し、
有効活用できるようにお願いします。(CD品の場合)
