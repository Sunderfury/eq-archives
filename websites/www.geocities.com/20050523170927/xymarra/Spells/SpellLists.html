I am no longer playing EverQuest, so these lists will not be updated in the future. Lucy's database does not appear to have the new spell level information stored in the same way, so my existing program does not generate lists correctly. I will leave the old lists up for reference, since most of the locations are probably still correct.

-Xymarra

These spell lists are generated from the spells_en.txt file in my EverQuest game directory using the Lucy spell parser and database. (I no longer use the EQ Caster spell data file parser, because it appears to have been abandoned.) The spell locations database was mostly derived by visiting all spell vendors on Norrath and Luclin and parsing the log files with the EQ Log parser program. The remaining information was gleaned from Casters' Realm, Graffe's Wizard Compilation, and The Mage Compendium.

Because these data files are generated by programs and not by hand, there are some errors that I am aware of, but am not able or willing to repair. In particular, when two spells have the same name, they share an entry in the locations database. Also, there are spells that appear in the spell data file that are not actually implemented in game. (It seems that AA abilities no longer appear as level 1 spells.) With that in mind, if you come across an error in the locations listed for any spell, please let me know.

For the sake of brevity, not every location is listed with every spell. If a spell is generally available at the guild halls for a class, other sources of the spell are not listed. Other classes that can research a spell are only listed if the spell is not for sale anywhere and cannot be researched by that class. Shadowhaven is particularly annoying, since some classes have most of their spells there, some have a seemingly random subset, and some have only the new Luclin specific spells; thus, Shadowhaven is only listed explicitly for spells that are not generally available in guild halls. Many low level Cleric, Druid and Shaman spells and most Beastlord spells are found in Shadowhaven.

Durations Key

Every spell has two descriptions of the duration, one fixed and one variable by level. The actual duration of the spell is the minimum of these two values, unless the fixed duration is zero, in which case the duration is the variable duration. Durations are listed first in ticks (a tick is six seconds) then in seconds, minutes:seconds, or hours:minutes:seconds. Many spells vary in duration by one or two ticks.
