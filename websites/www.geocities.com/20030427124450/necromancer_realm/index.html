Thursday January 9th, 2003

New Patch Message

We picked up the newest patch message, some interesting changes going in as of this patch, these include Everquest in a window(unfortunatly you can only run it one at a time, no multi-window EQ). The new Temporary buff window has been implemented as well. The Broken Golem in Fear has been fixed, as well as a few other random bugs. The original and Velious interfaces have now been disabled for good as well. For the full patch message, click here
