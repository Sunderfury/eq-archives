  当然ながら、熱で出来ません(w

・ネタが無いのも何なんで、INT/WIS Capが外れた、CasterのMANA考察。

  Manaの計算式は、一般的に
  
  {(int/5) + 5} x Level + ManaBoost = Total Mana 
  
  と、言われていて、それは、SoL導入後でも変わらないとすると…

  SoL導入以前は、INT200で頭打ち(INT200以降は、INT1に付きMana+1)だったため、
  INT200と、INT201の場合、
  
  [SoL 以前]
  ※Lv60の場合…
  INT 200 = {(200/5) + 5 } x 60     = 2700 Mana 
  INT 201 = {(200/5) + 5 } x 60 + 1 = 2701 Mana 

  SoL以降は、INT/WIS Capが外れたので、

  [SoL 以降]
  ※Lv60の場合…
  INT 200 = {(200/5) + 5 } x 60 = 2700 Mana 
  INT 201 = {(201/5) + 5 } x 60 = 2712 Mana 

  まぁ、ウダウダ書いてみたけど、要するに、Lv60の場合、

  INT+1あたり、Mana+12

  前者は、まぁ、忠実なので良いとしても、後者のほうの、うゎ、あんだけ虐待されてんよ。
  って描写がやはり映像的にまずいのか、全然無かったのが悔やまれる。

  俺的には、魔法使いがどーたらこーたらってのもあるけど、それ以前に児童虐待本だと思ってるので（ｗ

・あと、『ハリー。人の名前はちゃんと呼びなさい。』って言われるシーンが
  ダンブルドアの懐の深さを表している感じがして、一番好きなんだが、
  そこも無かったのがちょっと萎え。

・全体的に80点というところかな。

・久々に夜まで遊んだ感じ。
  車があると遊びの幅が広がると、再認識。
