Specifications For the Server

Yeah, this is the boring stuff, except for us tech heads. ;-)

The server is a hand built system that I put together.

These are the updated specs for the server as of 7/26/2001.

  • Motherboard - ABIT KT7-RAID

  • CPU - AMD Athlon 1.3 GHz

  • RAM - 512 Meg PC 133

  • Storage -
    2 - 40 Gig Western Digital 7200 RPM drives. 
    The Primary drive is being Mirrored to the secondary for data backup.
    1 - BTC 52x CD-ROM Drive

  • Video - Diamond A55 8 Meg Card w/ SiS chipset

  • Sound - Creative Labs Soundblaster 16, full length card 
    (YES, it's about 7 years old and it still works GREAT! LOL)

  • Ethernet Cards -
    1 - 3Com 3c905b-TX 10/100 
    (This card hosts the connection out to the Internet)
    1 - Asante Fast Ethernet 10/100 
    (This card is for my internal network)

  • Modem - US Robotics Sportster 56K 
    (Well, I don't use it anymore, but it's there, just in case)

  • Operating System - Windows 2000 Advanced Server

  • Internet Link - DSL connection
    DSL Provided by Qwest with ISP service via Sundog Communications

  • DNS Service - I am hosting our DNS now on this machine

  • Email Service - Using Exchange 5.5 and hosting our own email
