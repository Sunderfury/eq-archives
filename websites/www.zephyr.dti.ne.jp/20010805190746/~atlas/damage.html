ダメージレートの算出
数ある武器の中でどれを選択するか、最重要な要素はやはりDMG/DLYであると思っている方がほとんどではないでしょうか。DMGとDLYの比の高いものを第一として要求していると思います。
それならば、片手系武器として限定してみるとWurmslayer(DMG25/DLY40)が最強の武器のひとつとなったはずです。しかし、事実はそうではありません。Wurmslayer発見当時は、そのDMGとDLYの比率の大きさから注目されていましたが(また、当時はRange装備が可能だった)、実際に使ってみると強いと感じられる武器ではありませんでした。それは何故でしょうか。
実は、実際に与えるダメージとは、武器のDMGやDLYといった性能のほかに、いくつかの要素が絡み合うことで実現されているのです。全体的なダメージのレートを考慮すべきなのです。
そして、このダメージレートに影響を与える要素とは以下のものになります。

■武器の性能
■キャラクターのStatusやSkill
■ダメージボーナス(これはPrimary装備にのみ適用されます)

ほかに、対象との相対的なLvl差や、武器そのもののEffectといったものがあります。これらは決して影響の小さい要素ではありませんが（むしろ相対Lvl差が最も影響される）、残念ながら容易に計算できるものでは無いので、ここでは考察外としています。



■武器の性能
当然のことながら、ダメージレートには武器の性能が多分に影響されます。DMGやDLYといった要素もさることながら、数多くの武器にはStatus修正が含まれているため、この要素の占めるウェイトは大きくなります。


■キャラクターのStatusやSkill
Statusは特にSTRに関して、Skillは使用する武器のSkillによって、与ダメージ量に影響します。他の要素に比べ変動は小さくなりますが、決して無視できる要素ではありません。


■ダメージボーナス
これは、攻撃が命中したその度に直接的に加えられる修正値です。
この数値は統計的にも数式にも結果が出ておりますので、ほぼ確定的な要素です。値は、対象者のLvlにより決定されます。Lvlが高いほどダメージボーナスは大きくなります。

このダメージボーナスほど、直接的なダメージ要素はありません。どれほど強力な武器を所持しようとも、武器のみでの予測されるダメージ最低値は1であり、これは確率的なものなのでどうしようもありません。が、実際にはこれにダメージボーナスが加わるので結果、最低値は1では無くなります。Ｄ＆Ｄ等のＴＲＰＧで例えるならば、武器のダメージレートは2Dなどというダイス値になり、そしてダメージボーナスは+3などといった修正値になるでしょうか。
ただ、EverQuestでいうダメージボーナスは先程述べたようにLvlに左右されるので、対象となるキャラのLvlによって武器の見分け方が変わるということになります。

ダメージボーナスの値は、Lvl30から効果が出てくるようです。
Lvl30で+1、Lvl31から+2となり、Lvlが3上昇するごとに１ずつ増えていきます。そしてLvl60(Lvl58も同様)では+11の修正値となります。
つまり、Lvl60のWarlordはどのような武器であっても、Primary装備でのダメージの最低値は12点を下回ることは無いということになります。

＜片手系武器ダメージボーナス計算式＞
DamageBonus = (Lvl-25) / 3 (小数点以下切り捨て)

ちなみに、両手系武器の場合はもう少し複雑な式を表します。DLYの大きさによって、ダメージボーナス値も変動し、DLYが大きいほどボーナス値も大きくなります。

＜両手系武器ダメージボーナス計算式＞
・Fast(DLY27以下) → 基本値 + 1
・Medium(DLY28以上39以下) → 基本値 + (Lvl-27) / 4
・Slow(DLY40以上42以下) → Medium値 + 1
・Very Slow(DLY43以上) → Medium値 + (DLY-34) / 3

この計算式以外は、片手系、両手系の差は見られません。

さて、ダメージボーナスを鑑みるに、片手系武器の場合は、同じダメージレートの武器であればDLY値が小さいほど、時間単位でのダメージボーナスの適用率が大きくなるということで、つまりDLYの小さい武器のほうが直接的なダメージレートの期待値は大きいということになります。
ですから、Primary装備に限定はされますが、武器を選択する際にはダメージレートの他に、DLYの大きさも考慮に入れ、そしてSecondaryには武器そのもののダメージレートだけで選択するべきであるということになります。



ここで例を挙げてみましょう。
本来の攻撃力を求めるためには、まずは見込まれる最大ダメージ値を考えます。
例えば、あるキャラクターがある武器を使用した際のPrimary武器の最大ダメージ値は、以下の式から導き出されます。

最大ダメージ値 = (( WeaponSkill + STR ) / 100 ) x DMG + LevelBonus + 1

この式によって求められた値と最小ダメージ値を足して2で割れば平均値が出ます。これが実際の平均ダメージ値となります。Primary装備の場合は、最小ダメージ値は1にダメージボーナスを足した値となります。

平均ダメージ値が求められたら、次は単位時間でのダメージ値を計算しましょう。
これは、ただ単に平均ダメージ値を武器のDLYで割るだけです。武器のDLY値は1/10sec単位なのに注意して計算します。そして最後にDouble Attackや、Secondary装備であればDual Wieldの発動率を含めて計算にいれれば、ようやくダメージレートが導き出されることになります。

しかし前述したように、ここでは対象との相対level差を考慮に入れていないため、この計算結果を実際の戦闘ログと比較した際には、ある程度の誤差が発生するはずです。また、戦闘ログからのアナライザーなどによる計測結果も、EverQuestの戦闘システム上、厳密にダメージレートを計測することは難しく、計算結果と大きな隔たりが生じることでしょう。
つまり、計算結果と戦闘ログとの比較をする際には、システム上、変動の大きく計測されるダメージレートよりも、ある程度安定した計測結果が出される平均ダメージ値を基にするほうが確実です。



実は、まだ考慮すべき要素があります。それは武器の種類です。
武器には、片手系、両手系といった種別のほかに、Slash系、Blunt系、Pierce系とに分けられます。 当然ながらこれらはすべて、使用する際はそれぞれのSkillを用いることになります。Warriorの場合ではそれぞれのSkillに大きな違いはありませんが唯一、Pierce系のSkillが他のSkillに比べ上限値にして10点劣ります。では一体、この差はどういった影響を与えるのでしょうか。
先ほどの最大ダメージ値計算式を見てみましょう。

最大ダメージ値 = (( WeaponSkill + STR ) / 100 ) x DMG + LevelBonus + 1

この式から、武器のSkillはすなわちSTRと同等に影響を成すということが解ります。つまり武器Skillに10の差があるということは、STR10の差があると同等であるということです。この差から実際のダメージにどういった影響が出てくるかは、武器のもつDMGにも変動しますが目安として、武器のDMGが10程度であれば最大DMGにして1点の違いが発生すると考えてください。平均DMGでは0.5点の差となります。

それでは、Skillに差が無い場合には、種類の違いによっては性能に差が出ないのでしょうか。
実は、武器の種類によって、対象へのTaunt能力といった面で違いが出てきます。Taunt能力はSlash系が最も秀でており、次にBlunt系、Pierce系の順で劣ります。また片手系よりは両手系が上となります。

使い比べてみるとはっきり体感できると思いますが、2HSの武器を使用した場合とPiercingの武器を使用した場合では、2HSのほうが頻繁に対象を後退させます。一度に与えるDMG値の大きさによっても原因はありますが、やはり武器の種類によっても後退させる頻度に違いがあるようなのです。Hateの上昇はダメージ量だけでなくこういった事象も含まれているようで、後退させる頻度が多いほど、Taunt能力も高いということが言えます。最上級の性能を誇る武器の中にSlash系が少なくPierce系武器が多いのは、こういった要因があるのは否定できないでしょう。
