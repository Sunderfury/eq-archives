序章
全てのClassでEpic Questと呼ばれるものが実装されました。
それは、「叙事詩」との名の通り壮大なQuestであり、踏破するのは大変困難でとても一筋縄で行くものではありません。
しかし、最終的に入手することができるItemは相当に強力なものであるので、ある意味、究極のQuestと言えるものです。


WarriorのEpic Questの進行手順は、数点のQuest Itemを揃えたあと、最終的にそれらをまとめてコンバイン、といった形になっています。
Quest Itemとは、コンテナとなっている鞘及びそれぞれ二点ずつの刃と柄の計五点。
刃と柄の２セットをコンバインすることによって、片手武器の二本ができます。そしてさらにその二本を鞘に入れコンバインすると、一本の両手武器ができます。片手武器に戻すには、両手武器を鞘に入れてコンバインします。
つまり、この武器は状況によって、1HSや2HSの使い分けが可能になるのです。

Ancient Blade
Ancient BladeはQuestからではなく、Chardokの奥地Royal AreaにいるQueen Velazul Di'Zokを倒して入手することになります。入手した時点で、自動的にこの部分は終了となります。

Queen Velazul Di'Zok

■場所
Chardok Royal Area最奥
■強さ
Lvl63。2000DMGの超強力なAE DDを使用する(正確には18secのDoT)。Recastは12sec。また、Gateも使用する。Meleeは300程度とそれ程では無く、そしてHPも20000程度と少ない。
■攻略法
短期決戦に臨むしかないようです。MAGのFire ElementalならAE DDを無効化できますが、あてにはできないのでTankはQueenを集中攻撃、CasterはDD連発といった形にするのが良いでしょう。QueenはMagic系に比較的弱いようでDDは入りやすいようです。同行するWIZをできるだけ多く連れていくと良いかもしれません。
Queenと周りの雑魚をどう捌くかということを考えるのでは無く、そのままChargeして雑魚は眠らせる等の戦法なら、比較的勝率は高いのではないかと思います。

Epic Quest Itemの強さ
DMGとDLYだけで見ると、Epic Quest Itemを凌駕する武器は数多くあります。
しかし、Epic Quest Itemは単体としての強さにあらず。

◆ 自在に1HSや2HSに変化させられる為、状況に応じそれぞれの利点を最大限に生かすことが可能になります。性能的に見て、通常は1HSでの使用となるでしょうが、敵がDamage Shieldを張っていた場合等においては2HSでの使用が有効となります。

◆ SV-ALL+10の修正があります。今日では、敵の魔法や特殊攻撃等に対する耐性が重要視されています。

◆ Statusの修正が驚異的です。実は、二本の1HSのうちの一本、Blade of TacticsのEffectは恒久的なもので、装備するだけで効果があります。これを踏まえれば、実際に一対の1HSを装備した場合のStatusは、

14/24 Dual
AC+12 ATK+30 STR+20 STA+15 DEX+55 HP+60 SV-ALL+10
さらにProcとしてFire系100DMGのDD、ATK-40、AC-11の効果

ACの修正もさることながら、このATKの上昇はSTRと同様にMAX DMGに影響しますから、攻撃力にして実質STR+50程度の意味があることになります。また、DEXもかなり上昇している効果なのか、Procが相当出やすいです。
Lvl60にしてHP+150の修正も見逃せません。

◆ 外見が派手です。目立つこと請け合いです(笑)。


事実上、三本の武器を同時に入手となることが、Epic Questの最大のPointです。



さて、ここで通例の、武器比較をしてみましょう。
キャラクターの状態はLvl60、STR200、Skillは上限値ということで仮定します。

まずは、Blade of StrategyおよびBlade of Tacticsを装備した状態での攻撃力を求めてみましょう。


平均DMGを求めます。ここで、ATKの上昇はつまりSTRの上昇と同義であると考えます。
最大DMGは、

(( 200 + 20 + 30 ) + 250 ) / 100 x 14 + 11.7 + 1 = 82

最小DMGは12なので、平均DMGは47になります。
さらに、平均DMGをDLYで割り、ダメージレートを求めます。

47 / 2.4 = 19.6 [dmg/sec]



次はSecondaryです。Secondaryはダメージボーナスが無く、そしてDual Wieldの発動率があります。

(( 200 + 20 + 30 ) + 250 ) / 100 x 14 + 1 = 71

最小DMGは1なので、平均DMGは36になります。これをDLYで割ります。

36 / 2.4 = 15.0 [dmg/sec]

SecondaryにはDual Wieldが影響します。発動率は61％なので、

15.0 x 0.61 = 9.2 [dmg/sec]


ゆえに

18.9 + 9.2 = 28.1 [dmg/sec]

Epic Itemによる二刀流の攻撃力は28.1[dmg/sec]となりました。



さて、比較対象を考えます。
今回はダメージレートで同等、もしくは上回る武器を選択してみました。
Primary: Willsapper(Piercing13/20 AC5 Effect:Energy Sap)
Secondary: Exquisite Velium Warsword(1HS 12/21)

Tantor's TuskやSword of Tunareなどはもはや次元が違うので論外(笑)。


では、まずはPrimaryから。他の条件は同様なので最大DMGは、

( 200 + 240 ) / 100 x 13 + 11.7 + 1 = 69(小数点以下切り捨て)

最小DMGは12なので、平均DMGは40.5。さらにDLYで割って、

40.5 / 2.0 = 20.3 [dmg/sec]

ダメージレートが出ました。


次にSecondaryです。最大DMGは、

( 200 + 250 ) / 100 x 12 + 1 = 55

そして最小DMGは1なので平均DMGは28。DLYで割ります。

28 / 2.1 = 13.3 [dmg/sec]

最後にDual Wield61％を掛け、

13.3 x 0.61 = 8.1 [dmg/sec]

よって、

19.6 + 8.1 = 27.7 [dmg/sec]

この場合のダメージレートは27.7[dmg/sec]です。

ダメージレートでは僅差ですね。
となると、あとはStatusやEffectの差になります。

ここで焦点となるのは、WillsapperのProcの発動しやすさ。Proc:Energy SapはSlowの効果を含む為、Taunt能力はかなり強力です。
全般的に、特殊な効果を持つProcは思うほど発動しないという傾向があります。私は、Procというものは使用者によって確率が変動するだけでなく、そのItemそれぞれにも独立した発動率(いわばEffect毎に発動率が変動する)があると思っています。Energy SapがBlood PointのProc:Siphonと同程度の発動率となると、Willsapperの評価を多少下げなくてはなりません。
また、Slowの効果はTauntの強力さを考えれば、絶大な効果があるということが容易に窺えますが、所詮ProcでありSlowの効果そのものをあてにはできません。WillsapperのProcはいわば、Tauntするためだけにあると言えるでしょう。つまりWillsapperの強さとは、そのダメージレートもさることながら、超強力なTaunt能力をもつこと、と言えます。

それに比べEpic Itemはというと、強化される部分といえば、主要なStatusおよびHPの上昇、そしてSVの上昇といったあたり。
また、Procとして100DMG DDとともにDebuffの効果もありますが、このDebuffは大した効果を感じさせません。Procはダメージソースのみとして考えるのが自然です。とはいえ、このProcの発動のしやすさは相当なものです。戦闘ログに取ってみましたが、だいたい1戦闘に1度の発動率です。極端な計算として、100secに一度のProc発動率であれば、武器のレートに1dmg/secとして加えることと同等になります。しかし実際は、それ以上の発動率を使用者は体感できるでしょう。


つまり、Taunt能力に関して、もしくは直系のダメージソースとしてであれば、Willsapperクラスの武器はEpic Itemを十分に上回る可能性を持ちます。それに対しEpic Itemの強みは、なんと言っても一対の武器になっているという点でしょう。一本だけならともかく、このクラスの武器を二本揃えることが大変困難なのは火を見るよりも明らかです。
また、Epic Itemは、StatusはさておきHPやSVの上昇があるために、総合力といった点では他に抜きんでています。

常時携帯
WarriorのEpic Itemはその特殊性から、Combineに失敗して武器を失ってしまう事態が起きてしまう場合がありました。これに対する救済措置がとられています。
Magicianから召喚武器のSword of Runes一本を受け取り、それを鞘に入れCombineすると、1HS二対のEpic Weaponが作製できます。
これを利用して、1HS二対と2HSを同時に所持することが可能です。方法はとても簡単です。
これら全てはLore Itemであるために、まずは現在所持しているEpic Weaponを2HSにします。そしてMAG召喚武器を鞘に入れてCombineするだけです。これで鞘を常時携帯する必要が無くなり、鞘の代わりに通常のバッグを所持することができるようになります。鞘はBankにでもしまっておきましょう。
