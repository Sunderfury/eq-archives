另外，int與mana有一定的比例，巴哈上eeq版精華區裡有一個公式：
在int或wis在200之前 MANA=1(INT或WIS) X 12 X (等級/60)
在int或wis在200之後 MANA=1(INT或WIS) X 6 X (等級/60)
推算了一下，等級10時，1 int = 2 mana
等級20時，1 int = 4 mana
等級30時，1 int = 6 mana
等級40時，1 int = 8 mana
等級50時，1 int = 10 mana
等級60時，1 int = 12 mana
所以重int或重mana，請自行取捨。
