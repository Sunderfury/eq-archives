[ Guest ] . . . .. :: .: ..: . . ..: [ Step-by-step Setup Guide ] : . . . . : .

Setup of EthernalQuest emulator is easy in most cases. You should read readme.txt that is comes with Emulator and it should be enough. If you got into a problem setting it up, here is more detailed step-by-step setup instructions written by SamWoodchopper on the Emulator forum. You can find the same guide in the forum with a lot of follow up comments by clicking here

If this is your first time, setting up EthernalQuest Emulator, do NOT use EQW for the first install. Follow this step-by-step. Once you have the Emulator running alright and played around a bit, THEN you can try to set it up to run from the normal EQ directory to save space, fiddle around with the database files so others could have access to your server or to get EQW working with it (I have to warn you, that EQW will get you headaches if you're not strong of heart, it might work flawlessly for you, it might not at all)

Read each step carefully before doing it (a lot of people try to "read" and "do" simulaniously and as a result oversee some crucial things), don't try to skip things (not even if you think you already did them sometime before), change things or do things in a hurry (you have all time, no need to hurry and make little mistakes due to that).

  1. Patch EverQuest to the latest patch.
  2. Copy the complete EverQuest Directory (C:\Program Files\Everquest Triology\) into a new directory. Rename that Directory to "EqEmu" (so it's C:\EqEmu)
  3. Download the latest EthernalQuest Emulator from: Sourceforge
  4. Unzip the contents of the EthernalQuest Emulator file while retaining the directories (the last part is important and will give the emulator it's own directory and placeall files in the correct sub-directories) with destination: C:\.

    If you use "Winzip", that specific funtion is default off. You can switch it on by thicking the small box: "Use Folder Names".
  5. Notice the directory C:\EthernalQuest that has been created
  6. Go into the directory C:\EthernalQuest and open the file [b]emuconfig.dat[/b] with notepad.
  7. Change the top 2 lines of the file to read: 
    # This is directory where Everquest is installed
EverquestDir=C:\\EqEmu
    Note the 2 "\\"'s instead of one and their orientation.

  8. Save the file
  9. Select the file eqhost.txt, rightclick on it and select copy
  10. Browse to the C:\EqEmu directory and right click on a white space and select "paste"
  11. Click "Yes" when it asks to overwrite an existsing file
  12. Go back into the C:\EthernalQuest and double click on the "StartEMU.bat" file.

    DON'T CLOSE ANYTHING DOWN NOW, just keep everything that's open, opened

  13. Click "start" on the left downside of your screen on the windows taskbar and select "run"
  14. Type: 
    "C:\EqEmu\eqgame.exe" patchme
    Please note the placement of the "'s and the orientation of the slashes

  15. Loginto the game using the username "Jane" and the Password "Fonda", or whatever you like, just NOT your actual password and username.
  16. Select the one server that is listed
  17. Create a character and play
If this does not make it work for you please read the Pre-face section of http://www.smoothwall.org/download/pdf/docs/0.9.9/doc.faq.pdf to understand what to include to make a good helppost and what you should do before posting a help request (it IS worth your time to read it) then post a message in this forum.
