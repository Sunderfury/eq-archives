New versions of EMU and EQColelctor is out :: Jun 17, 2004
The latest versions are:
  • EMU: 0.7.17
  • EQCollector: 0.3.11
- zordon
More about merchants :: Jun 17, 2004
There were some problems with processing merchant data. All of them have been resolved by now. None of the data you have collected was lost. There are still over 200 merchants without items. Please keep logs coming.
- zordon
Merchants have been updated :: Jun 15, 2004

I finally found some time to write a script to process merchant data. All merchant data that was submitted up to date is in the database and you can browse it on-line.

As you can see here there are still 255 merchants that needed to be visited. Most other merchants still might have invalid items on them. It is up to you to make it better. Use EQCollector. Every merchant you visit will make this database more accurate.
- zordon
More collectors needed. :: Jun 14, 2004
Collecting of the data is very slow right now. Over 100 people got new version but only 3 have contributed to EqDb.

If you are using EQCollector please share data you have collected with other people. There are no personal or your character information in the data that will be uploaded.
- zordon
Foreign Servers (cont) :: Jun 14, 2004
I have removed all data that comes from foreign servers and all mobs, zones and items should get back their English names. If you still see some invalid names when browsing EqDb using this website, let me know. Database that comes with EQCollector and EMU will be updated soon.
- zordon
