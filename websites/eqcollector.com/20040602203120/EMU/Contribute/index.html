[ Guest ] . . . .. :: .: ..: . . ..: [ Helping Emulator Project ] : . . . . : .

You can help emulator in different ways. If you think that you have to be a C++ programer to help - then you are wrong. Here are few ways to help this project to grow faster and better. If you think of any other help, you suggestions are always welcome.

1. Helping us to develop EMU.

So far I am the only developer of Ethernalquest Emulator as well as EQCollector and EqDb projects. This means that it not be developed as the fast rate. If you are experienced in C++, able to at least compile Emulator and willing to put your own time into it, just email me at zordon@ethernalquest.org. I will not be able to put a lot of time into educating people. You should be able to understand (more or less) current EMU code as well as willing to write new code in the same way it is done by now.

2. Testing and discussing emulator features.

On the new website you can find list of all features that emulator have with status on every one. It is far from beeing completed so if you think of any addition to this list or some of the features are not clear let me know. In addition you can submit comments as well as vote on what feature you want to be implemented next.

3. Helping others in forums. Writing documentation.

I would like to thank all people who are now answering questions in the emulator forum. Whithout them forum would be dead by now. If you willing to help, you always welcome to join them. Also I have to admit that emulator is not well documented. Even readme.txt have to be re-written. This is a big area where people with good English and good undestanding of Emulator can be very helpful.

4. Helping to support website.

Emulator website (as well as Hackersquest) need to be supported on more regular basis. If you willing to help, please contact ashran@ethernalquest.org

5. Donating to this project.

And the last but not the least. Support of this project takes a lot of time. Hosting of related websites are not free to us. If you willing to help us finantially, you can donate to this project via PapPal. Just click on the icon below to do so.
