EMU 0.7.08 is out :: Jan 09, 2004
Changes:
  • item should appear on the cursor when you looting them with left click
  • bug fix: damage should be calculated correctly for high level weapons
  • bug fix: items would give you correct stat buffs on server side
  • spells are implemented (many server side effects are still not working)
  • bug fix: fixed crash when NPC mobs were trying to follow each other
  • bug fix: login server should properly report 'DOWN' servers to playes
  • bug fix: login server should correcly show 'OWN' expantions (on HQ login server all players are created with access to all 5 expantions)
  • bug fix: beastlord spells should no longer crash EMU
  • bug fix: server you will not longer be putting a container inside another container while looting a corpse
  • bug fix: merchants should be working again
  • EqDb changes:
    • New class 'Berserker' is added. All other class ID's are fixed
    • New races are added. 'Highpass Citizend' and 'Ware Wolf' mix is fixed
    • EqDb is partually available for browsing online on http://eqcollector.com

EMU 0.7.06 is out :: Nov 15, 2003
  • work with latest version of EQ client
  • ability to setup process priority in configuration file
  • mob will die at 0 hp's (not negative)
  • player will die at -10 hp's (not -11)
  • attack animation will be sent only to nearby players
  • new logic for tracking mob's and player's. As a result:
    • memory usage and zone startup time is significantly down
    • but only ones that are near the player will be send to the client on startup
    • /who command will only show players who are near you - not all players in the zone
  • when player is damaged, it's hp will will not be sent to client
