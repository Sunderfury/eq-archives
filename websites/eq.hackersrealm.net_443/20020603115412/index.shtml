Mozilla 1.0 rc3 released
As most of you know this website is best if viewed with Mozilla. Mozilla 1.0 RC3 has just been released. You can check out the release notes here. According to the release notes there have been “139 bug fixes since RC2, including fixes for plugins, usability, crashes, and internationalization issues.”

Windows users can download Mozilla here.
Linux users can download Mozilla here.
Mac users (OSX) can download Mozilla here.

Mozilla includes tabbed (Opera style) interface support, built-in pop-up and pop-under ad blocking abilities, the ability to disable annoying status bar scrollers, the ability to block images from particlure servers (e.g. ads.doubleclick.net), skinning ability, optional quick launch (browser loads into memory like Internet Explorer so when you launch it, it loads instantly without having to wait and stare at a splash screen), full PNG image support, HTTP Pipelining, encrypted form & password autocomplete, the built-in ability to pause & resume downloads, and a crapload of other stuff. I highly suggest you check it out if you either haven't used Mozilla before or if you haven't used it in a long time.

Users who are most familure with Internet Explorer on Windows 9x/Me/2000 might want to check out the Internet Explorer (pre-XP) skin for Mozilla here.

And you can check out some cool add-ons and such at mozdev.org.
