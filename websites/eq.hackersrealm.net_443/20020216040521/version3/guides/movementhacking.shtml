How to Find the Speed Offset
By: IceDuster
Print This Guide

 

How to find the Speed Offset First you got to get WinHack because that’s what I use =). What you need to do is:

The Long Way

  1. Load Up WinHack.
  2. Load Up EverQuest and enter the game completely.
  3. Alt-Tab out.
  4. Go to the WinHack Program and click EQGAME under process
  5. Make sure you are under the Memory Search tab.
  6. Click Start New Search (this process should take some time depending on your computer).
  7. After WinHack has finished with its tasks alt-tab back to EQ.
  8. If you are running change to walking speed. The other way around if you’re currently walking.
  9. Alt-tab out.
  10. If you changed to walking speed you would pick the decreased option and if you changed to running speed you would pick increased option. Note you must have the reg. version of WinHack to do this.
  11. After clicking decreased/increased option click on continue search button at the top.
  12. After WinHack is done searching go back to EQ and switch to the opposite speed (walk/run) then alt-tab.
  13. Now click increased option if you switched to running speed and click decreased option if you switched to walking speed. Repeat steps 11-13 tell you get to only a 1 or a few offsets then test them out 1 by 1. That should do it.

The Quick Way

This way requires that you have the previous old offset to find the new one.

  1. Do steps 1-4 in the Long Way.
  2. After completing that click on Edit Memory tab.
  3. Enter in the old address in the go to address text box.
  4. Click Go To Address
  5. Now from here click on the Hex String option and enter in 010101 In the text box to the left if your character is running. 010001 if your character is walking.
  6. Click Find Next. This should find the offset you need at first try. The offset you need is the middle one (ex. 010101). Test the offset by changing the number if it doesn’t affect your running speed then you’ll have to do The Long Way.
