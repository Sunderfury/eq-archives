Spell Tips

  • Flash of Light - This is good to cast if you want to get a monsters attention away from someone. It only makes a monster run away if you are not within hitting distance from it.

  • Spook the Dead - This spell allows you to make an undead mob run away. With sow, you can chase after it and still attack, while not taking damage. Casting root on a mob will negate the effect of a fear spell.
