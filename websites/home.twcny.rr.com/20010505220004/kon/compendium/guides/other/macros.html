Guide to Paladin Social Macros

Basics

You can cast any spell in spell slots 1 through 8 from the keyboard by typing the command /cast # where the # is the spell slot.  So lets say the spell in your first spell slot is Healing. Typing /cast 1 in the chat box will  make you cast healing.   Alternatively, you can also press alt+# Just press the ctrl key and a number that coresponds with the spell slot you wish to cast.

Useful Social Commands

Stuff in ( ) are optional when command is used

/book (#) Opens spell book and goes to page #
/sit (on/off) Makes you sit down
/stand (on/off) Makes you stand
/doability # Does ability that coresponds with the ability # number in the abilities list
/camp Makes you camp out (must be sitting!)
/invite Invites target to group
/disband Disbands (target from) group
/attack (on/off) Attack toggle

  Target Variables

   There are different variables that you can use for macros that depend on what you are targetting.

%T Returns the current target.
%S (He, She, It) Returns the subjective gender-specific pronoun for the target.
%O (Him, Her, It) Returns the objective gender-specific pronoun for the target .
%P (His, Her, Its) Returns the possessive gender-specific pronoun for the target .
%R Returns the race of the target  (only works on Player characters).
  

Examples:

Command Text Incoming! Ack! Its a <<<< %T >>>>!
Actual Text Incoming! Ack! Its a <<<< Fire Beetle >>>>!
 
Command Text Incoming! Ack! %S is coming!
Actual Text Incoming! Ack! she is coming!

Useful Macros

Pulling Macro

Line 1 Incoming! Get up! Its << %T >>!

Healing Macro

Line 1 Healing %T!! This says who you're healing (target)
Line 2 /cast # # is the spell slot your heal spell is in

Simple Chain Stun Macro

Line 1 /cast # This is Stun 1
Line 2 /pause 50 10 = 1 second, so this is a 5 second delay
Line 3 /cast  # Stun 2
Line 4 /pause 50 Delay
Line 5 /cast # stun 3 (could also be /doability for bash)

Simple Any Stun Macro

 This macro will use whichever stun is available at the time.

Line 1 /cast # Stun 1
Line 2 /cast  # Stun 2
Line 3 /cast # Stun 3

Advanced Social Stun Macro

Line 1 :prays for Karana to smite %T! emotes the smite RP text
Line 2 /cast # Stuns mob with spell #
Line 3 /pause 50 10 = 1 second, so this is a 5 second delay
Line 4 /cast  # Stun 2

Lay Hands Macro

A disadvantage of doing this is if you have finicky fingers and often LoH a mob, it would be quite embaressing if you emote 'prays to Marr to save Lady Vox!'

Line 1 :prays for Marr to save %T! emotes the RP text
Line 2 /doability 6 Does ability 6 (in the abilities list) which is Lay Hands

1 Button Camping Macro

Line 1 /sit Makes you sit down
Line 2 /camp Makes you camp out
