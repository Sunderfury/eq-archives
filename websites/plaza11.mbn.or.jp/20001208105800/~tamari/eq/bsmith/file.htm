File作成

Studsを作る時に必要なItemの一つ。
Metal Bits同様にSkill上げを兼ねて大量に作成する事を薦める。
しかしMetalBitsと違い積み重ねる事が出来ないので作りすぎに注意。
新しいPatchによりFileは一度作成してしまえば、その後何か作る時にFileを使用しても、その作成の成否を問わずFileは戻ってきます。
いっぱい作っても一つを残して全部売る事になるので、作りすぎに注意。

必要なもの
Metal Bits(1個)
Water Flask(1個)
File Mold(1個)

やり方
ForgeにMetal Bits(1個)、Water Flask(1個)、File Mold(1個)を置いてCombieボタンを押す。
File

Basic Blacksmithingを読めば分かりますが、Item作成時にFileが必要なのはStudsに限りません。
File Moldをどこで買ったか、どこで買えるのか覚えておきましょう。
