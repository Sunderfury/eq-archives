Links

Adventures of the Silver Blade is the current incarnation of the guild that I have been in for over a year now.

Monks Links

Monk Mailing List at Onelist - A very good source of information for monks.

EverQuest Monks - This site has some very good information for all monks.  There is also a message board at this site, where the inspiration for my site grew.

EverQuest Monks Message Board - A direct link to the above mentioned message board.

EverQuest Monk Ideas - This is a new monk message board.  It seems to be more of a rant board than anything lately.

General EverQuest Links
Official EQ Homepage - Not really that great of a site, but it does have a couple of animated gifs of monks doing round kick.

AllaKhazam's Magical Realm - A very good source of EQ information.  Especially the maps section.  He has links to pretty much every map out there.

Everquest@STOMPED - A rather new (to me at least) site, but has a very nice searchable database of equipment, which has more than 1200 items currently.

EQ'Lizer - Has one of the most complete lists of equipment.

EQSS - A good source for EQ news.

EQVault - Another good source for EQ news.

EverQuest HeadQuarters - Another EQ news site.

An EverQuest Glossary - Has the meanings for all those obscure phrases you hear EQ, like 'kite', 'SoW', 'camp', etc.

Cackling Klaknak - An EQ Humor site.   Pretty funny site.

Ask Azeraphel - A pseudo-advice site.  Good humor and some good advice occasionally.

EQItems - Searchable Database of EQ Items.

Jeff's EverQuest LOG File Parse CGI Page - This is a really cool utility that parses your eqlog file so that you can see exactly how well you are doing in a fight.
