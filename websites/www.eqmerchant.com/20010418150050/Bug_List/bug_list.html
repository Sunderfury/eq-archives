Dyes

  • Iron Oxide - not working when mixed with 2 or 3 resins - it DOES work with 1 resin
  • Russet Oxide - not working when mixed with 2 or 3 resins - it DOES work with 1 resin (confirmed 09/29/00) - there are some reports of this working if you stack the resins, but I’m not sure if you get the same result as you would with one resin (due to the stacking) or if you actually get a 2 or 3 resin dye
  • Permafrost Crystals - According to EQ Traders, 1 Resin Blue (Permafrost crystals) don’t show up when applied to the breastplate (I dont’ know about other pieces). Someone else said they made a full set of 1 resin blue plate and didn’t have any problems.  This might be related to the fact that Erudin Plate is dark to start with, so it might be hard to see the dark blue color.
