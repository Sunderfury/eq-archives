  リベンジその3 　[　2002/5/8　]

目下 5連敗！ Chardok の Queen 様に挑戦です。

これまでのおさらいとしては、

・HP は 20000と少なめ
・Fire 系の AE-DoT を使う（ 4-5秒で 1800前後持ってかれる）
・AE-DoT が 10秒ちょっと毎に飛んでくる

とにかく AE-DoT が曲者（凶悪）で、HP バーの減り方見る限りある意味 DD ですあれ。
結局、強襲して瞬殺するしかありませんが、過去 5回中 4回は Pull ミスや
移動中に Queen ＆ 護衛全部 INC 等のため、なかなか相手をさせてもらえません。

よって、今回は KingRoom への移動に細心の注意を払います。
（ Invisi を使い、1パーティーずつこっそり移動）



KingRoom の雑魚 Repop を掃除して Queen 様の Pull を開始。
とりあえず全部 Pull して FD 。最近密かに Faction 上げてる Crochan 氏が
Hail で Queen 様を引き止めると言う姑息な手段を使って 2段 Pull をすることに。

Puller のとかげさんは袋叩きにあって即死したので、CLR モードに。

「 Melee Go 」の合図とともに減りだすパーティメンバーの HP。
例の AE-DoT は KingPoP 場所（階段下りた窪み）に居れば当らないので冷静に Heal などしてみます。

が、ぜんぜん Heal 追いつきませんよ〜。
窪みに居るので上の状況は見えませんが、戦闘開始十数秒で既に死者多数な感じが・・・

開始から 30 秒程度経過。「あーまずいなぁこりゃ」と思った直後、
なんだかチャット画面が騒がしくなってます。どうやら Queen 様が死んだ模様。




お〜死んでる。で、どうも横で死んでる Wiz の Gekkoh 氏が何かした模様です。

チャット画面を巻き戻して見ましょう・・・

・・・ホームランだ。


なんだか反則勝っぽいけど、無事リベンジ完了。お土産も頂きました。
