　リベンジその1　　[　2002/3/22　]

前回の敗退から 2ヶ月、サーバーのショートダウンを狙って
PoF にて Dracoliche さんに再挑戦です。

ダウン前に一緒に NW に潜入した 1パーティーと共に掃除をしながら、メンバーの集合を待ちます。

合流後の人数 25名。前回の反省を活かし、珍しく念入りに戦闘準備を行います。

作戦としては、

・Slow が入らないと勝負にならない気がするので、意地でも Slow をいれよう。
・Slow が入るまでは、MainTank を死なせてはいかん。
・あとは出たとこ勝負で。なんとかなるでしょ。

てなわけで、Dracoliche さん Pull・・・



・・・「 Dracoliche　Slowed！」、MainTank も健在。
2 ヶ月で Lv・装備を強化しただけあって、余裕な空気が流れます。

とは言え、油断は禁物。周りの状況を見ながら全開でぶん殴ります。



で、危なげもなく勝利。リベンジ完了。

DropItem はハズレぽかったですが、気にしません・・・気にしませんよ！・・・


　ToV−East　　[　2002/3/17　]

ToV こと Temple of Veeshan です。
前回に引続き、鎧の素を求めて EastWing（Test Hall）へ行ってきました。

今回は今更ですが、ゆかいな TestHall の紹介でも。

○Entrance Hall



入口。Brise は普段ここで Camp してます。図の右後ろにドアがあります。
正面の柱の左右は敵の AE の退避場所です。

後ろのドアの外には CoVFac の Mob がいますが、KoS でなくても近くで他人の死体を動かすと凄い勢いで襲ってきます。
他の人には NonKoS なので、素直に見捨てましょう。
あと、戦闘中にドアを開けるのも禁止。なぜか外の Mob が Add する場合があります。先日知人から「絶対 Add しないよ。」
と、言われましたが、やっぱり怖くて開けれません。結論としては、「開けないに越したことはないはず。」ってことで。

FullPop 状態では、左右正面に Wurm が 1匹ずついます。あとは巡回が足元を歩いてることもあり。

最初の Pull ですが、まず適当に Buff を貰いメンバー全員ドアの外に出てもらい、
右でも左でもいいので外側の壁に張付いて Wurm に弓か何かを当てる。

最初はたぶん左右 2匹とも上がってくるので、FD するなり CoH 貰うなり死んじゃうなどして帰ってもらう。
2回目同様に Pull すると、たぶん 1匹しか来ないはずなので確認後再度 FD。

1匹しか来ないことが確認できたら、メンバーに入ってきてもらい戦闘準備。
あとは、巡回いなことを祈って Pull しましょう。

Mob の Repop は 72分。最初の Wurm を殺した時刻を記録しておきましょう。
そして 3〜5分ほど間を開けて 2匹目を Pull します。

※上記の内容については、私個人の経験に基いて書いてあります。助言・補足は歓迎ですが、苦情はかんべん願います。


○Entrance Hallの下



先ほどの図の柱を挟んで反対側から見た図です。階段を上がると飢えた人達が待ち構えています。
階段ですが、見ての通り途中までしかありません。いやん。

以前ここで Camp した時は、多数 INC 対策で Puller が下で FD して階段の上から Pick する方式でやっていましたが、
Mobの高速移動＆ワープで Pick しづらい事と、そもそも「 2INC 以上してたら Puller が途中で死んじゃう・・」
と、言うことで自力で階段まで上がる方式に変更。



乗り方としては、階段手前脇の少し低い所にジャンプで乗りそこから階段へ走る感じ。
説明難しいですが、図の角度でジャンプして着地前に方向キー右上を押しておいて、階段のほうに滑らせるような。

上手く乗れたら、階段の途中で Mob の数を確認して皆の待つ広間へ帰りましょう。


○敵について

敵は、大きく分けて3種類。Wurm 系・Drake 系・Wyvern 系です。色んな名前のやつがいますが、
基本は、「見た目でかくて、名前の長いやつは強い。」です。かんたんです。たぶん。


・Wurm系



でかくてドアにはまりますが強引に突破してきます。
赤いのと白いのがいて、白いやつは CH・Gate あり。Enc 様 Mana 吸って下さい。

MeleeMax は 300〜400前後。400の AE あり。SvF・C が必要。
戦闘中壁にうまりぎみなので、上手く場所調整しましょう。


・Drake系



基本的に弱いが、変な DoT とか Slow あるので厄介。あと小さいので皆が群がって叩きにくい。

MeleeMax は 300前後。Resist 高いやつもいるけど弱いもんは弱い。


・Wyvern系



攻撃速度が速い。特に「Vehement Wyvern」は強烈。長距離だと Pull 帰って来れる気がしません。
うっかりしてると MainTank が瞬殺されるので注意。

MeleeMax は 350前後。とにかく叩きが痛い。Resist は高くないので早いとこ Slow 入れちゃって下さい。


○Itemについて

ここでの DropItem は通称 SoV-Armor と呼ばれるもので、ここで鎧の素となる部品とVL・SG・DNP等で
取れる Gem を Skyshrine に持って行って Quest を行うことで、性能の良い鎧の出来あがり。

鎧の素には、Plate・Chain・Leather・Silkの4種類あり、
それぞれ、Chest・Legs・Arms・Hands・Wrist・Head・Feet、計 28種類のパーツが Random で Drop します。

Drop 率は日によってややムラがあるが、概ね 30％前後。強いMobからは Drop しやすいと思う。
連続で Drop しなくても、Loot 役の人を遠まわしに責めてはいけません。Loot 役もつらいのだから・・・

　空中戦　　[　2002/3/3　]

先週は、Palne of Sky へ行ってきました。
目的は 7th 島での MagEpicItem の取得。Epic リーチの Mag2 名、やる気満万です。

敵の移動経路がおかしかったり、Link がすごかったりと、Pull が大変でしたが、
今回最難関らしい、6th 島へ到着。

6th のボス（見た目カワイイ）ですが・・・
・MeleeMax850！
・AE500＋DoT（病気Base）
・DT有り（45Sec？）

「こんなのに勝てるのか？」と思いながら戦闘開始。
鬼 Melee と DT ドンで、パタパタ死んでいく人達。
FD 使って DT をかわしまくるトカゲ。

力押しでどうにか勝利。



で、7th 島。
物は雑魚からの RandomDrop 。敵の移動経路が変。DT持ちが8体・・・
物が出るまで帰れません。しかし、Mag2 名は Epic 最後の山場。
楽しそうに DT もらいに逝きます。

7th 島到着から、3日目。無事予定していた3つ（？）の MagEpicItem を Get して、
時間が余っていたので 8th 島を見に行くことに。

小さな島に蝶みたいなのが 1匹。とりあえず半壊。
Melee 忘れたけど、もれなく Proc500Dmg 付いてきます。



しかし、HP の回復がやけに遅く、2戦目準備した時点で 45％ 位。
楽々殺して、Drop は QuestItem 1個。まあいいです。

で、無事 MagEpic 完成の 2名。おめでとです。
