Pattern Sets

There are different sets of patterns in Everquest. A merchant who sells patterns will always sell all the items of one set, or the other, but generally not both. For convenience, I'll refer to these as Sets A and B and C. Note that as I search the Kunark and Velious continents, I'm finding some variations (hence the addition of Set C).

Set A - This set includes the original armor items that could be tailored when the game was released.

Boots, Cap, Gorget, Glove, Pants, Sleeve, Tunic

Set B - This set includes non-armor items, as well as armor items that were added after release.

Belt, Cloak, Gorget, Mask, Shoulderpad, Wristband
Backpack, Whip, Quiver, Fleeting Quiver

Set C - This set includes all the armor items.

Belt, Boots, Cloak, Cap, Glove, Gorget, Mask, Pants, Shoulderpad, Sleeve, Tunic, Wristband
