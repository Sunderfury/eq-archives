Gameboy NTSC

These are the materials you need to provide for a Gameboy NTSC T/O. All forms are to be filled out by the Developer/Producer unless noted otherwise.

  • Specification Sheet (filled out by the developer) Downlaod (cgbspecs.zip 15KB)

The Sections that need to be filled out by the Developer are as follows:

MAIN SECTION

  • Game Code
  • Language

PROGRAMMING FEATURES

  • Entire section

MEMORY CONTROLLER

  • Entire Section

MEMORY CONFIGURATION

  • Entire Section

All other information on the specification sheet will be completed by a Turnover Technician

  • 1 Copy of the instruction manual
  • ESRB Rating Certificate
  • All available screen text (in all applicable languages)
