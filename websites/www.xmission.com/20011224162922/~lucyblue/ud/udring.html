Join the Utah Dogs Webring

1. Instructions on how to join
To add your site to the Utah Dogs Webring please complete the steps on this page. First log in to Yahoo so you can access the webrings. Then go to the Yahoo Web Ring Site for adding your web ring. If these links are not working well for you, click on the 'join now' button on the ring navagation bar.

2. Enter the necessary HTML code given to you by the Yahoo registration pages.
The HTML code fragment will cause the following to be included on your page at the point where the HTML code frament is inserted:

3. Wait for authorization.
Your site will be added to the ring as soon as I have time to look it over and add it to the ring itself. If you have questions please feel free to email me. An email will be sent to you when your page has been added. If you don't include a valid email address your site will not be added to the ring.

4. If you need more help installing the navagation bar
If you need more help installing the navagation bar on your website, visit the yahoo help page. Only you can install this bar on your web page. Also, please install the navigation bar on the same page you report as a part of the web ring. If you install the navigation bar on a page other than the one you report, your site may not be added to the ring, or it may be removed at a later time. As the ring grows it will be too difficult for the ring mistress to check each site to veryify which page the navigation bar is on. Thank you!

PLEASE NOTE:
As responsible dog owners we must preserve the privileges we have with our dogs. To do so we must follow some basic rules:
Always pick up after your dog.
Always make sure your dog responds to voice commands.
Always make sure your dog is not aggressive towards other dogs or people.
