The Utah Sailing Webring

1. Description of Webring
The Utah Sailing Webring consists of sites dedicated to sailing in the State of Utah, mainly sites belonging to people who sail the Great Salt Lake. (For more information about what webrings are and how they work, click here).
I will be the one responsible for approving sites for this ring. Since it is new I will consider any site which applies, and expand the criteria for the sites if I receive enough coaxing to do so.

2. Instructions on how to join
To add your site to the Utah Sailing Webring, you must complete two steps--you must complete the following form, and you must add a fragment of HTML code to your Utah Sailing-related web page.
First complete this form and click on the 'Submit Site' button. Then write down the information on the subsequently displayed page. Then use the 'Back' button on your browser to get back here and continue with the next step.

3. HTML Code Fragment
Before your page can be added to the Utah Sailing Webring, it must also include the following HTML code fragment

<!--CODE FRAGMENT--> <P> <center> <table cellpadding=5 cellspacing=5 width=500> <tr><td> <center><img src="images/rlogo.gif"></center> </td></tr> <tr><td> <center> This <a href="http://www.angelfire.com/pq/LucyBlue/gslring.html"> Utah Sailing</a> site is owned by <a href="--your email address--">--your name--</a>. <br> [ <a href="http://www.webring.org/cgi-bin/webring?ring=gslsailing;id=--id--;prev"> Previous Site</a> | <a href="http://www.webring.org/cgi-bin/webring?ring=gslsailing;id=--id--;next">Next</a> | <a href="http://www.webring.org/cgi-bin/webring?ring=gslsailing;random">Random Site</a> | <a href="http://www.webring.org/cgi-bin/webring?ring=gslsailing;list">List Sites</a> ] </center> </td></tr></table> </p> </center> <!--END CODE FRAGMENT--> Note that you must also:
1) Replace --id-- with your site's assigned ID. Your site's ID will be e-mailed to you once the above form has been completed.
2) Replace --your email address-- with your email address, and --your name-- with your name.
3) Upload the rlogo.gif graphic to your web site location

HINTS:
1) You can use any graphic you want (or none at all), but it will be easier to find the Utah Sailing Webring link on your page if you use the same graphic as everybody else. To obtain the graphic, right-click on it and save it to your local fixed disk drive (hard drive).
2) You can view the HTML source of this page and then cut-and-paste the required HTML Code Fragment out of this page and paste it into your page--you will still need to change my ID to your ID, my email address to your email address, and my name to your name.
3) If you're an HTML expert and want to customize the way the text, graphics, or links are presented--go for it. The only limitation is that visitors to your site should be able to easily discern that it's part of the Utah Sailing Webring.
4) If you need more help, you can e-mail me at LucyBlue@Softhome.net

The HTML code fragment will cause the following to be included on your page at the point where the HTML code frament is inserted:
This Utah Sailing site is owned by Cynthia.
[ Previous Site | Next | Random Site | List Sites ]

4. Wait for authorization.
Your site will be added to the ring as soon as I have time to look it over and add it to the ring itself. If you have questions please feel free to email me. An email will be sent to you when your page has been added. If you don't include a valid email address your site will not be added to the ring.
