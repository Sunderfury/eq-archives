EQ Macros

"Taking the grunt work out of playing EverQuestTM"

Product Description
EQ Macros is the first software tool that provides recording & playback of keystroke and mouse macros under EverQuestTM.

Features

  • Allows you to record and playback keystroke and mouse macros.  The playback is continually looped.

  • Easily increase such skills as Sense Heading, Instill Doubt (monks), Sneak, Mend, Casting skills (Divination, etc.), Fishing. Foraging, meditation, etc.

  • With New mouse support, you can create complex macros, such as fishing.

  • Record any keystroke or mouse movement.  You could record moving back and forth, switching to a different keypad, switching view modes, etc.

  • Each user gets a custom executable file compiled specifically for them.  Each executable has different settings & attributes that make it harder to detect its presence.

Using EQ Macros
Record: Press the "[" key to start recording, you should hear 1 beep.  When finished recording, press the "[" key again, you should  hear 2 beeps.
Playback: Press the "]" key to start playback, press "]" key again to stop looped playback.

Notes for Mouse Support
To get the best results, you should calibrate EQ Macros before recording any mouse macros.  This is very easy, simply move the mouse pointer to each of the 4 corners of the screen.  EQ Macros will detect this, and calibrate automatically.  You can now safely record a mouse macro. 

System Requirements
Download and install the latest version of DirectX available for your system.  Do this before you install EQ Macros.  Go to the Windows Update page to see if a newer version is available.  On Windows 2000, DirectX v7 is the latest.

Q:  Will I get caught if I use EQ Macros?
A:  Not if you're smart about it.  ;)  Just like anything, you must be careful how you use it.  Don't macro right in the middle of a busy area.  Don't macro any kind of repetitive speech.  Toggle your "AFK" status while macroing, etc.  Since each user gets a custom executable, it can only be detected if Verant uses invasive (and illegal) methods to detect it.  Ultimately, you are responsible for how you use EQ Macros, and you must use EQ Macros at your own risk.
