Welcome to the Uninet beta site. There are currently three groups of software you can choose from, but these groups will be expanding in the future as we accept new software projects. You can choose from: EverMon, a popular EverQuest server monitoring software; EverGuild, a popular software for EverQuest guild masters; and Other, of where there are currently no projects as of yet. 

WHY OPEN BETA?

EverMon and EverGuild will be beta tested under an Open Beta policy.  This is done to increase the number of testers that we have.  Please let us know if you would like to be added to our ICQ list where we will be announcing the avalibility of new versions. 

RULES FOR USING BETA SOFTWARE

If you are going to be using any of this beta software, please keep in mind the following rules : 

  1. Beta software is NOT supported, and is provided AS IS... There can and will be MANY bugs in the software.  This may or may not cause adverse effects on your computer system. 
  2. You should only use the beta software if you have had experience in the past with beta testing, and are prepared to assume the risk of using un-tested, new versions, whcih may contain bugs.  If you want to use more stable versions, do not use the beta software. 
  3. By using the beta software, you agree to hold Hydra Computers, HydraSoft, the programmers, developers, and anyone else associated with the project harmless from any problems either directly or indirectly caused by the use or misuse of the provided software. 
If you have any other questions about the beta program, or would like to report bugs, or anything else, please feel free to contact the UniNet beta team coordinator...
