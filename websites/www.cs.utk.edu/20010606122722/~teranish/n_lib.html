LAPACK

 LAPACK (Linear Algebra PACKage) is the standard dense linear algebra software on uniprocessor environment. LAPACK supports solving system of linear equations, sigular value decomposition, and eigenvalue problems. Most computational routine is built on the BLAS to utilize the hierachical memory of the modern computers.

PETsc

 PETsc is aimed to develop large-scale parallel PDE solver with ease. The needs for solving non-linear systems result in the support of various iterative linear solver. In PETsc, everything (Scalar, Vector, Matrix, Linear Solver, etc.) is managed as an object. For Fortan programmers it might be troublesome, but this designing feature gives users a great flexiblity and usabilty to develop a large-scale scientific application. Also PETsc gives an inteface for various scientific computation libraries such as SPAI, BlockSolve95, etc.
See PETsc homepage

DSCPACK-S

DSCPACK-S is multifrontal symmetric sparse direct solver on uniprocessor environment. All the information about a matrix is hidden during execution. It supports 3 orderings ( Nested Disection, Modified Minimum Degree and Modified Minimum Deficiency). Parallel version will be available, soon.
Contact Padma Raghavan, if you want to grab the code.
