This web site was designed for YOU. This web site doesn't work unless people like yourself interact with the web site. The following are ways you can help us help you.

Send us your screen shots
If you see something funny, interesting, cool, or anything else that you think other people would like to see please send us a screen shot of it. We will be putting up a new screen shot everyday and if you send us a screen shot and we find it interesting we'll post it.

Bookmark Us
Please bookmark our website! Do it right now so you don't forget! That way you'll always know how to get here. Don't know how to bookmark? Just go to the start page (by clicking the "Daily News" link and hit the button at the bottom of the page that looks like this...

Make us your start page
We update this web site EVERY DAY! The best thing to do is make us your start page so when you start your browser this is the first site it goes to. This way you can find out the latest EQ news, read the message board, jump into the chat room, or see what's for auction in the EQBay auction house!

To make us your start page just go to Tools - Internet Option and then type in www.corematrix.net/knightlife

Support our sponsors
Running a large scale web site like this one isn't easy.. or cheap. Please visit our sponsors by clicking on their banners whenever you can. All banners will open up in a new browser so that you can minimize them and go back at a later time. We'd just like to say thank you in advance for supporting us and our sponsors!

Report dead links
If there are any dead links on our web site please contanct us and let us know so we can fix the problem. Also, if you have any other problems with the web site please let us know!

Put your 2 coppers in
Have ideas? Comments? Questions? We're always glad to here them. Just send us an email and we will always respond as soon as we can.

And give feedback!
Let us know what you think of this site!! We designed this site for you and we always love to hear your comments!
