Cleric Quest

Lv30くらい以降のClericにとって有用なItemが得られるQuestをまとめてみました。一応、適正Levelと必要人数も書いてありますが、あくまで参考値でしかありません。Party編成やキーマンのLevel、装備その他に依存しますので、一概には言えません。難易度も、イコールMobの強さではありません。面倒さなども含めた上での私感です。

Bazzarの登場によって、Questをしてまで入手するようなものではなくなってしまっているものも、少なからず混じっています。それでも自力でQuestをしたい人の参考に、少しでもなれば幸いです。

Epic Weapon

各Classに導入された長編叙事詩Quest。伝説の武器Questといってもいいか。伝説の武器と呼べるだけの性能を持ち、登場してから拡張パックが3つ発売され、ステータスのインフレが進む今なお現役で使えるものとなっている。独力での入手不可能であり、それなりの装備をそろえた3party+を集める人望を必要とする。

  • 適正Level : 55+
  • 必要人数 : 18+
  • 必要Faction : -
  • 難易度 : ★★★★★

SoV Armor

Velious大陸追加と共に導入された、Class専用ArmorのVer.UP版。見た目はSoVで新規に導入されたもので、種族によって異なる。

  • 適正Level : 50+ 〜 55+
  • 必要人数 : 6+ 〜 18+
  • 必要Faction : Coldain | Kromzek | Claws of Veeshan
  • 難易度 : ★★★ 〜 ★★★★★

Symbol of Tunarian Worship

Plane of Growth発のQuest。Tunare信仰のHigh Elf/Clericのみ装備可能なBracerが入手できる。

  • 適正Level : 55+
  • 必要人数 : 18+
  • 必要Faction : Servants of Tunare
  • 難易度 : ★★★★

Talisman of Benevolence

Skyshrine発のQuest。俗に言う「uber」なMobから出るものを手に入れるまではお世話になるであろう性能のRange Itemが入手できる。

  • 適正Level : 50+
  • 必要人数 : 6+
  • 必要Faction : Claws of Veeshan
  • 難易度 : ★★★

Circlet of Falinkan

Skyshrine発のQuest。Caster/Healer向けのCrownが入手できる。性能的にはSoV Armorよりやや上か。

  • 適正Level : 55+
  • 必要人数 : 12+
  • 必要Faction : Claws of Veeshan
  • 難易度 : ★★★

Girdle of Reflection

Wakening Land発のQuest。難易度を考えれば満足できる性能のBeltが入手できる。

  • 適正Level : 45+
  • 必要人数 : 3+
  • 必要Faction : Kromzek
  • 難易度 : ★★

Aegis of Life

North KaranaのNPCから始まるCLR/PAL用ShieldのQuest。背中に担げるShieldとして有名。

  • 適正Level : 30+
  • 必要人数 : 1+
  • 必要Faction : The Knights of Thunder
  • 難易度 : ★★

Paw of Opolla

South Karana発のQuest。中Level以降で世話になるITEM。

  • 適正Level : 30+
  • 必要人数 : 1+
  • 必要Faction : -
  • 難易度 : ★★

Testament of Vanear

Erudin発のQuest。中級Healerご用達のWIS10 Book。

  • 適正Level : 30+
  • 必要人数 : 2+
  • 必要Faction : -
  • 難易度 : ★★

Sol C Armor

Rubicite Armorの廃止に伴って各Classに導入されたSolsek Temple(SolC)のQuest。Clericには金色のPlate ArmorがHelm以外各種と、武器・盾が用意されている。

  • 適正Level : 30+
  • 必要人数 : 1+
  • 必要Faction : The Temple of Solusek Ro
  • 難易度 : ★

Dragon Name ←作成中...

Earringとしてはかなりの性能持つものが入手可能だが、難易度はEpic並それ以上。

  • 適正Level : 60+
  • 必要人数 : 18+
  • 必要Faction : Coldain / Claws of Veeshan
  • 難易度 : ★★★★★

Protection of the Cabbage ←作成中...

Trade Skillで作れるFT付きEarring。材料を購入する資金があれば難易度は低い。

  • 適正Level : 51+
  • 必要人数 : 1+
  • 必要Faction : -
  • 難易度 : ★★★★

Coldain Prayer Shawl ←作成中...

8段階まであるShawlを作る成長型Quest。最終的にはAlchemy、Tinkering、Make Posion以外の全ての生産Skillを必要とする。Questは戦闘よりも、Skill上げがメインといえる。なお下の難易度が高いのは、Skillあげの辛さと、それに伴う出費を考慮してのもの。Mobを倒すという意味での難易度は低い。

  • 適正Level : 55+
  • 必要人数 : 1+ 〜 18+
  • 必要Faction : Coldain
  • 難易度 : ★★★★

Ring of the Shissar

Ssraeshzra TempleのEmperor Roomへ入るための鍵である、鍵のクセにいい性能なRingが入手できる。

  • 適正Level : 60+
  • 必要人数 : 6+ 〜 18+
  • 必要Faction : -
  • 難易度 : ★★★★
