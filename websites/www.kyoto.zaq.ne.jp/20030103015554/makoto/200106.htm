Diary

6月30日
        今週もToV Armor Camp行ってきました。
        水曜日のパッチ後、PlateのDrop率が上昇したのか、一日で7個でました。
        Loot順をうっかり間違ったのですが、Armplateを頂きました。
        ACがNon-Buffで1100行きそうな感じです。でも、STA＆Haste Itemがしょぼい^^;;

6月17日
        昼頃目が覚めたので、Login。
        Enrye、Axamin、Brunhild、Zalaan、Gedou、CalenというメンツでSebilis UGPへ。

        Encがいないので、BardのGedouさん頼りです。
        Breakは問題なかったのですが、RepopとPullが重なって逃走。
        Adeptが逃げて、Priest、Warriorがaddしてしまい、逃走。
        2〜3回Breakし直したでしょうか。
        キノコ Warrior、Priestあたりは、なんとかなるのですが、Adeptはきついです。

        ご飯落ちするまで、もうちょっと粘ろうかと最後のBreakをしました。
        Breakが終わってさてCampというときに、某CleがPortalにはいって行きます。
        ということでCamp終了^^;

6月15日
        Farming 5日目。
        本日の1st Breakはすんなりと行けました。Drakeを順調にClearしていき、WurmをPullです。
        しかし、このWurmがくせ者で、死にかけると、Snareがかかってるのにwarpをしてしまい、
        それにSummonされたりで、RoamerのDrakeがAddに次ぐAdd。
        何回かそれをくりかえし、数が回せません。
        5日間で得た情報として、Wurmを壁に押しつけでやってみようということに。
        warpはありましたが、Drakeのaggro rangeまで行くことはなく、addなしで始末出来ました。

        Campが終わってから早速KaelでBootsを作成してきました。色は深緑っぽい色でなかなか
        いい感じです=)

6月14日
        Farming 4日目。
        今日の1st Pullは、1incでした。いい感じで行けるかな？と思ったのですが、次のWurm戦
        で、死にかけのWurmがワープしてしまい、追いかけていったところに、4add　^^;;
        CR後は順調にMobをKillしていきました。
        が、しかし、最後に落とし穴がありました。
        SVが異様に高い、トンボTypeのDrakeにGateされてしまい、Summoned&RD。
        LogoutとEvacは成功したようなので、全滅は免れました。
        Non-KoSの人たちに、CRしてもらい、無事回収出来ました。

        本日のDropはCloth Dayでした。

        Plate Class　：　Bracer　Legging
        Chain Class　：　Arm Boots Bracer
        Leather Class　：　Glove
        Cloth Class　：　Glove Cap Bracer Sleeve

6月13日
        Farming 3日目。
        前日と同じような感じで、Safe Pointから再開です。
        本日も初めのPull時4incがあり、いったんEvac&CR。Full POP時は難しいですね。

        本日のDropはBP Day!!。

        Plate Class　：　BP X2
        Chain Class　：　BP X2　Glove　Boots
        Leather Class　：　Glove　Bracer
        Cloth Class　：　Bracer　Legging

        Plate、ChainのBPの引きがすごいですね。

6月12日
        Farming 2日目。
        昨日、Center Areaの下にあるSafe PointにてLogoutしていたので、そこからStartです。
        はじめのPull時に4incしてしまい、無理だということで、GG CasterがあるGroupはEvac。
        そのほかのTanks、Casterはその場で死亡。
        Non-KoSのMag、Cleの人たちの、CoH、Res WorkでCRしました。

        その後は順調にPull&Killしていき、East方面のWurmやWyvern等まで回しました。

        本日もDropは好調です。

        Plate Class　：　Glove　Legging　Arm
        Chain Class　：　Boots X2　Bracer　Legging　Legging（CoV用）
        Leather Class　：　Arm　Legging　Glove　Boots
        Cloth Class　：　Robe　Sleeve

        なんと言っても、Caster用Robeが出たのはすごいことです。
