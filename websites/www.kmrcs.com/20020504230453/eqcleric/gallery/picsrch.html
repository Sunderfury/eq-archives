Information:

  1. Please use JPEGs (.jpg) only to upload.  Everquest stores the pictures in bitmap (.bmp) format; you must use a graphics package to convert the screenshot first.  You can find a good packages at http://www.tucows.com , http://www.download.com , or http://www.shareware.com.
  2. Please use the 640x480 resolution when taking portraits for upload.  This will help reduce the size of the files and also make the picture easier to view in its entirety for those who run a lower resolution.
  3. Please try and keep the file size under 100k.  If you followed the recommendations above the picture should be well under that size.
  4. Please do not upload pornographic or other questionable materials.  This feature of the EQ Cleric site is intended for character portraits.  There are many mailing lists where pornographic submissions would be quite welcome.
  5. If you wish to update your character information, please use the Delete feature above and then re-submit.  I plan on adding an Update feature, but it is not implemented at this time.
  6. Enjoy this site.
