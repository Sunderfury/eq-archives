  • vs a frenzied ghoul
    久々に Ruins of Old Guk (Lguk) なんぞ行ってみたら、なんと３人しかいない。
    「わーい、Sash Get できるかなー」
    ってな感じで気楽に frezied の所に行ったらなんと空いてる。
    早速、Knight さま呼んで Full buff して camp 開始。

    Place Holder を２回ほど kill した所で a frezied ghoul pop!
    「おっしゃー、Sash- Sash-」
    ってな感じで Dispell pull を開始。
    Root -> 殴られ（笑）-> Knight さま attack 開始 -> Tashania -> Shiftless Deeds (slow) -> Berserker Spirit -> Feedback (DS) -> Gasping Embrace(DoT) -> Color Skew (Stun) -> Incapacitate -> Weakness -> Berserker Spirit ......
    ってな感じでスタートしたものの、「HP 全然減らねー。(^^;」
    「しかも、Berserker Spirit 消されるのはえー。(^^;;」
    「こりゃ、やばいかな。」
    と、思いつつこっそり Gate を slot in。（笑）

    暫く Color Flux と Color Skew で Stun しまくってみるものの、HP の減りの方が少ない。このまま Stun やり続けていても Mana が無くなるのは目に見えているので Stun を止めて Berserker Spirit しつつ Medi をすることに。
    運良くというか、運悪くというか、そこへ a urd ghoul wizard な巡回 mob 登場ぅ〜。(^^;
    とりあえず、Dazzle して Mana ちゅるちゅる(Theft of Thought)１回する。
    が、やっぱりどう考えても Mana が足りない。(^^;
    「くそーーー！」
    と負け犬の遠吠えを Guild chat に流して Gate。（笑）

    その後 Silru に help して貰って、Root と DoT だけ（笑）で無事 a frenzied ghoul を kill。しかも、DoT が切れるまで暇だからってなもんで Executioner まで kill するという荒技付き。（爆）
    やっぱ Necromancer の DoT つえー。(>_<)/
    ちなみに drop はハズレの Ring だったとさ。(T-T/
