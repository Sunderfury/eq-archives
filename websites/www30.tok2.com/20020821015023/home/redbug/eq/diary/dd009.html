久々に最近の日記など〜。（笑）
空白期間はそのうちに書きます....ええ、書きたいです....書くと思います....。(^^;

login すると Fungus Grove で、最近 exp を稼いでいる場所が全部埋まっていたので、実験を兼ねて Sebilis に行くことに。
Eyenack/Rng、Sorayax/Wiz、Basia/Enc の 3char が揃った時点で、Emerald Jungle に GG。問題なく Sebilis まで走り抜ける。

Sebilis に zone in して camp check を行うと、狙っていた guardian は PLer が camp 中。
necro (jail) と kitchen が空いていて、disco が open soon だそうな。
「うーん、healer いないからなぁ。disco はちとやばそうな感じだし、necro で camp しつつ kitchen から pull して様子見ながらいくか....」
などと考えていたら、「heading to disco」などとぬかす奴がいて、自動的に necro へ向かうことに決定。

2inc とかしたら嫌なので、Basia で pull しつつどんどん進んで行く。
特に問題もなく necro に到着。その間にも camp check な ooc とかあったりしたので適当に答えつつ、どんどん Basia で pull する。
到着して 5 分もしないうち、kitchen 方面に向かう party が出現。
necro だけでやっていてもつまらんのと、先ほど「heading to disco」とか言っていた奴がいなく、かつ camp check に反応しないので急遽 disco にチャレンジすることに。

「Healer 居なくて disco かぁ。やばいかなぁ....まぁ、やばくなったら evac すればいか....」
と思いつつ disco 方面へ向かい、同様に Basia でどんどん 1pull する。
kill speed は Eyenack、Sorayax ということもあってか、それなりに早い感じ。
お陰で repop に遭遇することなく、disco に到着。
入って右側をクリアして、右側の部屋からちまちま 1pull。右側の部屋をクリアした後、disco camp point から Basia で、これまたちまちま 1pull。
左側は無視して、奥の扉を開けて、これまた Basia でちまちま 1pull。
ついでに、どんな mob が Direcharm できるか try。
結局 dar knight が出来ただけで、bok とかは無理でした。(^^;

さて、奥の部屋の ilis とかが 4pop している部屋をどうやって pull するか考える。
「これ 1pull できるのかなぁ？ ilis class が 2inc とか来たら、healer いねーからかなりやばいよな....」
とか戦々恐々としつつ、無事 dar を 1pull。
難無く撃破して、ilis を 1pull。
が、さすが ilis class。むっちゃ痛てーっす。(^^;
幾ら、短時間で敵を剥がすことできる taunt 力をもつ Rng でも hp buff の無い Basia から引き剥がす事が出来た時には、残り 2B でした。(^^;
その後は特に問題なく ilis を撃破したものの、repop が始まっている。
「むぅ。やはり 3chara、healer なしだとこの辺が限度かなぁ？」
Add しそうな位置にいる repop mob を殺しつつ、どうやったらさらなる奥地から pull できるか考えてみる。

camp check に答えているときに、いきなり Mejojo から tell get。
これから Trak を殺りに行くらしい。確かに who をしてみるとどんどん人が増えている。
伝説を作りにいくのかなぁ〜、と思いつつ人がいそうな感じだったので、そのまま discoに居座り続ける。(^^;

すると named な虫さん pop。
ちょろく殺して、変なシャベルと虫のトゲをゲット。いまいちかも。(T〜T
その後は、repop が始まるまで奥から pull するというのを繰り返しつつ exp を結構い い感じで get。

なーーんて思っていたら、お約束発動。(^^;
DC pet も居るし、Eyenack、Sorayax で攻撃力結構あるので油断していたら、Snare する前に mob が逃走。（苦）
止めときゃいいのに、Eyenack で先回りしつつ Snare try。
が、結局 disco を出たあたりでやっと Snare 入ったんだが、追いかけていた DC pet が無用な add を誘発して 3add + 外の建物から 1add。（笑）
さらに、運が悪いことに、Mez は Slot が足りないことからいつもの Dazzle ではなく Kintaz しか入っていなく、Mez 出来る時間が激短い。(^^;
逃走 mob は kill したものの、keep mez の mob がなんと mez を 2 連続 resist。
これで他の mob が起き出してきたり。(^^;
AE Stun で固めつつ Mez を try しつつ、Eyenack で DC pet が attack している mob に attack。瞬殺シフトを引くために、Sorayax を Eyenack に target し assist 後、3sec DD を cast して放置。
が、これまた運悪く、寝てない mob に AE Stun が Resist され Basia の HP は残り 1.5B に。
「むぅ、こりゃ駄目だ。Basia 死亡と引き換えに evac だな....。」
覚悟を決めて Evac を cast。2 テンポほど置いて Basia 死亡。
Eyenack の操作に専念し、起きている mob 全てを Taunt try するが Sorayax に行った mob を剥がすことができず、Sorayax 死亡。(^^;
「うは、こりゃやべぇ。(^^;」
「とりあえず、Eyenack の装備さえ回収することできれば、また break できるので少しでも Entrace の近くまでいかねばぁぁぁ！(>_<)/」
速効、Entrace に向かって、Eyenack を走らせるが、結局、坂の途中で死亡。(T〜T

「ぐううううう、やっちまった....(^^;;;」
Eyenack と Sorayax から Basia に consent。すぐに Eyenack を落として、Eyenos/Clr を login させて Eyenos、Sorayax、Basia で Sebilis に向かう。
Sebilis について死体回収に向かおうと思ったが、途中結構 poped。
しかも、See invisible 持ちの mob もいたり....。
まぁ、そこは Enc なので Basia でいろいろと小細工しつつ、なんとか坂の途中で死んだ Eyenack の死体を Entrace に引っ張ってくることに成功。(^^v

Eyenack を Res して、disco までの道のりを break。
無事全滅 point に到着したので、たまにしか mob が来ない半安全地帯で、まず Basia から res。Mana が回復するまでは、Eyenack と Eyenos に警護してもうことに。
Basia の Mana が十分回復したところで、Eyenack を落として Sorayax を res。
Eyenos をここで logout させて保険としておいて置くか考えたけど、結局全滅した後で Cleric のみで Disco 内で CR するのは不可能と思い、Gate させることに。
その後、最初の 3chara、Eyenack/Rng、Sorayax/Wiz、Basia で Disco camp をしていると Trak 組み全滅という話が....。救援部隊も向かったようなので、まったり Disco で camp を続ける。

暫くして、Halt から tell が。
Healer 無しで、Disco camp していたのを不憫に思ってくれたみたいで、Eyenos/Clr と Eyenhalt/Wiz で来てくれることに。
Sebilis に到着したら迎えにいくことになっていたので、近くまで来たことを確認して Entrace 方面の mob を pull しつつ移動。
と、外の mob をうまく pull できず、気がついてみると Disco 出口、扉のところで 2inc & 2add。（笑）
「うぎゃーー、あちいぃぃぃぃぃぃ！！」
と叫びつつ、なんとかさばききる。(^^;
でも、Eyenack と Basia の残り HP はお互い 1B。（笑）
「あぶなかった....(^^; さてうんじゃ迎えに行くか....」
と、移動を開始しようかと思った矢先、目の前に Eyenos と Eyenhalt が。（笑）

途中に camper いて、mob いなかったそうな。（笑）

chara の操作を担当を替えるために、一度 Eyenack と Eyenhalt を logout させる。
これで、
Halt 操作担当
Eyenack/Rng、Eyenos/Clr

おいら操作担当
Basia/Enc、Eyenhalt/Wiz、Sorayax/Wiz

ってな感じに。
buff も終わって、Halt が Eyenack で pull。
pull してきた mob に Basia で slow をぶち込んで、Eyenhalt と Sorayax を Rain DD で自爆させないように移動させて.....
同時に、Rain DD を Cast。 Rain DD の Cast が終わった直後に同じく同時に Icecomet を Cast。
結果、Mob 瞬殺。（笑）
Dar class だと HP が 3 分の 2 ぐらいから一気になくなります。
600dmg x3 と 645dmg x3、1120dmg x2 で全部入ったとしたら、えーーと、1800dmg + 1935dmg + 2240dmg = 5975dmg と....
うむ、すばらしい。（笑）
「これで AE できたらどんなにいいことか.....」
とか思ったけど、羽織でそんなん無理っぽいので、速効諦め。(^^;

と、mob を 5,6 匹倒したて DC pet も get したぐらいに、Eyenack が SST に呼ばれて解散と相成りましたとさ。
