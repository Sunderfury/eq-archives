★★★ 概要
 この EQ_OmiMap は ＥＱ の /loc コマンドで表示される座標を入力することにより 自動的に地図を作成するツールです
★★★ 特徴
 ・ 地図の大きさを自由に変更できます（東西、南北の座標 および 縮尺） これによりプリントアウトの紙の大きさにジャストフィットさせることができます
 ・ 描画は 「●」 と 「折線」 の ２種類です 色、大きさ 変更可能です
 ・ 地点描画の 「●」 の近くに説明文字を書き込めます 文字の色、大きさ、描画位置の変更可能です
 ・ MS-Access 97 のアプリケーションですので一覧表での検索、並べ替え、抽出など MS-Access 97 の機能を自由自在に使えます
 ・ 入力したデータは MS-Excel 97 形式のファイルとして保存できます もちろん読込もできます MS-Excel 97 を使ってデータの加工も可能です
