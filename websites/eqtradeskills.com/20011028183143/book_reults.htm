Hi,

In relation to having a guide book when working on the related skill, this
is what I have (empircal evidence) found (at least with Blacksmithing):

* No book or book inside container (including opened containers), you
either rely on base skill or have skill reduced (not 100%).
* Book in main inventory slot (closed), seems to be base skill level
or slight bonus.
* Book in main inventory slot open, definite skill bonus.

I haven't had the time to check this yet, but I have this idea on that might
make leveling a trade skill easier or cost less (assuming basic tasks are
cheaper to perform than harder tasks) :

1. Perform basic task, till trivial, with book in inventory slot and open.
2. Perform basic task, till trivial, with book closed.
3. Perform basic task, till trivial, without book.

The reasoning behind this is that if you get a bonus from having the book,
this bonus might make the task trivial at a lower skill level (a book may
have a + to skill level or + to success chance).
