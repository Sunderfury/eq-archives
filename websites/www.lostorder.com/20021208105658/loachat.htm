Connecting with LOA Members via IRC:
 Step 1
If you haven't installed a chat client, do so now. Once installed, run that application.
Step 2

You must now tell your chat client what "chat server" to connect to. There are 'thousands' of chat servers to choose from. Luckily for you, LOA uses a great service provider called "mystic.net".

You simply tell your chat client to connect to: irc.mystical.net
Step 3

You need to give yourself a nickname for the chat server. To do this, type the following command in the "Status window":

   /nick YourNickname

Now you can register this name so that nobody else can use it. To do this, type this command in the status window:

   /msg NICKSERV REGISTER
   yourpassword
Step 4

Now that you are connected and have a unique, registered nickname, you can enter/join our channel/s.

To do this, type the following command:

   /join #akalabeth
   /join #loa-uo
   /join #loa-ac
   /join #loa-sb
   /join #loa-ww2

If you have any questions, just email our council. We will do our best to help you.
