Sunday, November 18, 2001
Another patch is on the way!

From the Front: (and you know its a mad house at CRS)

Version 140 will be released this week, probably before Wednesday. Note that how they version the program has changed. The second number now denotes a version that does not significantly change to graphics or game engine. That being said, here's whats in the new version, straight from the mouth of Mo.

-We re-enabled the J key for riding trucks. troops can jump off at any time again, however, we also added a velocity check when they do, and lets just say it can be "dangerous" to leave a moving vehicle. Troops can also be "thrown" from trucks and tanks if those vehicles impact buildings or trees etc at speed. And again, being in a vehicle mishap can be very nasty for riders. As well, troops can be run over and killed by vehicles again.
-We redid all of the map icons with less saturation. The map should look a lot better now. We also gave the in-game map a transparency setting (you can change the setting using "settings.exe" and the "misc" tab). In that same tab in the settings app you can select if you'd like the mouse on or off at spawn as a default.
-We redid all in-vehicle screen display text to a font that is much less "horrible".
-We redid all the icon display technology, they should look sharper and function better, and aircraft types or player Ids won't show until the IFF color draws. This will make ID'ing planes a bit more fun.
-We fixed the bomb collider to detect high altitude drops more effectively.
-Crew members on multi crewed planes can now red/black out.
-We added an in-game clock, tied to the game time of day (use "numlock" to view)
-The Bf109/Bf110 elevator trim changes are now visible in the 3d model, also aileron droop on flap deployment is modeled.
-We improved the mobility on the 47mm AT gun, it should be movable over most terrain now.
-The P38(t)'s machinegun is now aligned with gunsight
-The P38(t) and PzIIIf's ammo loads were revised to match historical loads
-The Char B1's hull machine gun has been realigned
-We added new interior main gun sounds for most tanks.
-We added separate color for squad-member icons, these will show white for now, and we will eventually allow players to set this color in the settings app.
-We added a forced exit after death. This basically shift esc's for you after a timeout.
-We changed the red-out at wounding effect to be less opaque, so unless you are almost dead, you'll still be able to see a bit.
-We enhanced the "Death Cam" feature to be a bit more dramatic.
-We hunted down another cause of the yellow flags, these should now be fixed.

Also mentioned in a different post, is a major crash bug fixed. Apparently data was not matching from the player to the server (the sever wasn't getting what it thought it was getting). This has been "tortured to death" as Mo was quoted as saying, and should make the game a lot more stable.
