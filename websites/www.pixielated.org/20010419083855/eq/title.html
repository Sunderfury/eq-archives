Micro Rant:
Resurrection Timers

I was told / reminded a few days ago that when the servers go down and the timers are reset, this includes any resurrection timers. Unfortunately the default setting is 'expired'. So if you have a corpse before the server goes down, you won't be able to be rezed once it comes back up. Be sure to get rezed before a scheduled downtime.
