全てのArchetypeは常時発動してるタイプです。

Class Abilities

Class abilitiesはレベル59以上でArchetype Abilitiesに12スキルポイントを費やしたキャラクターが使用出来ます。そのキャラクターのクラス以外のものは選択出来ません。
Class abilityには2種類あり、最初のタイプはArchetype abilitiesに似ていて、上昇に3スキルポイントが必要なスキルです。2段階目で6スキルポイント、3段階目で9スキルポイントが必要になります。

Class abilityの2番目のタイプはスキルレベルが1段階だけです。それらのスキルは異なったスキルポイントが必要で、それぞれのコストは説明欄をご覧下さい。

全てのClass Abilitiesはリフレッシュ時間が無い場合は常時発動型です。
