EverQuest Center's Class Guide.
Welcome to EVQCenters Class guide. From this page you can learn all about the classes you can choose in the world of Norrath. This page will be updated as we discover more of EQ's class information. If you have anything to contribute, please mail it too us. All contributed material will be given credits. If you contribute a lot you will be given an e-mail account here at EVQCenter.

Casting Abilities
Different spell casters get their ability to cast spells from different statistics. It is a good idea to know the answer to the question, "Wisdom or Intelligence?," for your magic using class.
