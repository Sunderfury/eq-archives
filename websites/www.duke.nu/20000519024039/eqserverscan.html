EQ ServerScan

EQ ServerScan is a program that shows you the status of the game servers for the popular online game EverQuest. The program also shows you the number of users playing on each server. Furthermore, you can set the program to signal when a certain number of users are logged in to one or more servers. This will insure a true indication that the server(s) are up if the servers have been down for an extended time. You can also have EQ ServerScan start the EverQuest program. This version has automatic patching.

Information May 13 2000:
The eqserverscan.log file format:
It is a standard "line by line" log with timestamp and trailing serverinfo.
The server information is in binary format to keep the log compact.
The format is:
YYYY:MM:DD HH:MM:SS-UUSUUSUUSUUS...
The time format should be self explaining, the server information is as follows:
UU = Users on server, h0000 to hFFFF
S = Status, h00 to hFF. Values above zero means server is up.
The servers.dat file shows the order of the servers in the eqserverscan.log file.
With any questions about this, please mail me at eqserverscan@duke.nu or post on the EQ ServerScan board http://www.duke.nu/eqserverscan.html.

Features:

  • Compact, uses few resources.
  • Easy to use.
  • Can sound alarm through Internal PC Speaker (great for playing another game or music while you wait for the servers to go online).
  • Multiple alarm trigger settings (makes sure you get a true indication the servers are up when the alarm sounds).
  • Sortable server display.
  • Autodownload and patching of new program versions.

Verant interactive has examined the way this program communicates with their status server and found it most efficient.
