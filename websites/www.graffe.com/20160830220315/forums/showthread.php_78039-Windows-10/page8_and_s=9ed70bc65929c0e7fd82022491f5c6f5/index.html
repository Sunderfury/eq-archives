    Re: Windows 10

    Originally Posted by Dawlin
    I thought I fixed it, by telling Windows Update only to update when I tell it to (after it warns me that it needs to install updates yadda yadda). Didn't work. Dwammit.
    Only a very limited number of things should be able to wake a machine up from a full power down.

    Windows update is definitely not one of them, as it requires the kernel to be in resident memory in order to function.

    Everything that can wake up a machine from a full power down has to be at the firmware level.

    iLO/DRAC, WoL-enabled NIC, various other proprietary things (I've seen backlit keyboards that do it when they send a heartbeat signal to the USB Bus to see if they should be lit or not).

    Try opening an Administrative command prompt and typing:

    powercfg -devicequery wake_armed

    That'll list everything that's currently ALLOWED to wake the computer, although that's SUPPOSED to only be "Wake from Sleep".
