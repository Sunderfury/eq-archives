    Re: 5.4

    Small change: the world boss that requires a legendary cloak will now allow any alt on an account that has a legendary cloak to participate. This doesn't really fix the problem (you need a legendary item, which requires a very long, multiple raid quest chain, to do an open world boss), but I'm not sure the devs see that as a problem. Yet. It sounds like there's four other world bosses on Timeless Isle, the four celestials, which spawn in order as they're killed (so one is almost always up).

    The feedback so far on the Timeless Isle seems disappointing. Lots of multi-million health trash mobs with insta-kill mechanics, likely designed to be done by groups. One quest for exploration, one daily quest to gather piles of crap, and lots of random things spawned around the island (like chests with 496 gear in them). Like the dinosaur isle, you run around killing stuff to gather coins to buy loot. The devs are calling this "self-directed gameplay". I'll let the words of one of the playtesters speak for me:

    Originally Posted by Feedback thread
    While this is more personal opinion then fact, I would have enjoyed something more tied to the actual events of the patch, rather then some "farewell to pandaria" that is barely connected with what we have done, other then all these characters showing up for no discernable reason.

    The grinding for loot type of gameplay is pretty boring, and kind of lazy. While people complain about daily hubs all the time, something along the lines of 5.1 would be the best route to go, over something like this in the future.
