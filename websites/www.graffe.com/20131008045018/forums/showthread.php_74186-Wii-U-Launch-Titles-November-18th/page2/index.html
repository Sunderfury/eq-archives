    Re: Wii U Launch Titles - November 18th

    So I finally got my Wii U and set it up. Having to patch right away is really annoying, but life goes on I suppose. This is sadly becoming the life of console gaming. The system loads everything horribly slowly. Like 15 seconds to pull up anything. All the games have optional patches as well, but that is all for Miiverse. You can skip them, but you miss out on Miiverse which is really cool, fun, and possibly conjured by some long lost magic. More on that later.

    The tablet controller is really interesting. It acts as a universal remote making it a breeze to get a game going which I like. It supposedly works with DVRs, Satelitte, etc but I only have a simple TV so I can't speak to those. The battery life is pretty bad. At max brightness it will die fairly quickly. I am still testing how much more time you get at lower brightness/sound settings. The AC adapter has a fair reach at least. The button layout is such that even those with small hands should be able to hit all the buttons needed for a game with ease. I rather like the larger size as my gigantic hands can rest comfortably on it. My fiancée says she likes it as well and has more human sized hands. The touch screen is very responsive, and you could use it without the stylus should that float your boat.

    The eShop is really nice as well. Almost all the games available for the Wii U are up there. All I have found missing are Tekken Tag Tournament 2, and Mass Effect 3. The former is supposed to be available in other regions as a download to boot. The latter, well, you would have to be really desperate to play ME3 to even consider spending money on this version so no loss there. I am certain a few others are missing, but the important titles are waiting to be bought. The layout is not the greatest, but it is the best Nintendo has offered up yet. Hopefully it gets a bit of work.

    Miiverse is what really impresses me though. Nintendo has managed to create a gaming community that is positive, helpful, and absent of any penis drawings. There is some very impressive art work to be found though. The integration with the Nintendo Network ID means you can send friend requests the exact same way you would on any other platform with the option to make it work more like the 3DS where both sides must agree. By default it is setup the easier way. It also acts as a form of achievements. Whenever you pull off something cool it will prompt you to post it to Miiverse. Alternatively it will prompt you to post if you get stuck on a certain point to get advice. From what I have seen you can get some fast, friendly, helpful responses every time.

    The games have all been extremely fun. Nintendo Land is amazingly adorable. The games are simple, but entertaining. They have some novel uses for the tablet as you would expect. Nothing that makes me fall in love with the concept of the tablet, but most of them manage to sell how it could be pretty nice. NSMBU is as amazing as could be hoped. A proper sequel to NSMBW. Very fun level design, fun new power up in the squirrel suit. Haven't had a chance to try multiplayer yet, but likely as aggravating as ever with the extra fun of letting a fifth person screw everyone up for shit and giggles.

    More later if I think of it.
