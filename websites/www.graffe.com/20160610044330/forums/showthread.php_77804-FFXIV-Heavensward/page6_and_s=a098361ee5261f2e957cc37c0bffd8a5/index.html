    Re: FFXIV: Heavensward

    Hit 53 last night myself, taking my sweet time. Our Free Company bought our workshop yesterday, so some folks have been lost to that new feature I'm afraid to say.

    This expansion just continues to impress. The zones, and the story, gets more epic as you progress and I'm blown away at how much there is and how polished. My biggest criticism of FFXIV prior to Heavensward was the size of the zones and how claustrophobic some of them felt. That's gone... out the window now. These zone are epic in size and scale. The mountains and structures in the Dravarian Forelands are a perfect example of this.

    This is easily becoming one of my favorite expansions of all time.

    Re: FFXIV: Heavensward

    Patch Notes from last night... lots of hotfixes.

    FINAL FANTASY XIV Hotfixes (Jun. 30, 2015)

    The following adjustments have been made:
    Spoiler for Patch Notes:


    • The NPC "Pasdevillet" at The Pillars (X:7 Y:10) now sells the following items:
      Ishgardian Tea / Dravanian Mistletoe / Abalathian Rock Salt / Abalathian Mistletoe
    • The amount of experience points received from the Ixal beast tribe daily delivery quest will now be reduced if the player is above level 50 and the item to be delivered has a recipe level of 50.

    The following issues have been addressed:
    • An issue in Thok ast Thok (Extreme) wherein Ravana would only auto-attack under certain conditions.
    • An issue in The Bowl of Embers: (Extreme) and during the Cerberus battle in The World of Darkness wherein the damage from certain debuffs was was extremely low.
    • An issue in the level 60 main scenario dungeon wherein it was possible to attack monsters inside the cultivation capsules.
    • An issue in the level 60 main scenario dungeon wherein ground-targeting actions could not be used in some locations.
    • An issue in the level 60 main scenario dungeon wherein monsters could not be targeted occasionally.
    • An issue in The Aetherochemical Research Facility wherein some of the bioculture node's actions would have no effect.
    • An issue in The Final Coil of Bahamut: Turn 3 wherein the debuff Cloak of Death lasts longer than intended.
    • An issue in the FATE Prey Online wherein certain monster behaviors were incorrect.
    • An issue wherein the elite mark Bune's actions were not triggering correctly.
    • An issue in the main scenario quest Fire and Blood wherein camera control and field of view did not function correctly.
    • An issue in the main scenario quest Fire and Blood wherein the NPC Tataru was not wearing the correct clothes.
    • An issue wherein it was possible to accept the sidequests The Courage of Nuts and Protecting the Pom at the same time.
    • An issue wherein it was possible to accept the sidequests Who's Got the Baskets, My Friend the Enabler, and Back to Baskets at the same time.
    • An issue in the sidequest Landing a Stable Job wherein speaking to the NPC Proving Grounds Guard as a Disciple of the Hand or Disciple of the Land would cause the selection window popup to appear blank.
    • An issue during the job quest Original Sins wherein NPCs from a different quest would appear.
    • An issue in the job quest The Green Death wherein the map could not be opened occasionally from the Journal and Duty List.
    • An issue in the job quest A Lesson in Patience wherein text would not appear on the screen when acquiring the new action.
    • An issue in the job quest A Flare for the Dramatic wherein unnecessary effects would appear on certain characters.
    • An issue wherein the warrior action Raw Intuition would cause the user to receive critical heals as well.
    • An issue wherein bards would be temporarily unable to perform actions under certain conditions.
    • An issue wherein the white mage ability Assize would not restore MP under certain conditions.
    • An issue wherein the white mage action Divine Seal, which only affects Spells, would also affect Assize, which is an Ability.
    • An issue wherein the black mage action Ley Lines and machinist action Rapid Fire would affect the cast times for actions such as Teleport and Return.
    • An issue wherein the black mage action Fire would not trigger the Firestarter additional effect correctly under certain conditions.
    • An issue wherein the ninja action Suiton would grant the effect even if it misses the target.
    • An issue wherein the dark knight action Dark Arts would apply its effect to multiple actions used in sequence.
    • An issue wherein the machinist action Wildfire was appearing in the wrong category in the damage log.
    • An issue wherein the astrologian action Celestial Opposition would shorten buff durations, such as Well Fed.
    • An issue wherein the astrologian action Benefic would not trigger the Enhanced Benefic additional effect correctly under certain conditions.
    • An issue wherein characters who has unlocked Patch 3.0 duties would receive an error when queuing for Duty Roulette when logged on to a platform that does not have Heavensward play rights.
    • An issue wherein it was possible to queue for Duty Roulette: Expert with party members that did not have Heavensward play rights.
    • An issue wherein materia effects of PvP equipment introduced with Patch 3.0 were active outside of PvP areas.
    • An issue wherein company crests could not be added to certain gear introduced in Patch 3.0
    • An issue wherein the Sightseeing Log would only display Patch 3.0 areas under certain conditions.
    • An issue wherein the quest levels displayed in The Unending Journey are incorrect for certain dragoon job quests.
    • An issue wherein the flame effect from Thok ast Thok would appear upon moving into a housing area.
    • An issue wherein sorting by Auto-attack on the Market Board would cause items to be sorted alphabetically.
    • An issue wherein the item Sable Death Mask could not be discarded if it was purchased from the NPC Calamity Salvager.
    • An issue wherein text commands for the following actions were not recognized correctly:
      JP version: Painflare
      EN version: Ricochet / Scourge / Ascend / Painflare / Prune
      FR version: Reassemble / Ricochet
      DE version: Leg Graze / Time Dilation / Stone III / Fell Cleave
    • An issue wherein certain text was incorrect.
    • An issue in DirectX 11 wherein closing the game would cause an error in certain environments. (Windows only)
    • An issue in DirectX 11 wherein repeatedly pressing the PrintScreen button in Full Screen Mode could cause the controls to freeze. (Windows only)
      * This hotfix is a follow-up to one of the fixes in the Jun. 25, 2015 Hotfixes, which previously did not allow the PrintScreen key to be used on other platforms.
    • An issue wherein pressing Alt + Enter during chat entry would cause the screen to black out and prevent you from playing the game.
    • An issue wherein changing areas while typing text in the macro text edit field would cause the text to remain on the screen. (Mac only)
    • An issue wherein clicking the macro text edit field would prevent mouse clicks from being recognized. (Mac only)
    • An issue wherein the typing carat could not be moved via mouse click while typing in the macro text edit field. (Mac only)
    • An issue wherein changing the screen resolution with the mouse would cause the cursor to not appear in the correct position. (Mac only)
    • An issue wherein attempting to change the screen resolution by selecting the same resolution as before would not cause the confirmation message to appear. (Mac only)
    ​
