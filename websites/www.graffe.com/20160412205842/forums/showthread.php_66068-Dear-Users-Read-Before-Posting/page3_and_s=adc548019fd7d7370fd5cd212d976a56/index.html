    Re: Dear Users. Read Before Posting.

    Moving posts can be a pain, and the point of this forum is to have clear, focused discussion that does not get distracted by personal attacks, sniping, fighting, and various derails that can be entertaining and meaningful but tend to get in the way of decision-making and policy. To that end, threads started here should have a clear, narrow purpose so that participants can focus so that departures from the main topic will be unambiguous. Facts and logic can be analyzed without all the background noise.

    Kaelon's thread is a radical, sweeping proposal that is likely to uncover a many interesting questions and meaningful discussion. The thread did not have a clear, narrow purpose. A thread like Kaelon's is valuable for the discussion it provides, the questions it uncovers and the directions it takes. There is not so much benefit to strict enforcement of topic in a thread like that. You can eliminate fighting and side-threads (like when I called out Maledict) without hurting much, but beyond that the topic requirement is a heavy burden on productive discussion.

    That exactly describes the sort of thread that is worthwhile to have in General Discussion. Not rants. Not Site & Forum Admin.
