  • Site Update ( Graffe has set things in motion ) :)

  1. Just wanted to give everyone a heads up on the outage. On the 3rd, users may have noticed a database error, this was because the system was out of disk space. When I logged in I found several things were eating up the space, and there wasn't a lot I was able to do without a longer maintenance period, but I freed up enough space to get everything back online by adding a bit more disk space. It went down again the morning of the 6th. I added more disk space, cleaned out some stuff, then went into full disk repair mode to repair the system. This allowed me to clean up nearly 30GB of frozen files.

    I also used this time to complete several pending upgrades Linode has had waiting for us. One of those upgrades me cause search results to return a little slower than you would expect. If you see something happen, please let me know on the bug report forum.
