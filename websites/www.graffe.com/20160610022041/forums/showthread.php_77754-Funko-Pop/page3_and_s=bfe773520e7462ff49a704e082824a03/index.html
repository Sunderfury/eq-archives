    Re: Funko Pop

    I start with a Funko DIY blank figure and then use Sculpey modeling clay. This stuff remains soft until you bake it in the oven.

    In the GRRM figure, the hat and glasses were done using Sculpey, but just about everything else was done using a two-part self-hardening epoxy clay called "Apoxy Sculpt".
