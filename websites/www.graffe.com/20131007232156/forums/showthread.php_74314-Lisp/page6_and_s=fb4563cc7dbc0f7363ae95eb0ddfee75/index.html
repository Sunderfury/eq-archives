    Also, RE: keyboard. I think the uTron keyboard is nicer both in general and for learning a new layout as you are not both learning a new layout and learning a different way to hold your hands while typing, and the board has a palm rest which gives a false sense of security for how to maintain the hands while typing. Also, it does not have a fn key by default, which is not good in the mac world, and useful to have outside of that, so the scroll lock key was murdered in its honor
    (KeyRemap4Macbook) and I remapped caps lock to ESC since ESC is a rubber key on the keyboard. Then I changed the system preferences in OS X such that the special functions were mapped to F1-F12 and the real F1-F12 was activated by pressing the Fn Key (Scroll Lock now for me). The uTron keyboard is only purchasable in Japan though, also it is 580 USD (58.5k JP).

    Although not quite as nice of a layout as the uTron, a comparable keyboard is available for only 90 dollars and uses the split keyboard concept still, and has a dedicated function key. http://www.ergocanada.com/detailed_s..._keyboard.html

    The mac layout on the above is:


    The PC layout is:


    Plus, since most of the keys (except for Fn) are a waste of space they can act as a macro bank (similar to the logitech G15 etc. line). The Kinesis contour (brown, linear feel uses red) uses mechanical switches, which I for one am not a fan of. The Freestyle line costs 90-130 USD depending on some details, so it's ~120-150 dollars less expensive and requires a smaller learning curve.

    I increasingly feel that this layout is totally fubar for programmers, which is likely the primary audience, or data entry people possibly.
