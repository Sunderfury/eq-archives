Feature List:

  • Load the Original or New Default colors
  • Choose between different backgrounds (Water, Sand, Grass and Ice)
  • Switch Between Views so you can verify your colors
  • Edit Most Commands stored in EQClient.ini
  • Saves to any file name so you can create custom .INI files
  • Loads your current (or any) eqclient.ini file so you can make changes to your favorite .INI files
  • Loads a custom background that you choose (recommended a 640x480 screenshot from the game in BMP format)

 

Known Issues:

  • Ultimate Editor doesn't display correctly in some Resolutions.
