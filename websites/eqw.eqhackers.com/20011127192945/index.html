EQW is a program that allows you to play EQ in a window rather than fullscreen. This not only gives you access to other programs while playing EQ but allows you to run multiple copies of EQ at the same time. See readme.txt for more info.

Last updated: 9/24/01

Click here to download EQW beta 2.
Click here to download the debug version of EQW which will create a file called "log" that helps me fix problems/bugs.

Send any bug reports to me by email at justsomedude@hotmail.com.
