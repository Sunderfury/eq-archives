§ 色々…… 2001/12〜

SoL 出て日記全然書いてなかったですね……。
Down ばかりなので毎日 ToV North とか、毎日 Tormax だとかそんなのばかりなようです。
11/22 には 1Days で初めて ToV North を Clear しました。翌日休日じゃないととてもじゃないけどやってられません……。15 kill で大体 9hour ですかね。壊滅なくせば 6hour 程度でいけるでしょう。

■spell info
まず vendor spell は Shadow Haven の EQ Atals の No.23 からteleport したところに vendor が。初期 faction では dubiously の場合があるので、sneak を使うか EQ 2001 にある この quest で上げておくといいかも。出るときは真ん中の木をクリック。

次に drop 場所。
- Falcon Eye と Jolting Blades は Fungus Grove のChakta Fungusbeast がよく出します。キノコはもうダメだけど、他のなら出るかな？大体 mob によって出す class が決まっています。caster、healer、hybrid な感じで。
- Mark of the Predator は Maiden's Eye の強めの敵、Ssraeshza Temple の雑魚（temple guard は spell を drop しない）や Akheva Ruins、Deep の奥など。
- Eagle Eye はなかなか出ないけど Ssraeshza Temple の Lv60 から /con blue の named などから出るみたい。あと Greig's End など。
- Warder's Protection は Lv63 以上の mob から。特に Ssraeshza Temple  の隠しの部屋にいる Rhag`Zhezum （/con red） が一番とりやすいけど、Lv60 の2〜3 party で 軽い CCH 必要。かなり堅く、slow 効かない。flurry 持ち。

結局 spell は以上のゾーンでなんでも出るっぽいけど、比較的 mob の種類によって出す spell に偏りがある気がします。

感想。
- Tangling Weeds : 最初は snare と同じ速度低下だったけど patch で変更。速度低下が 35% （snare は max 55%）に落とされました。要するに逃げモードに入って、さらにヒットポイントが減るまでしばらく逃げる。それでも cast time の速さを生かして taunt、pull、snare 忘れた時などに有効。
- Ｈawk、Falcon、Eagle Eye シリーズ : archery の命中率、ダメージアップ。Falcon には see invisible、Eagle には see invisible と ultravision が付く。plane で使えないのは bug？
- Jolting Blades : mana を吸われる敵（Yelinak、Aaryonar など）に対してかなり有効。icon が赤色なのは bug でしょう。self only spell。
- Wardar's Protection : solo 用かな。CoE、SoN、CoP、DS 等々ほとんどの spell と重ならない。使い方次第ってとこでしょうか。

■alternate skill
- general ability : あまりいいのがないので run speed、regene あたりがいいかも。ステータスは日々変化するので、この辺好みかな。

- archetype ability : dual wield 持ちは combat fury がいいかな。rog、monk などは特殊攻撃に critical が付くのでさらに good。point 1 でもかなり critical 出ます。dmg14 の武器で critical max155（通常 max91）くらい。dmg13 の武器でcritical 147（通常 max86） なので大体 1.7 倍になるみたい。finishing blowは趣味かな……（笑）。逃げモードの時に対して約 32K ダメージを与えます。1pt で 1% だけど solo の時は Lv45-50 位の敵で 70% くらい出る感じ。強い敵にはなかなか出にくい感じがする。これは snare 持たない class にお勧めなのかな。
Combat Stability、Combat Agility、Natural Durabilityなど この辺は 3pt つぎ込むと効果が感じられそう。その他 healer系はお好みでしょうか。

- class ability : 良さそうなのが Innate Camouflage、Endless Quiver あたりかな。Endless Quiver は ammo に装備された種類の矢を無限に撃てる。なので 150 range と、8 damage の arrow を 1本ずつ用意しておけばあらゆる場面に対応出来そう。quiver の haste も有効みたい。Mass Group Buff なんて CoP だけのために誰がとるのか謎は深まる……。
