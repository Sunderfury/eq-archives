DPS Chart

DPS (Damage Per Second) = {(Damage x 2) +1} / {(Delay /10)}
Haste Delay = Delay / (1+Haste)

例 : 13/20 の 武器の DPS は？
→ {(13 x 2) + 1} / {(20 / 10)} = 13.5

例 : 13/20 の武器に 60% Haste をかけたときの Delay は？
→ 20 / (1+0.60) = 12.5 ≒ 13

例 : 13/20 の武器に 60% Haste をかけたときの DPS は？
→ {(13 x 2) + 1} / {(13 / 10)} = 20.77

結果
同等の rate の武器でも haste の値が低いときは delay が速いほうが有利、haste の値が高い時は delay が多少遅くても damage が高い方が有利という結果に。これは haste により delay の差が縮まるためと思われます。125% haste で delay 21 → 9.33、delay 19 → 8.44 となり、16/9 と 14/8 の武器で比べていることになります。
