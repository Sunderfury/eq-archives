Advanced Arrows
Shadows of Luclin において新たに導入された arrow です。作成するためにはいくつかの材料を monster から入手する必要があります。まず Shaft を作るための Small Brick Acrylia Ore、Arrowhead を作るための Chunk of Condensed (Shadow/Flame/Ice) を入手する必要があります。これらを組み合わせることによって三種類の魔法ダメージの付いた矢を作成することが出来ます。

Image Color Name Location
Brown / Rock grol baku ~ The Tenebrous Mountains
Bundled Acrylia Arrow Shafts を作るための Small Brick Acrylia Ore を落とします。Large Type も落としますがここでは small のほうを使用します。他に grimling などからも同様に入手できます。
Black / Shadow Shak Dratha ~ Shadeweaver's Thicket
Condensed Shadow Arrowheads を作るための Chunk of Condensed Shadow を落とします。このほかにも The Maiden's Eye や Akheva Ruins などの alien（エイリアン）や robe（Xi Vius）などからも入手できます。
Red / Fire tro jeg ~ The Scarlet Desert
Condensed Flame Arrowheads を作るための Chunk of Condensed Flame を落とします。tro jegs は他のaliens（エイリアン）と敵対関係にあり、他の 3 faction を上昇させることも可能です。
Blue / Water vas ren ~ The Twilight Sea
Condensed Ice Arrowheads を作るための Chunk of Condensed Ice を落とします。Twilight Sea の シークレットエリア（EQ Atalas の No.1 参照）全般を巡回しています。このエリアに行くためには Enduring Breath の魔法が不可欠です。

まず shaft を作るための材料を forge で combine します。skill は smithing で trivial は 122 です。Arrow Shaft Mold は Katta Castellum （EQ Atalas の No.7, Forge は No.5 参照）
  • Small Brick Acrylia Ore + Arrow Shaft Mold + Water Flask
    = Bundled Acrylia Arrow Shafts x 2 + File

次に arrowhead を作るための材料を forge で combine します。skill は smithing で trivial は 135 です。
  • Chunk of Condensed Shadow + Water Flask + File
    = Condensed Shadow Arrowheads x 2 + File (return)
  • Chunk of Condensed Flame + Water Flask + File
    = Condensed Flame Arrowheads x 2 + File (return)
  • Chunk of Condensed Ice + Water Flask + File
    = Condensed Ice Arrowheads x 2 + File (return)
材料が完成したら fletching に移るわけですが、combine するための材料が限定されています。各（shadow / flame / ice）の arrowhead と 作成した Bundled Acrylia Arrow Shafts、残りの二種類が Several Shield Cut Fletching と Small Groove Nock のみ使用して fletching kit で combine することになります。Trivial は 240 前後で一度に10本の矢が出来ます。

Vendor Arrows
  • 上の表を見てもらえば分かりますが、店売りの矢よりFletching Skillで作成した方がより安く、性能のよいものが出来るのが分かります。Flight Arrow（1dmg/150range, 210cp/1本）に対してCLASS 2 Wood Point Arrow（1dmg/150range, 15.6cp/1本）で作成できるのです。


Magic Arrows
  • Point（鏃）に "Silver Tipped Arrowhead" を使用した物にMagic属性がつきます。ghoul やwillowisp 又はdragonなどのMagic属性がないとダメージを与えることが出来ない敵に対して使用します。pull のだけのためなら特にMagic属性を使う必要はありません。ダメージを与えることが出来なくてもhateを与えることは出来ます。Silver Tippedを使用した物だけは一度に10本の矢が出来ます。その他の物は5本ずつしかできません。このほかにTolan's Drakwood Bracerのeffectや、MigicianによるSummon ArrowもMagic属性がつきます。


Best Arrows
  • Trueshotなどを使うときにおすすめするのが、 "CLASS 2 Steel Silver Tip Arrow" です。これはrangeが150の矢の中で最もDamageが高い物です。しかし一本あたり7pp近いこともあり、Costを抑えるなら "CLASS 2 Wood Silver Tip Arrow" あたりがよいでしょう。これはRange150のArrowとLong RangeのBowを組み合わせることにより、敵のArea of Effect魔法やブレスの範囲外から攻撃することもが出来るからです。逆に言えばArea of Effect魔法のない敵や、十分な距離をとれない場所では、Summoned Arrowで十分効果を発揮します。またpull のだけのためなら "CLASS 2 Wood Point Arrow" あたりを使うのがよいでしょう。


Classification
  • 矢の種類はClass1やClass2といったように、使われた部品に応じて分類されます。基本的に使用したFletchのタイプによって分類されています。Round Fletchを使用した物はすべてClass1となり、Parabolic Fletchを使用した物はすべてClass2となります。ただこれはFletch以外の構成部品によって性能は大幅に変わるので、一概に性能だけを示しているとは言えません。
