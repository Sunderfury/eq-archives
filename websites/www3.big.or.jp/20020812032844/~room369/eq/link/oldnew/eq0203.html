§ Patch Day

Halfling Ranager がいっぱいな今日この頃です（笑）。だって性能いいもんねぇ。

melee filter が導入されたわけですが、なかなか設定が難しいですね。

/hitsmode <my|me|other> <on|off|m2|m3|m4|m5> <X> <Y> <COUNT> <FONTSIZE> <HEADER> <BG> <WIDTH>

/hitsmode my m2 600 52 5 1 0 1 300
/hitsmode me m2 296 52 5 1 0 1 300

1024*768 環境ですがこれが上部に表示させるようにした感じです（例です）。my（自分の与えるダメージ表示）、me（自分が受けるダメージ表示）、other（他人の攻撃表示）は上のように個別に設定します。この window は自分が殴ったり攻撃を受けたとき表示され、しばらく何も起こらないと消えます。

元のチャット表示に戻すときは、/hitsmode my on とかにすれば OK です。

<on|off|m2|m3|m4|m5> on は 元の chat window で表示。off は 完全に表示 off。m2 は別 window で通常表示、m3〜m5は以下のような感じで短縮表示。
  • m2 : You slash a golem for 50 points of damage.
  • m3 : slash a golem for 50
  • m4 : 50
  • m5 : slash 50 （my のみ）
<X> <Y> は window 自体をクリックで動かすことが出来ないので eqclient.ini などを見てある程度見当をつけるといいでしょう。

<COUNT> 1-10 : 表示行数です。5 あたりが適当でしょうか。

<FONTSIZE>　0-2 : 文字サイズ。この設定を変えても window の高さや行数自体に影響ありません（行間が空いたり詰まるだけ）

<HEADER> 1=on/0=off : は攻撃が始まったり、ダメージ受け始めると、一番上にその旨が表示されるだけ。行数食うだけかな。

<BG> 1=on/0=off : on で chat window と同じの background color が適用されます。off で透明。

<WIDTH> その名の通り幅の設定。m4 あたりの設定だとかなり狭くて OK かな。

とかややこしいですね。自分も何パターンか作りましたが実際殴られているときは DS メッセージは表示されるので相変わらず見にくいです。
