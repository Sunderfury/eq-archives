Zoldak Ragefireについて(2002/09/04まで有効)

CLR epic最後のパートであるRagefireですが2回目のパッチが当たって その舞台は現在はSkyfire Mountainへ変更されています。 儂は運良くepicを手に入れることが出来ましたが、パッチ後にepicを 取ったJPNは非常に少なく、E'ciでは恐らく儂が3人目だと思われます。 その原因はpop timeが不明確である事、pop即killという状態が 続いている事が大きな要因と思われます。
したがって一人でも多くのJPN Clericがepic holderとなるチャンスを 多くしたいと思い、campした結果から判っている事をメモとして残しておきます。

Ragefireの強さ

まずRagefireの強さを簡単に紹介しておきます。
SolB時代と同様にRagefireは最初にHuman typeがpopして、これをkillすると Dragon typeがpopします。
Human type : Lv55 Cleric Type 弱い(SolCのPlasmaticPriestと同じ外見です) KoS for ALL
Dragon type : Lv60 AE CM(1個消し), AE Fear, AE FireDD(max500) Enrageあり KoS for ALL

これをkill出来る戦力はLv60のフルパーティで最低2group、4groupで余裕でしょう

Spawn Time

さて、気になるpop時間ですが、儂が今までに調べた&経験から以下のように まとめる事が出来ます。多分ほぼ当たっていると思います。

■server down後の最初のpop

server upから24時間経過後の24時間、つまりserver upから24時間〜48時間の間にpop
↓
server upから24時間〜72時間の間にpopという可能性も出て来ています

■server down後の最初のpopをkillした以降(通称2nd pop)

kill以降、120時間経過後の40時間(一説には50時間)つまり、 1st kill以降の120時間〜160時間(170時間かも)の間にpop

以上のようなspawn timeとなっているようです。
かなり長い揺らぎがあることが判ると思います。
そしてserver down直後を狙うのが効率がよいことも判ると思います。

server down直後を狙うのでなければkill timeを常に把握しておく必要があります 最近のchat channelシステムのせいかRagefireがkillされてもoocやshoutで なかなかcallがあがらない傾向がありますので厳しいかもしれませんが。。。

Heartのloot権利

現在のE`ciではRagefireに関してはcamp宣言は無効であり、 早い者勝ち(First Come, First Served)となっています。 これは即ち、いくら長時間campを行ってもRagefireがpopした時点で ほかのcamperに取られる可能性があるという事です。

別サーバーでのGMの見解を取れば次のような条件をクリアしたものが Ragefireのheartをlootする権利があることになります

・Human TypeのZoldak Ragefireとの1st Engage(戦闘状態に入っていること。kitingは無効)
・Human TypeのZoldak Ragefireをkillして経験値を獲得すること
・Dragon TypeのZoldacalicus Ragefireをkillできる戦力を集結していること

この条件ゆえに早い者勝ち(First Come, First Served)となるわけです。 したがってその競争率の高さからも単に監視キャラをおいて置けばよいというわけではなく 他にcamperがいる場合にはpopをある程度見越してあらかじめ戦力を確保しておき、 他のcamperの戦力を常に上回る状態にしておく事が必要になると思います。 時間帯にも拠りますが基本的にpopしてから戦力を集めるのでは遅いと思います。 最低1パーティはpop point付近で周りへのアピールを兼ねてcampしている必要があると思います。

Campの仕方

1パーティもしくは他のcamperより多くの戦力を常に確保しておいて Human typeがpopしたらMTがある程度HPを削っておいて可能であれば ENCでmez keep(RaptureかKintaz?Kintazは無理かな?)しつつ 戦力を増強しbuff up等を終えて一気にkillが良いと思います。 もしまわりのcamperが手を出してきたら時間を稼ぎながらのkillで 戦力が集まるまではDragon typeをpopさせない事が重要だと思います。

実際問題、監視キャラでピンポイントのpop checkでは確率が かなり低いので、ある程度山をはっておく必要があると思います それでもserver up後の1st popであればゆらぎが24時間ですから 見張っている事はある程度可能だと思いますし、あらかじめ 戦力を予約しながら交代でpop pointでのcamp keepも良いと思います

儂が行った方法はserver up後の24時間〜36時間くらいは時間的にJPNが 少ない事などからpopしてから戦力集めて間に合わない場合は次回を狙う 方針でただ見張っているだけでしたがその後の36時間〜45時間くらいは 知り合いに頼み込んでcamp keepしてました。 もっとも10時間ちかいcampが可能だったのは休日だった事も幸いしたからで これが平日だったらかなり難しかったかもしれません
