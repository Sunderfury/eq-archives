EQInside Version 0.2 beta:
  • Spell section added to EQInside with following features:
    • Mana required
    • Skill required
    • Spell messages
    • Casting / Recover times

    • Experience level required

    • Character class

9-21-1999
EQInside Version 0.13 beta:
  • "File"->"Save..." and "File"->"Save As ..." menu items now compress and save modifications to archive file you made. Be careful, as before it was saving archive entry to the disk.
  • "View"->"Tips" menu item shows/hides tips on the bottom of the window.
  • "Archive" menu contains commands to modify EverQuest archive:
    • "New" will add new entry to the current archive. You may then use "Load Data" command to load data for this entry from the file.
    • "Add" will add new entry to the current archive and load it from the file.
    • "Delete" will delete selected entry of the current archive.
    • "Rename" will rename selected entry of the current archive.
    • "Save Data" will save current entry to the file. This is the same function as "Save As ..." in previous versions.
    • "Load Data" will load data for selected entry from the file
    • "Update CRC" you may update CRC code for this entry. Currently I do not know how to calculate this number not even if this is really CRC for this entry or some other information used by EverQuest.
EQInside DevKit Version 0.13:
