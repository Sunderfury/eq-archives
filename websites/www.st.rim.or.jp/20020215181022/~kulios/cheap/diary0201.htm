<<< MMORPG >>>

　HagiとICQで話しているうちに、不満だったことがよりはっきりイメージできたので、それについて軽く書いてみちゃおう。

　/* What's MMORPG */

　ここを読んでる大半の人には全く無意味な説明である。

あぁ、飛ばせばいいのか


　大変良く理解してくれたことを期待しながら次へ。

　/* 最近色んなCloneが出てるけど、どうなの？ */

　EverQuestの成功後、結構な数のCloneが発売され、また今後さらに発売されようとしている。現在βテスト中のFFも稼動マシンがPS2（実はこれが大きな違いなのだが）であることを除けば、間違いなくEQ Cloneの一つである。

　Cloneであることには利点も多い。インターフェースについて一から覚え直す必要がないし、そのうちの一つに慣れ親しんでしまえば他のゲームでも全く戸惑うことなく参加が可能だ。これ自体はゲーム業界では当り前のことで、例えばスト２の成功後、幾つかのパンチ・キックボタンを利用して攻撃、後ろへ移動することでガードというのは以後多くの格ゲーで共通フォーマットとなった。そのタイプのゲームを好みとする顧客への間口を広げる為の当然の戦略であろう。
　例えばＴＶ一種類毎にマニアックなリモコンがついていたらどうなるかを考えて欲しい。

　問題となるのは、これら多くのEQ Cloneがインターフェースだけではなくシステム面までもがくりそつな事だ。例えば
  • 経験値ベース。LevelによるPowerUP（Skillはあったりなかったり）
  • 種族・職業有り
  • 魔法有り
  • Party制（自分の弱点は他人が補うことでParty制への依存度Up)
　まぁ最後のParty制を除けばCRPGはそんなものだって話もありますがそれは棚上げ。各ゲームが上記基本フォーマットにあまりに忠実なため、画面に表示される画像を除けば、とどのつまりやってることは再放送。ケアルだろうがホイミだろうがMinor Healingだろうが

cure light wounds


　にすぎません。↑の例が一番User数の少なそうなやつに収束させるのはどうかって突っ込みは有りですが、各自自分の中で適当に置き換えて下さい。

　/* そんなこといったら、映画とか小説は？ */

　そうです。これらはゲーム以上にフォーマットが同じわけですが、それにも関わらず飽きると感じることはほとんど無い。この部分が今までうまく文章化できなくて困っていたところ、ようやく昨日形になりました。それが以下の内容。

「強くなるほど弱くなる」


　最初は同レベルのモンスターを一人でも適当にあしらえたのに、なんで最高レベルになるとそれが出来なくなるのか？

　これには、①RPGであること②Party制度があることの二点が大きく関わってきます。

　まず①ですが、所詮CRPG。Program上で用意された限界を超えて成長する事は出来ません。となると、販売側は単一賞品での寿命を稼がねばならないので最初は興味をそそるべく成長を軽快に容易に行わせますが、ある程度客を捕まえた辺りからその成長速度を送らせます。当然ですね。強くなりたい！というのは顧客の欲求の基本部分です。その刺激が無くなってしまっては商品としての魅力に大ダメージが発生しますので、最高レベルに到達させてはなりません。かといって、いつまでも到達しないわきゃないので、適時追加Programを導入することで商品の寿命を延命させるわけです。現在のEverQuestがこれに該当します。

　次に②です。Party制度を作った以上、Partyを組んでもらわなければ困ります。ならば需要が発生するように、システム的に誘導してやればいいわけです。例えば、成長に伴う役割分割であったり職業毎の能力格差の拡大です。レベルの低い初期段階では差もそれほど大きくない為、どんな職業で何をしようとそれほどの差は発生しないが（各ゲームでのバランス設定に依存しますが、概ねこんな感じでしょう）、成長に伴いその差は歴然とし始め、次第にPartyでの行動が不可欠となり、最終的には各クラスとも特化された部分以外は役に立たなくなったりします。

　これが、強くなったにも関わらず、個としての能力は逆に弱くなっていくという矛盾の原因です。勿論、一応能力は上がっているので格下の相手に対してどんどんと強力になりますが、一度負けた敵は雑魚という少年漫画の法則を考えたり今更、富樫や虎丸に強くても全く意味が有りません。しかしいつも通り例がアレですね。



　奇妙なパラドックス。

　ただし、これはあくまで個という視点からの事象であり、群という視点にたった場合は個が強くなることは今まで通りのレベルアップからなんら変わる部分を持ちません。逆に言えば、違和感を感じる者は群では無く個として遊戯していたプレイヤーです。

　ようやく自分の抱いていた違和感というか不満がはっきりしました。

　なるほど、つまらなくなった理由は自分の協調性の無さとか社会性の無さが原因だったわけです。

　だからおいらは再びマジックへと戻りました。己の力が全てである競技へと。で、戻ってみたものの。今ひとつ構築だと燃えないなぁっていう理由が今日の日記で解りました。構築は既に個人戦じゃなくなってきているからです。逆にドラフトは完璧な個人戦です。ドラフトを行うのも対戦するのも完全に己の腕一つで闘うレギュレーションです。なるほど、こりゃ燃えるわけだ。

　珍しく、なんか日記っぽいじゃん！てな感じがしますね。


　今後も真摯な哲学サイトとして頑張っていく予定は特になく、ネタが無かったわりには結構書けるもんだなぁと自分に乾杯。

Torment Card Review : Mutilate

Name Mutilate
Casting Cost BB2
Type Sorcery
Effect All Creature get -1/-1 until end of turn for each swamp you control


　見ればすぐ解るように、効果の程は沼の数次第。キャスティングコストにも黒黒が含まれているため、タッチでの投入は考えずらい。最も効果を発揮するのは黒単においてであるが。

　類似カードで比較対象となるのは、Wrath og God、Pernicious Deedといったところ。このカードの利点は
  • 除去域を（置いた枚数で）制御可能
  • 再生クリーチャーを殺せる
  • クリーチャー以外には影響無し
　逆に欠点は
    効果が沼の枚数に依存する クリーチャーしか殺せない
　元々黒はクリーチャー除去が豊富な色、正確にいうと生もの以外殺せない色。この色に投入される除去は、数多くの既にある除去と性能勝負を挑まなきゃいけない宿命があったりするわけです。

だから、Flametounge が壊れてるんだってば


　勝負はInvasion Blockが落ちてからってことで一つよろしく。

Torment Card Preview: Cabal Coffers

Name Cabal Coffers
Casting Cost -
Type Land
Effect 2,Tap:Add B to your mana pool for each swamp you control.


この土地には３種類の使い方がある。
  • ①Mana Boost
  • ②Change to Black Mana
  • ③Mana Reduce
　まぁ③は需要が無いので放置する。勝手にマナバーンでもしといて下さい。次に②。例えば、沼沼山山これと５枚のランドが有った場合、最初に沼沼からＢＢ、続いて山山これで更にＢＢと沼を使わずとも黒マナを出す使い方だ。でも、そもそも沼が無いとマナすら出ないこの土地をこんな使用法の為に導入するのは働けど働けど。

　となると、やはり①の用途で使うのが最も理性的といえそうだ。では、そのブースト能力はいかがなものだろうか。

沼沼Ｃ沼沼沼Ｃ沼沼沼沼Ｃ沼沼沼沼沼Ｃ
黒ｘ２黒ｘ４黒ｘ６黒ｘ８

　うーん今一つ。ブーストを行う為には最低４枚の沼が必要になるわけだ。それ以降は沼が一つ増加する毎に黒マナが二つずつ増える計算。

　出番があるかどうかは環境次第といったところか。序盤のデメリが気にならないほどゆったりたっぷりの～んびりとしてればいいんだけどねぇ･･･

フードバトルと総合格闘技

　決勝ラウンド。

　勝ち上がった三名は、

　「地上最強の胃袋」　小林尊
　「大食い大魔人」　白田信幸
　「伝説を作る男」　山本昇也

　 　初代王者小林と山本は早食い、一方二代目王者白田は大食いを得意とする選手。このそれぞれ違ったフィールドを背景に持った男達の間で勝利を決めるために選ばれた競技内容が
カレーライス　10Kg　早食い


　…

　えーと、カレーハウスCoCoの看板メニューに1300gチャレンジってのがあります。1300gを食べるのは結構大変な事なんですよ？　多分おいらは途中でぐったりしてしまうと思うぐらいのかなり馬鹿げた量なわけですが
10kg ← 1300gを７杯以上


　おかしいだろ(;´Д｀)

　試合開始前の煽り文句の中に以下のようなものがありました。

「大食い選手はどこの時点で大食いに持ち込むのか」

　どこの時点も何も量は決まってるんだから大食いも糞もないんじゃないかなぁ。しかもです、このカレーというメニュー設定が怪しい。これは前回のフードバトルにおいて小林が小林食いをかました彼の得意ジャンル。そう、つまりこれは小林君素敵～（ハート）なＴＢＳが彼に勝ってもらうために作った競技に違いない。ちぇー、やっぱブッカーには勝てないやぁなどと思った次第。

　ここにフードバトルと大食い選手権の違いがあります。

　後者の場合、決勝は大抵一人の選手以外がもう何も食えません状態に突入して勝負が決します。一応時間制限はついているものの、それは大食いという主軸になんら影響を及ぼすものではありません。

　一方のフードバトル。後進であるこの番組は大食いの中に早食いという要素を絡めています。それに加え、各選手を無理矢理キャラ付けしていくプロレス展開により視聴者のハートをギュッ。勿論マッチメイクもプロレスしていて、準決勝６人中の上位３人がきちんと別ブロックに収まったのは見事というかアレな気もしないでもないですが、そうしないともったいないし、むしろ見てる方もそれを望んでいるもんだから全く問題なしです。これぞプロレスです。

　ただし、ルールは別でしょ。格闘技の世界において勝負その時までルール不明な他流試合は存在しません。ありえません。そんなんではそもそもどう何を練習していいもんだかすら、全く解りません。しかしフードバトルではその時までルール内容は伏せられているのです。少なくとも視聴者にはそう見えます。つまり、もしえこひいきルールを作りたければ、いくらでも可能な状態なわけです。贔屓の選手がいる場合は、そいつに有利なルール構成にすればいいだけ。打撃系に勝たせたければ寝技に制限を入れラウンド時間を短く、寝技系に勝たせたければ打撃に制限を入れラウンド時間を長く。

　滅多やたらとどんなルールでも勝っちゃう奴なんていうのはそういるもんではありません。大抵の場合はそのルールに強かった人間が勝利するのです。かつて総合格闘技においてグレーシー一族が強かったのは彼らがそのルールに慣れ親しんでいたからです。ルールが勝者を選ぶのです。
でも　白田が圧勝
