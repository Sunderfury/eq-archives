　/* いい娘いるよ */

　この言葉の中での最大疑問要素は「いい」。さすがにいくらなんでも生物学上娘じゃないとかそもそも怖いお兄ちゃんしかいないってのはぼったくり的にも難しそうです。となると評価対象は主観的意見に過ぎない「いい」の一点。

　/* 海外正規販売 */ 　まぁ「海外」はいいとしましょう。きっと「販売」も間違いないと思います。売ってるし。ただねぇ「正規」はいただけません。「いい」は主観が入りこむ余地のある表現でしたが、「正規」にはそんな曖昧な感情・判断の入りこむ余地など微塵も。

　個人勝率が再び70%まで復帰。実は上記の他に2-0のまま保留になってる試合と1-1で勝った試合があるんだけど、例によってまた忘れたので割愛。それ含めるともうちょい成績上がるんだけどなぁ。

　さて、ちょっとマネドラについて再度まとめてみよう。

　/* What's マネドラ？ */

　2 vs 2 の４人で行うブースタードラフト。ドラフト後、各々が対戦チームの二人と試合を行う。つまり基本で４試合が行われるわけ。その成績結果が4-0、3-1となった場合は勝敗が決定。もし、2-2とevenの場合のみ、各チームから一名を選出し大将戦を行って勝負を決する。

　/* マネドラ基本戦術 */

　マネドラは通常４人で行う。（変則的に 3 vs 3、4 vs 4 でやる場合もあるけどね）これが何を意味するか、軽く解説してみよう

１２パックしか開けないぞ


　８人ドラフトであれば２４パックが開封される。この差は大きい。僅か１２パックでは総カードプールの偏りが激しく、全く出やがらねぇコモンカードもばしばし存在する。更には選んだカードの除去が全くなくてまじで大発狂モードになることも。

一枚取ると、三枚取られて戻ってくる。


　４人が順番に一枚を取るのだから当たり前っちゃ当たり前。だが、この基本原則を理解することなしにはマネドラでの成功はおぼつかない。８人ドラフトが流れに身をまかすゲーム（おいらの中ではね）であるのに対して、４人ドラフトは狐と狸のコントロールゲームなのである。

　まず、おいらのやり方を簡単に紹介しよう。これはあくまでおいらの場合であって、人それぞれ自分なりの戦略を考え練習し、こっそりと運用することが重要である。従って勿論おいらも全てを説明するわけでは無いし以下には嘘が含まれているかもしれない。自分の金には自分で責任を持とう　=D

　/* ドラフト指針の作成 */

　まずは開けたパックを十二分に吟味する。特に大事なのは各色でデッキに入る枚数を確認した上で、どのカードを取るとその後３人がどのカードを取るかを考えることだ。開始から数パックの間は各自が自分の色を決定する重要な時期であり、その決定材料に対しいかに味付けを加えていくかは非常に重要な事である。

　/* ２〜４手目 */

　この間では最初に作成したドラフト指針に従い、他人の手が加わったカードプールの中から必要なカードをピックしていく期間である。この間では、他のメンバーが何をしようとしているかを推測する時期でもあり、各色の人気度合いを把握することに努める時期でもある。大抵の場合、混んでいる色を選択してもゴッドデッキは作成しずらい。それよりも人気薄の色を素早く見極めよりライバルの少ない楽なドラフトを行うことこそが肝心である。
