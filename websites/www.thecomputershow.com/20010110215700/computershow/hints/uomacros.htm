Ultima Online Macros

Macros

  1. The macros system based in the alt-o options menu on the macros tab, allows you to add an almost infinite number of macros to the bottom of the macros list. Simply click on the empty macro at the bottom and add a macro letter, select any combintion of alt, shift, and ctrl, select from the first box selections, such as use skill, use last skill, etc., select skill or spell in the case of spell book and you are set.
  2. Macros are especially useful when training, when standing around you can be hiding, inscribing, cutting wood and a number of other things simultaneously. For example, the peacemaker bard spell and the music skill can be practiced at the same time.
  3. program your mouse button, so that you can use the same macro to repeat the last action so you can do repetitive tasks more easily without remembering or using the many macros needed for each skill and spell.
