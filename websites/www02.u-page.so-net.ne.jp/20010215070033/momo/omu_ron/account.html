アカウント

　EverQuestをインストールすると左の様なアイコンが２つ出来ます。

　１つがEverQuestの起動、一つがチュートリアルの起動です。

　チュートリアルは初心者のためのEverQuestの入門講座みたいな物

　です。


　→

　EverQuestを起動するとまずこの画面が出てきます。

　まぁ、お決まりの定型文ですね。

　「ＩAccept」をクリックして先へ進みましょう。


これが最初のメニュー画面です。

・CONNECT　Loginする為の画面に進みます。

・ACCOUNT　アカウントの登録、修正等へ進みます。

・CREDITS　制作に携わった人の一覧を表示します。

・OPTION　オプション画面へ進みます。

・EXIT　終了します。





これがアカウントメニュー画面です。

・NEW ACCOUNT　アカウントの新規作成です。

・CHANGE ACCOUNT　アカウントの修正です。

・CANCEL ACCOUNT　アカウントのキャンセルです。

・ACCOUNT HISTORY　アカウントの履歴?です。






NEW ACCOUNTをクリックするとこの画面が表示
されます。

・Last Name　名前を入力します。(例:Tarou)

・First Name　名字を入力します。(例:Yamada)

・M.l　表示が?なんですがミドルネームの事かと..
　　　 入力しなくても問題無しです。

・Address　住所を入力します。
　　　　　東京都千代田区秋葉原1-2-3-456の場合
           Akihabara1-2-3-456 Chiyoda-ku
           Tokyo-to Japan で良いと思います(^^;

・City　　上記の例だとChiyoda-ku

・State/Province　上記の例だとTokyo-to

・Zip/Postal Code　郵便番号を入力します。

・Country　上記の例だとJapan

・Phone Number(Day)　日中連絡可能な電話番号を入力します。

・Phone Number(Eve)　夜間連絡可能な電話番号を入力します。

・Email Address　e-mailアドレスを入力します。

・Where do you attend Work/School?　たぶん社会人か学生かを聞いてるのかと...WorkまたはSchoolで良いかと(^^;

・Date of Birth MM/DD/YYYY 誕生日を月/日/年の形式で入力します。

・Sex　性別です。男ならMALE、女ならFEMALEをチェックします。

・Are you a member of the Press?　記者ですか?と見たけど自信なし(^^;

・Are you a game Developer?　ゲーム制作者ですか?と見たけど自信なし(^^;




アンケートの画面です。

・Where did you hear about EverQuest?　EverQuestに関してどこで聞きましたか？

・Processor Type　CPUの種類は何ですか?

・Processor Speed　CPUの速度はどれ位？

・System RAM　メモリはどれ位？

・3D Card Chipset　3Dビデオカードのチップセットは何ですか？

・3D Video Card Texture Memory ビデオカードのメモリーはどれ位？

・Sound Card　サウンドカードは何ですか？

・Internet Connection Speed　インターネットへの接続速度は？

・Internet Service Provider　プロバイダーはどこですか？

・2D Video Card　2Dビデオカードは何ですか？

・Game System Owned　ゲーム機はどんな物を持ってますか？




アンケートの続き兼重要な項目の入力画面です。

・Favoritte Game Type　お気に入りのゲームの種類は？

・Do you wish to receive Promotional Material EMAIL？　お知らせをe-mail
　で受け取りますか？と見た(^^;

・Choose a Chat Handlle　チャットのハンドル名は？

・Enter Account Key　CDケースの裏にあるアカウントキーを入力して下さい。
　　　　　　　　　　０(ゼロ)とＯ(オー)の違いに注意。私はしっかり間違えました(爆)

・Station Name(Login Name)　任意のLogin名を入力して下さい。

・Station Passward　任意のパスワードを入力します。

・Re-enter Station Passward　入力したパスワードの確認です。


以上でアカウント作成の終了です。
後は支払い方法を選択する位かな(^^;
