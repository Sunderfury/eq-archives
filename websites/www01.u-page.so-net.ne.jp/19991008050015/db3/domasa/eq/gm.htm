ゲームマスター

　ゲーム中、外部の助けが必要となる場合もあるでしょう。時によって必要なものは、情報・援助・「神性奇蹟」といったさまざまな形式をとるでしょうが、いずれの場合も、 EverQuest の支配人である Game Master を呼ぶのがいちばんです。

　ただし、 Game Master は道案内をしたり、死者を蘇生したり、特殊な願いをかなえるために存在しているわけではありません。 Game Master の目的は、みなさんが予定外の不具合による妨害なしにゲームを楽しんでいただくことです。 Game Master によって持つ権限の代償に違いがあるため、状況によっては、より上位の Game Master に問題の解決を委ねることもあります。

GM とのコンタクト

　Game Master を呼ぶには、 "/petition" と入力してから、問題の要約を入力してください。 "/petition" によるメッセージはすべての Game Master に送信されます。

　さて、時には数千人のプレイヤーが同時にプレイしている可能性もあり、 Game Master の元にはプレイヤーたちの要求の津波に呑み込まれているかもしれません。それでも、 Game Master たちは受け取ったメッセージを整理し、可能なかぎり早くヘルプに向かいます。時間がかかることもありますが、辛抱してくださるようお願いします。 "/petition" 連打はお止めください、それは髪の逆鱗に触れる確実な手段です。

　コンタクトがとれたら、自分の状況を説明してください。必要に応じて、 Game Master からさらに詳しいことを伺う場合もあります。落ち着いて、礼儀正しく、また問題を正確かつ詳細に説明してもらえると、 Game Master としては非常に助かります。とにかく、しばらくの辛抱をお願いします。問題は、ゲームの支配者の手に落ちればすぐに解決するでしょうから。
