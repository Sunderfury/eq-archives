原題: Beginer's Guide
著者: Lazarus
編集: Cold Rain

導入

　コンピュータに詳しくない人、 EverQuest に詳しくない人々のために、 EverQuest をはじめてインストールし、キャラクターを作る際に戸惑わずにすむよう、簡単で乱雑なチュートリアルを書いてみました。

　このチュートリアルは、オンラインゲームをはじめて比較的に日が浅く、巨大なマルチプレイヤー環境に不慣れなプレイヤーを対象に想定しています。初心者にありがちなミスを犯すことがないように祈っています。

　このガイドの内容は、インストールから始まり、 EverQuest の世界に十分に熟達するまでの間を取り扱っています。

キャラクター作成

　キャラクター作成時、多くの人が次のような質問をしてきます。「私は何をプレイすればよいのですか？」　さて、この質問は無邪気な質問のようで、実に深い質問です。プレイするキャラクターによって、ゲームのプレイスタイルだけでなく、実際に行くことができる場所にまで影響が出るからです。例えば、ヒューマンのガードは、トロール、オーガ、ダークエルフのキャラクターを目にしたとたん攻撃してきます。このゲームで何をしたいのか、何を目標とするのか、どんなロールを演じたいのか、よく考えてください。そうすれば、自然と作るべきクラス/種族が判明するはずです。

　クラスと種族を選ぶと、次に名前をつけます。 KeWlDoOdZ なんていう世界観にそぐわない名前を考えたりせずに、ゲームに用意されているネームジェネレーターを使いましょう。 name ボタンを押せば名前をつけてもらえます。自分の気に入った名前が出てくるまで何度もクリックしましょう。 Superflykewljoe とかいった名前をつけても、ネームフィルタによって「空想的」でない名前ははじかれてしまいます。また、そのサーバにすでに存在する名前をつけた場合もはじかれてしまいます。 GM の活動のひとつに「空想的でない名前を変更する」というのがありますから、どうか空想的な名前をひねり出してください。 Brucelee という名前のモンクが、ある日ログインしてみると名前を変えられていたという事例もあります。

初飛行

　キャラクターは完成しましたか？　それでは、 Enter World ボタンをクリックしてください。時間がかかることもありますから、コーラでも飲みながら待ちましょう。飛沫が画面から消えて明るくなったとき、３Ｄの世界にいる自分を発見できるはずです。せわしない日々の生活とは違い、 NPC も PC も街中をのんきに歩き回っていることでしょう。私の場合、まず最初にホットキーの設定を行います。画面左下を見てください。１から６の数字がふってあるボックスがありますね？　ここに、行動、スキル、発言、 「Presonal」 などのショートカットを設定することができます。ひとつの bank に置くことができるボタンは６個で、 bank は６つあります。では、実際にホットボタンを設定してみましょう。ホットボタンにしたいボタンを左クリックし、そのままボタンを押しつづけてください。約１秒後、そのボタンがマウスカーソルの上に乗った状態になります (浮き上がったボタンの左上端がマウスカーソルの先端にあたります。 bank に登録する際に重要なことですから覚えておきましょう) 。次に、登録したい場所にまでボタンを移動させ、もう一度左クリックします。こうしてボックスの中にボタンをドロップしておけば、該当する数字キーでそのボタンを使うことができるはずです。例えば、 Bash スキルを #4 のボックス登録すれば、以後４キーを押すたびに、ターゲットしている NPC を盾で殴りつけます。**** 注意: このゲームの NPC について、モンスター NPC と 商人 NPC の区別はありません。ターゲットして上のボタンを押せば、人種差別することなく殴りかかります **** 別の bank に切り替えたい場合は、 Shift キーを押しながら 1-6 の数字キーを押してください。いちいち Shift キーを使って切りかえるのが面倒なときは、ホットキーボタンの上にある矢印をクリックしましょう。非常によく使われるボタンとして、 Personal ボタンが挙げられます。これは、インベントリーを開くのに不可欠なボタンで、所持金や所持アイテムを調べたいときは必ず押すことになります。登録しておきましょう。

ホットボタンの設定の次に、さっそくインベントリーを覗いてみましょう。 Pesonal ボタンをホットボタンに登録しているのなら、さっそくそのボタンを押してください。画面右上にあなたのキャラクターの画像が表示されたウィンドウが開きます。インベントリーを調べている間も、周りの出来事を眺めつづけることができます。キャラクターの画像の上にあるボタンを押すことで、インベントリースロット画面を表示されます。この画面では、現在何を装備しているかを見ることができます。この画面でさらにボタンを押せば、現在持ち運んでいるアイテムのリストも表示することができます。

　あらゆるキャラクターは基本的なアイテムを必ず最初から持っています。選んだ種族/クラスによって具体的なアイテムは変わりますが、初心者用の武器がひとつ、２通の手紙、５つの食料と水を持っているはずです。２通の手紙のうち、ひとつは PK note でもうひとつは guild note です。どちらも、右クリックで読むことができます。基本的に、インベントリーにあるアイテムを右クリックすれば、そのアイテムの詳細な情報をえることができます。左クリック一回でアイテムをマウスカーソルに拾い上げます。また、インベントリーには destroy というボタンもあり、ここにアイテムをドロップすれば永久に破棄することができます。 PK note は破棄してもかまいませんが、他のアイテムは取っておきましょう。いろいろとアイテムを操作してみて、 EverQuest のマウスオペレーション感覚をつかんでください。一通りの操作感覚をつかんだら、さっそく武器を右手のスロットにドロップして装備しましょう。この世界のガードたちは、武器を振り回しながら走り回るサイコ野郎をまったくとがめません！

　小粋な武器を右手に構えたら、guilde note を読みましょう (忘れていませんか？　右クリックです) 。この手紙には、ギルドに入るためには誰にこれを渡せばよいのか、詳しく書いてあります。目的のギルドは街中にあるでしょうから、まだこの時点で外に出る必要はありません。どうしても外に出たいというのならば止めはしめませんが、 guilde note を持ったまま死亡すると消滅してしまうことだけは覚えておいてください。ギルドマスターを探すのにはちょっと骨が折れるかもしれません。ノーラスの街には、たいへん巨大なものもあるからです。たとえば、 Freeport は街だけで３つのゾーンに広がっています。試行錯誤を繰り返しながら他のプレイヤーの助けを借りれば、すぐにギルドを見つけることができるでしょう。 guilde note を左クリックして、ギルドマスターの体に左クリックでドロップしてください。するとトレードウィンドウが開きますので、左下にある Trade ボタンを押しましょう。これでギルドに入ることができました。この比較的に簡単なクエストの報酬として、少しだけ経験値を獲得することができます。

訓練

　ギルドに入ってからは、ギルドマスターに訓練を頼むことができます。ギルドマスターを右クリックしてください。十分に近づいていれば、トレーニングウィンドウが開きます。画面左のリストにあるスキルが、あなたのクラス/種族が習得できるスキルです。スキルの下に practice という単語がある場合、そのスキルは現時点で訓練が可能であることを意味します。数字が表示されている場合、そのレベルに到達するまで、そのスキルの訓練を行うことはできません。プラクティスポイントはレベルごとに５ずつ獲得します。最初から５ポイントのプラクティスポイントがありますので、これを使って訓練するのもよいでしょう。どのスキルを伸ばしてもかまいませんが、私の場合、初期装備の武器スキルに投資します。ここでスキルを成長させておくと、野外に挑戦するときに大きな違いをもたらします。レベルが低いうちはスキルも急速に成長していきますが、レベルごとにスキルの上限が定められています。 One-Handed Slashing 100 の レベル２ウォーリアなどというものは絶対に存在しません。コンバットスキル (Defense, Offense, Piercing, One-Handed Slashing など) に関して言えば、レベル１につき５ポイント上限が増えます。ですから、上限を知りたいときはレベルに５をかけてください。たとえば、レベル10ウォーリアの One-Handed Slashing は50まで成長します。レベルが上がるにつれてスキルも成長しにくくなりますが、その一方、より高度なスキル (Riposte, Dual Wield, Parry, Disarm など) を習得できるようになります。プラクティスポイントの使い方についてですが、前述したように、レベルが高くなるとスキルは成長しにくくなります。ですから、大部分のプレイヤーはレベルの低い間、プラクティスポイントを使わずにとっておくようです。計画的な訓練をしましょうとだけ言っておきますか。

　トレーニングの後は、外に出て覚えたことを実践するときです！　街の出口を探し、ノーラスの荒野へ旅立ちましょう。外には広大な平野、密集した森林、もやが立ち込める砂漠があなたを待っています。どこへだって行くことができますが、十分な経験をつむまでは街から離れすぎないようにしたほうが賢明でしょう。楽に歩き回れるようになるまでは、本拠地のゾーンで経験値を稼いでください。

トレードスキル

　先日の EverQuest 大型拡張の際、このゲームの新しい要素、トレードスキルの追加がなされました。以前から機能していた Tailoring と Fishing に加え、金属鎧の加工、パンやケーキの調理などさらに多くのことができるようになったのです。ここ二日ほど Smithing スキルで遊んでいましたので言わせていただきますが、非常に厳しいです。今のところ、プレートメイルを着た初心者が走り回るという心配をする必要はないと思われます。私がフェイズ４で新たに作成したローグは今レベル14で、すべてのプラクティスポイントをためていました。 Smithing をはじめるときに60ポイントを生贄にささげました (つまり、 Smithing はおよそ60です) 。この状態で５回以上挑戦してみましたが、成功したのはバンデッドチュニック一着と、バンデッドアームズ一組だけで、その他はゴミでした。しかも、一回挑戦するごとに 5-6 のプレートが必要で、これが実に高くつきます。それでは、トレードスキル全般の話をしましょう。

　どのトレードスキルにも、スキルの使い方、必要な材料、必要な鋳型、数量を説明した本があります。

　トレードスキルを使うには、ある種の鋳型が必要です。この鋳型は、何かを作ろうとしたとき、それが成功したかどうかに関わらず破棄されます。

　使用された原料は成功不成功を問わず破棄されます。

　トレードスキルの成長はきわめてゆっくりとしています。 Tailoring については、ゲーム中ひたすら他の人にレザーアーマーを作ってあげていましたが、20 から 23 に成長しただけであり、 Smithing についてはまったく成長していません。

　Tailoring と Smithing 以外のトレードスキルについてはまったく分かりません。概要は変わりませんから、参考にした上で何ができるか自分で確かめてください！

死

　冒険には避けられないものとして、残念なことではありますが、死の存在があります。しかし、ノーラスでの死は永遠のものではありませんから、決して望まれるものではありませんが、恐れるべきものでもありません。落胆させ、また痛みを伴うものではありますが……。死亡すると、開始都市か、あるいは affinity* を結んだ場所で蘇生します。どちらで蘇生するかを選ぶことはできず、また一度に結んでおける affinity は一箇所だけです。 affinity の締結は魔術師系クラスの専売特許で、ウィザード、エンチャンター、マジシャン、ネクロマンサーともに Bind Affinity をレベル12で習得できるようになります。クレリック、ドルイド、シャーマンはレベル14です。これがないと、 Greater Feydark を探検中に死亡して Halas まで戻されるということになりかねません (事実、私の身に起こりました) 。レベルが低いうち (1-3) は、死んで得るものの方が多くなっています。食料と水が補給されますし、一瞬でヒットポイントが最大値まで回復します。その一方、死に伴うペナルティはないに等しいのです。死が問題になってくるのは、レベル４に到達してからでしょう。詳しく説明します。

*affinity = 親和力, 密接な関係, 訳にはなっていないが「リンク」がいちばん近い言葉かもしれない。

レベル４: ヘルスが残り 25％ 前後で全力疾走ができなくなります。 15％ 前後でまったく走ることができなくなります。死亡すると初期装備の武器が失われ、また食料の補給もありません。

レベル５以上: 死亡すると経験値を失います。これによって現在のレベルに必要な経験値を満たせなくなった場合、レベルが下がってしまいます。失われる経験値の量は、他にも条件はありますが、レベルに比例します。たとえば、レベル10ウォーリアはレベル５ウォーリアよりも失う経験値の量が多くなっています。レベル10ウォーリアは現在の経験値のおよそ 10％ 、bubble* の 1/2 を失います。レベル５ウォーリアは経験値が減っているのを認識することすら困難でしょう。

*bubble: exp.bar の目盛、５個たまるとレベルアップ。

その他

ドアを開ける: 左クリック、あるいは画面中央にドアを持ってきて U キー

地面のアイテムを拾う: 左クリック

ターゲットする: 左クリック

バッグを開く: 右クリック

あたりの人に話し掛ける: Enter を押してからメッセージを入力し、もう一度 Enter キー

特定の個人にメッセージを送る: /tell, /t と入力、あるいは T キーを押してからその人の名前を入力し、スペースをいれてからメッセージを入力。最後に Enter キー
例: /tell Andy Where are you? これで Andy は Jomar tells you: Where are you? というメッセージを受け取ります。

ゾーン全体にメッセージを送る (2種類):

  1. /OOC (Out of Character の立場で) メッセージ、最後に Enter キー
  2. /SHOUT メッセージ、最後に Enter キー

/OOCのメッセージは緑、/shout のメッセージは赤で表示されます。/tellのメッセージは紫で表示されます。

** コミュニケーションコマンドの使い分けを誤らないようにしてください。名前がわかっている人物に質問をするときは /tell を使い、決して /shout や /ooc を使わないように。特定の個人へのメッセージとして /shout, /ooc されるといらいらしますし、はっきり言ってしまえば、スパム行為にあたります。おそらく、メッセージ自体を無視されるばかりでなく、感情的なクレームが飛んでくることでしょう。以前、２人の人物が /OOC で会話しているゾーンに出くわしたことがありますが、まったく、なぜあんなことをするんでしょうね？
