「弓職人の手引 第五版」

弓矢作成術の歴史はエルダーエルフと同程度の長さを誇る。本書は、弓の作成に関して詳しく述べると同時に、その姉妹技術ともいうべき矢の作成に関しても情報を提供するものである。

弓を作る場合も矢を作る場合も、この技の実践には Fletching Kit が欠かせない。
Fletching Kit は弓屋で買うことができるだろう。

弓

　どの弓についても、作成には４種類のアイテムが必要となる。

【道具】
これは、弓の部品として必要なものではないが、作成の手助けとなり、より価値の高い弓を組みたてることができる。
Whitting Blade は弓の質を高めるが、さらに優れた道具として Planing Tool がある。

【弦】
広く用いられるものとして、 Hemp Twine がある。麻糸のひとつ上は Lining String である。
最高級の弓には、 Silk (絹糸) が用いられる。

【カム】
ほとんどの弓には必要ないが、 Composite Bow を作る際にはいくつか Cam が必要となる。

【木材】
この部分が、弓の質に直接の影響を与える。以下に、木材の種類をその質及び難易度順に並べる。

  • Hickory [ヒッコリー]
  • Elm [楡]
  • Ash [トリネコ]
  • Oaken [樫]
  • Darkwood [ダークウッド]
  • Steel [鉄]
  • Ceramic [製陶]
