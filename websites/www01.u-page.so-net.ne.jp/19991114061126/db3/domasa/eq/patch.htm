  • User Adjustable Clipping-plane
    • Clipping-Plane をユーザーサイドで調整できるようにしました。これによって、ハイエンドなハードウェアを使用しているプレイヤーは画面上の 3D を高く再現する距離感を得ることができるようになり、またローエンドなハードウェアを使用しているプレイヤーは距離感を犠牲にしてフレームレートの向上を期待することができます。この新機能を使用するには、[Option] - [Display] にある Clipping-Plane スライダーを調整してください。バーを左に動かすと距離感が減少し、右に動かすと増大します。ただし、ゾーンによって、二種類の使用制限がありますので注意してください。
      • インドア型ゾーンでは使用できません。
      • Fog を強くかけられたゾーン (Toxxius Forest, Innothule Swamp など) では効果が小さくなります。

Oct. 14th, 10:00pm [JST]

  • Patch Program アップデート
    • Patch Program version 1.1.6 がリリースされました。これは Optional Patch System を導入したもので、新ゾーン、BGM、キャラクター、動作などのファイルのダウンロードを任意に行うことを可能にします。通常どおりパッチプログラムが動作して、ダウンロードが終了した後、Optional File のダウンロードの可否をたずねられますので、ダウンロードを行う場合は Yes を、無視する場合は No を選んでください。
  • Optional Patch [任意]
    • 新ゾーン、Temple of Solusek Ro [Solusek C] がダウンロードできます。
  • General
    • EXP と Loot 権 はグループのトータルダメージに対して与えられます。これはすべてのサーバーで有効です。これは一種のテストであり、二週間ほどしてからこの変更が Kill Steal 問題に対する何らかの解決となったかどうかを調べる調査を行う予定です。
    • Rubicite Armor はもう新規に作成されることはありません。その代わり、Rubicite を装備可能であったクラスのそれぞれについて、専用鎧のクエストを導入します。
    • Jouneyman's Boots も以前の場所で得ることはできなくなり、クエストを通じて入手するものになります。
    • Trade 詐欺への対策として、アイテムや貨幣を取り除くためにはいったんそのトレードをキャンセルしなければならないようにしました。
    • PC / NPC が Common Tongue 以外の言語での発言を行った場合、話し手のスキルが十分であれば、その言語が何語であるかが表示されるようになります。
    • Fishing Pole は Fishing スキルを使用するときは常に、キャラクターの手に表示されます。
    • 各地に大量の新クエストが導入されました。
  • Trade Skill
    • Blacksmith スキルにより、Forged Weapon を作成することができるようになりました。
  • Spell
    • Magician が召喚するアイテムの性能が向上しました。
    • Fufil's Curtailing Chant, Tuyen's Chant of Flame, Tuyen's Chant of Frost [BRD] の性能が向上しました。
    • Burnout 系 [MAG] がもつ HP 減少機能が軽減され、ステータス上昇機能が向上しました。これまで効果時間は無限でしたが、これにより効果時間 15 分に変更されます。また Burnout III のコストは減少しました。
