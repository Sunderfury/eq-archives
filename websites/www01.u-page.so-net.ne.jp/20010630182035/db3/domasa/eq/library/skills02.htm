トレードスキル「ブラックスミス」

原題: Trade Guide: To Blacksmithing
著者: Vrunak

導入

　さてさて、Blacksmithing に興味はあおりですか？　鎧の補強に挑戦したいと思いませんか？　錆びた剣を研ぎたくありませんか？　このガイドはその基本を手広く扱っています。長い文章にはなりますが、興味さえあれば結局は貴重な金と時間を節約することができるでしょう。

Blacksmith に必要なもの

　数多くのスキルと同じように、 Blacksmith としての仕事をはじめるには Basic Blacksmithing の本を手に入れる必要があります。また、必要な物資 (Ore、Mold 、水など) を得るために、 Blacksmithing 以外のスキルも必要となるかもしれません。少なくとも 10 スキルプラクティスポイントを稼ぐまでは止めておいたほうがよいと私は思いますが、それより早くとりかかることもできます (この場合、より多くの物資を浪費することでしょうけど)。Ore (鉱石) は街の武器屋/防具屋で買い求めることができます。始めたばかりの時点では、揃えられる ore の量にも限度があるでしょう。本は、金属鎧を売っている場所でなら何処ででも買うことができます。ただし、種族によっては (オーガ、トロール) 厳しいスキルです。 Oggok には Forge (炉) がひとつだけしかありませんし、 Grobb で Ore を扱っているのは一人だけだからです。また、どちらの種族も、低レベルな鍛冶屋に必要不可欠な Mold (鋳型) を売っている商人がいません。他の種族については、ほとんどが必要なものを街で調達できます。

Lesson 1: 研ぐ

　錆びた剣を研ぎなおすには、 Sharpening Stone (研ぎ石) と 錆びた武器、それに Blacksmith の本と Forge が必要です。Sharpening Stone は Ore を扱っている商人から、 錆びた武器はスケルトンから奪い取ったり、釣りの成果 (これが好き) として手にいれることができます。まず、 Forge を左クリックしてコンテナを開きます。武器と Sharpening Stone を中にいれ COMBINE ボタンを押してください。失敗すると、両方とも破棄されます。成功すると、 Sharpening Stone は消費されますが、曇りがかった (Tarnished) 武器が手に入ります！　失敗は高くつくことがありますが、これについては後ほど成功率と一緒に説明します。曇りがかった武器の攻撃力は錆びた武器と変わりありませんが、攻撃発生速度は小さくなります。たとえば、錆びた Longsword の発生速度は 35 ですが、これを研ぎなおすと 33 になります。個人的にはあまり変わりないような気もしますが、武器を研ぎなおせばスキルを成長させることができますので、この点けっして無意味ではありません。

Lesson 2: 鍛える

　金属製のアイテムを作るためには、まず Ore を使える金属片に精錬する必要があります。 まずは、 Metal Bit という鍛冶原料の基本単位となるアイテムを作りましょう。Forge を左クリックしてください。 Ore (小) を２つとフラスコ１杯分の水を使います。 Ore はばらばらにしてから使ってください、まとめたままではうまくいきません！最後に COMBINE を押し、そして祈りましょう。祈りが天に通じれば、 Metal Bit が完成するはずです！　ある程度作り上げたら、今度はそれを使う番です。

　リクエストが最も多かったのはやはり Stud でした。ただし、20台半ばまでは挑戦すべきではありません。20台半ばでも費用がかさむことでしょう (後述) 。Stud を作るにはまず File が必要です。 Basic Blacksmith にちゃんとレシピが書いてありますから簡単でしょう。File 作りに必要なのは、 Metal Bit, File Mold, それに水です。これまでと異なる Mold を扱う商人を探すのは骨が折れるでしょうが、街中を走り回って見つけ出してください。 File Mold を 2,3 購入したら、Forge に戻って Metal Bit １つと File Mold １つ、フラスコ１杯分の水を COMBINE してください。 File 作りは比較的簡単ですから問題ないでしょう。そして Stud を作るには File １つとフラスコ１杯分の水、そしてばらばらにした Metal Bit が３つ必要です。Blacksmithing の使い方は以上です。要するに、本に書いてあるとおりにすれば問題ありません。

スキルレベルと成功率の関係

　下表は Blacksmithing 初期における成功率をアイテムごとに表したものです。

スキルレベル: 10 20
Metal Bit 60％ 80％
File 50％ 75％
研ぐ 20％ 33％
Stud 0％ 11％

　レベル21になると、Metal Bit を作る際「This item is trivial for someone of your skill level to make」というメッセージが表示されるようになります。しかし、この段階でも成功率は 75-80 ％というところでした。また、レベル20で９回 Stud 作成に挑戦したとき、成功したのはただの一回 (11％) でした。運が悪かっただけかもしれませんが、他の方の調査でも似たような数値が上がっています。 コストについて Stud の原料にかかる費用 (銅貨500枚以上) から計算してみると、 Stud シリーズをすべて揃えるには 銅貨 4,500 枚もかかってしまいます！　それから上の表について、レベル 30 の行はもうすぐ追加できると思います。

Blacksmith で利益をあげるには？

　私は二人のキャラクターを使って Blacksmithing を調査してきました。ひとりはヒューマンエンチャンター (カリスマ 100) で、もうひとりは Ogre Warrior (カリスマ 37) です。これにより、カリスマが商人との取引から引き出す利益を知ることができました。 Blacksmith にとっての黄金市場は言うまでもなくプレイヤー相手の商売ですが、ここにあるデータもきっと興味深いデータになっていると思います。下表は Blacksmith 関連アイテムについて NPC 商人と取引したときの相場で、 / の前の数字はカリスマ 100 の場合、後ろにある数字はカリスマ 37 の場合です。

Sharpening Stone 3 / 4 ― / ― ― / ―
Water 10 / 13 ― / ― ― / ―
Small Ore 52 / 69 ― / ― ― / ―
Small Brick Ore 525 / 689 ― / ― ― / ―
Metal Bits ― / ― 76 / 60 114 / 148

　もちろん、「作成費用」に示された数値は全工程において成功率 100 ％ を想定した数値です (希望的観測)。「売値」は私の製品に対して NPC 商人が示した買取価格です。また、単位はすべて銅貨で統一しています。

後輩たちへのアドバイス

　Smithing に興味をお持ちいただけたでしょうか？　これから志そうという方には、 まずはじめから 10 プラクティスポイントを投資することをお勧めします。次に、見つけた錆びついた武器を片っ端から研ぎ上げ (私は釣りで手に入れた錆びたダガーを使います) 、財布が許すかぎり Metal Bit を作り続けましょう。これだけで 10台後半までスキルを伸ばせるはずです。ここまでくれば File に挑戦することもできますが、 File はひとつにまとめることができず、バッグと銀行の空き場所を取ってしまうので、避けたほうがよいと思います。20台前半になれば Stud 作りを試みることもできますが、まだ金を道端に捨てるようなものです。結局、レザーアーマー作りと比べたときの利点を問われると……私にも確信が持てません。そこに１セット銅貨 5,000 枚の鎧を正当化する何かがあるのでしょうか？　確信が持てません。

　以上、私が知っていることはすべて書き終わりました。何らかの手助けになれば、と思います。訂正、データの追加などございましたら遠慮なくお教えください。それでは、鍛冶師たちに幸多からんことを！
