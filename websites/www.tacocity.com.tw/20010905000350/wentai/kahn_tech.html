左圖說明:

如何加入新的Kahn Server?
在 Active Kahn Servers 按滑鼠右鍵,再選 Insert Server, 然候在 Server IP Address 輸入kahn server 的IP, 在 Server Descripiton 輸入此 kahn server 的名稱(如下圖).然候在你新加入的kahn server 按滑鼠右鍵, 選擇 Always Active List.這樣新的kahn server 就會在最前面的列表出現了.



如何連上Kahn Server ?
點選要連上的Kahn Server, 再按下右下角的Connect 就行了~

進入Kahn Server 看不到中文?
在File中選Preferences再選Set Font,設定你要的字形和大小,再按OK.然候關掉聊天室, 再進去一次就行了.(如下圖)



如何加入其它的頻道?
在右邊有#aoe 等字樣, 只要點二下在該頻道上面, 就可以進去該頻道了(如下圖).如果你要更改預設進入的頻道, 只要在File->Preferences->Default channel 這個地方改就行了.(如上圖)

如何得知新的Kahn Server?
在國內沒有正式的 kahn Servers.都是由學生在學校架出來的.所以一但你突然連不上去的時候, 不是關了就是搬去別位址了~
如何下載相關檔案?如架站或是上網的軟體?
請到Kahn的網址下載.如果你要下載較舊的檔案(如 v0.99 的 client), 請去他們的FTP下找就有了.

~如何用KAHN玩GAME~
前言:
現在的Kahn Server已變成純粹是聊天及找朋友玩Games的場所了,
所以你只要在此說出你要玩的遊戲名稱及IP,如果有人願意加入的話,
就會自已連上你的遊戲IP了喔~不過以下的說明還是給剛接觸Kahn的人
看看,想必是很有幫助的才是.


世紀帝國(AOE/AOK)
選多人遊戲MULTIPLAYER後,再選TCP/IP連線. 建立新遊戲選CREATE,然後同時按下ALT+TAB跳回 WIN95/98.在KAHN中告訴你的IP就行了.(開始玩的時候 記得關掉KAHN喔!!)
原理:只是用來跟大家說明自己IP的用途~
星海爭霸(SC)
開著 chat視窗(不要關掉)然後執行sc,按多人遊戲--Local Area    Network(ipx)--然後再選擇join 或 create game就ok啦!
原理:利用Kahn的IPX->TCP/IP的功用.(所以如果遊戲只提供IPX的連線方式,可以用KAHN來試試看能不能多人連線)
暗黑破壞神(DIABLO)
將KAHN的CHAT開著~~然後選多人GAME後.再選IPX連線 就可加入別人的GAME或自己開GAME.
原理:利用Kahn的IPX->TCP/IP的功用.(所以如果遊戲只提供IPX的連線方式,可以用KAHN來試試看能不能多人連線)
