技能/屬性

基本的技能/屬性

    UO目前的技能學習, 以單一人物去學習完所有技能的話,目前設定最高限值為700.因為技能總合值是跟STR/DEX/INT的數值而有一定的算法.所有技能總合大約為1000-1200數值左右.但是再上Staus(STR/DEX/INT)計算的話,最高值為700左右,也就是說,你一個人物最多只能練到七個技能是100(GM).

    以下表格為技能再加上個人所擁有的Staus而做出來的評估表.技能如果將近100的話, 對Staus的互相影響較低.

      技能表示= SB X (100 - 必需技能)/100 + 必需技能
        SB(Staus數值(STR/INT/DEX)) = STR X STR% + INT X INT% + DEX X DEX%

    例1:當你的INT為45,Magery技能為50時,實際Magery技能數值(以上限值來說)為

      (45 X 15%) X (100 - 50)% + 50 = 6.75 X 50% + 50 = 53.375 .取四捨五入就是53.3數值

    例2:當你的INT為45,Magery0時,實際Magery技能數值(以上限值來說)為

      (45 x 15%) x (100 - 0)% + 0 = 6.75 x 100% + 0 = 6.75 .取四捨五入就是6.7數值

    因為UO目前限制一個人物最多只能學習技能總合為700.所以如果你的技能總合將近700的話, 練起來會相當累人,請特別注意這點.

    而UO目前限制一個人物最多可有Status(STR/INT/DEX)的總合數值為225.

職業

    目前職業名稱的稱號,是以什麼樣的技能成為GM,然候你的人物職業稱號就以該技能為首.但是如果你的人物同時有兩個以上的GM技能時,則以兩項(以上)技能的編號為考量.那個技能編號愈高的,就是成為你的人物職業稱號.
