If you are going to set up linux to just run SEQ, I dont think it is worth it. You will still need to know how to set it up, how to set up seq, and how to get it to run. So unless you have quite a few hours to spend scratching your head, dont bother.

If your going to set up the linux box because of the firewall, email services, web server or any of the other countless abilities, then it is probably worth the time to set up SEQ while your at it.

Linux isnt like Windows which requires very little thought when installing it (although it is getting there). There is still a good chance that at least one of your cards wont work and you will need to either find drivers for it or give up and get a different card. Now you know why they call it "plug and pray"

Anyway, just be prepared to do a lot of reading if you decide to install linux and good luck.
