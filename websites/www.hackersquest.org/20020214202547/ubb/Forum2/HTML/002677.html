Zombie 		posted 12-03-2001 08:03 PM            
Viper, delete Qeynos from the MYSQL database (search Tux's site for the way how. It's something like DELETE FROM database WHERE name="qeynos" . or if you can't find the way how, just redo the everquest database in MYSQLADMIN. Redo everything the same way you did it originaly. Then MAKE SURE that your db.ini file and qeynos.bat file have YOUR ip (or 127.0.0.1 if that's what you use). also make sure that you don't erase the port number from the .bat file. At character creation, use the same name that you entered into your MYSQL database (admin for example). It works perfectly for me now.
