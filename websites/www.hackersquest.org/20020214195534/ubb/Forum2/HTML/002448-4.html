Make some NPC races playable!

Or live in Kael....
This would open SO much more options. Assign a few PC classes to those races. Warriors, shamans and so on. Wouldnt be difficult to implement at all.

Make a zone similar to which GMs used to live in. Airplane or so. Place NPCs for every playable race there. (standard races and the additional ones)
The new player talks to the race he wants to be and then he is born as this character in their hometown.

I go to the Kael giant there for example tell him that I want to be born as a giant and poof I get a morph to giant shape and are placed in Kael with newbie equipment.
This will create new issues like newbie grounds in velious for example. But the zones arent populated anyway yet and it would be easy to create other spawn layouts (monster levels) compared to the Verant servers. Well we want to make the EMU more fun than the Verant servers, do we?

I would expect the following EMU servers:
PvP
PvP (Anarchy)
Roleplay

Make leveling easy and quick, dont wanna waste a year to get to 60.
I could imagine a PvP server where you start at 60 and its all about war between the races. Races might be allied like on Verant servers (good/neutral/evil). No need for dumb leveling, just log on and have fun and join in the battle.
This is what the game should be about. Fun!
Create a lot of artifact items that only one person can own and is not bankable. When that person gets killed the enemy can loot one artifact item from the player.

I could post much more detail here but you might be able to imagine details yourself.

These are my basic fun concerns. Trade skills, quests and those thing I dont expect in the EMUs anytime soon anyway. Getting combat and magic system to work is difficult enough. After we see these basics other things might be implemented.
