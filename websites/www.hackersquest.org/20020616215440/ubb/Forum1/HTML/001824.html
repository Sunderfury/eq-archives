I am pretty sure I have heard every error there can be with this program. So I will write a step by step thing as to what to do, to be error free.


1. RUN Everquest ALONE, without DXwindows loaded, without anything else loaded.
On the screen with the "Connect", "Account" buttons. Hit the "Options" button. Select Primary Display Driver.

2. Log into any character you wish. When completely in the game goto options, then display. DISABLE Mip Mapping, Level of Detail, and Same size windows. Log out of EQ completely.


3. Goto your windows display settings (right click desktop and hit properties, or via control panel) and change the windows color to High color (16 bit).

4. Download the DXwindows program http://www.hackersquest.gomp.ch/downloads/dxwindows/DxWindows0.1Beta2.zip

5. Unzip it to anywhere you like on your hard drive. Run DxWindows.exe, a dialog box should pop up stating "DXWindows is now running" Keep this box open, hitting OK will close DXwindows down completely.

6. Run everquest as normal. Log in, right after you select a server everquest should then be in windows form loading up character select screen. And that's all there is to it!

*Note* Changing resolutions and/or color bits in windows while EQ is running in it's window state WILL crash EQ.
If you are one of the people who can only see text while in window form, and graphics are blurred, etc. I believe that's just you're video card not being able to handle it. And i doubt there's a fix for that.

I believe i've covered it all.

Observations while running DxWindows:

First, system specs:
P3 450, TNT2 Ultra (32Mb), 128Mb RAM, 20Gb HDD, SBLive, Win2kPro, dx8.0

1. The only thing I did prior to running dxw was change desktop color from true to high (24bit to 16bit). The resolution of my desktop is 1280x1024. My EQ runs in 1024x768, no mipmap, screens same resolution. dxw runs fine.

2. Quirks. Getting out of the game causes eq to crash. While in game, there is no problem whatsoever. I think dxw is having problems relinquishing control to the server select process which wants to run in full screen. Which is odd since the screens shown up to and including server select are all in fullscreen mode. Once the splash screen comes on prior to character select and afterwards, all screens are in windowed mode.

3. Going between applications. ALT + TAB is your friend. CTRL+ALT+Z to get mouse, alt tab to the dx window (still have outside mouse) click in title bar once, then minimize. Just clicking the minimize will return the mouse to EQ. Do your business, then unminimize and click outside mouse in an unobtrusive corner of the eq window to go back to eq.

4. Looks. EQ is a bit dark. Probably just adjust the gamma on my desktop and it should clear up ok. I am playing a woodelf druid currently and the infravision has allowed me to put off screwing with this. The fonts are a bit blurry until I maximize the dxwindow. (doubleclick titlebar).
