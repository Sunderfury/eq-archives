EthernalQuest 0.3.1 Alpha released! :: 1.14.2002
Thanks to our Coding Guru Zordon, you can now download and try the new Version of EthernalQuest!
It is not as full as someone might want it to see but here if few hightlights:

** Works with latest version of Everquest
** Fully functional Login server
- more then one World server can be connected to Login server
** Fully functional World server
- Allow different zones to be hosted on different computers
** Zone server now supports
- automatic zone startup (when player zone in)
- automatic zone shutdown (when no player in zone for extended period of time)
- movement (within zone)
- regualar zoning & teleporting within zone
- zone exploring commands
#zone name, x, y, z
#goto x y z
#loc
- doors (simple doors work. elevators do not work)
- spawns (basic spawns: guards, merchants are generated in few zones)
but not yet active
** Easy scripting.
- Ruby language is used to allow any kind of scripting.
- Most of current Login/World/Zone functionality is implemented in ruby.
- All stable functionality will be later moved into C++ to get better performance.

As always, grab it from the filetable above or
if you'd like to participate on the coding, head over to the SourceForge Project Page

A Big "Thank You" to Zordon for making this release possible!

/wave
