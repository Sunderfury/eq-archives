Decaying Skeleton 		posted 07-03-2001 10:53 PM            
You might actually change the look of any spell in the game. Just edit the byte 604 of a spell bloc, then change for example the last bytes of the file to preserve the global crc32 and avoid the patch program to detect the change, and then run the game. Of course no one else will be able to see the altered effect.
