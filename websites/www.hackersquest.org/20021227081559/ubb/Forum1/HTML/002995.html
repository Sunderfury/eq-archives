Woot! Success! I have gotten the speed hack to work, and here's how!

WinHack has an ability to create tags on select blocks of mem. I was looking for a way to set a value, and well, Lock it! and Lock it tight.

With the tags, you can freeze them, thus locking them.
So, for the speed Hack, head to offset 67e335, and set it to your desidered value. Create a Tag, and goto the tag tab. From there, make sure it's frozen ( Happy face ).

Start the rest of EQ, and you're good to go!
