actually, the little script runs oe and stuff on its own. what I did, was put the script in /usr/bin , named it "odinseye" and did "chmod +x /usr/bin/odinseye"

then, from ANYWHERE you just type "odinseye" and it will run. the script itself compares the current version with your compiled version, and then proceeds to update itself or just run.

also note, there are some settings you can change at the top. it should work fine as is, but you might want to change them. mine sits in /root/odinseye, with some downloaded files in /root/odinseye/download. the first time you run the script, it WILL create the dirs mentioned, and download and compile it. if qt isn't where mine is, you will want to change that variable. it should be pretty self explainatory, i tried to comment everything very well. it was mostly meant so that my silly sister and her boyfriend who can barely use windows can use it :)
