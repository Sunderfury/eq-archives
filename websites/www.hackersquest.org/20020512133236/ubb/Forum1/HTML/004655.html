This probably happened when you /camped out and minimized the window. To fix this you should open the eqw.ini file and go down to positions. Find the resolution you run EQ in and change the x and y settings to 0. Change them all to 0 if you aren't sure which resolution you are running EQ in. Exit, save, and rerun eqw. Reply to this board to see if it worked or not.
