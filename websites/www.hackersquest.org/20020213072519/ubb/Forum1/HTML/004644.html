you can get dual monitor support to work in windows with multiple cards or dual headed cards.

But, once EQ starts it will lock the primary display the way it normally does, and you wont be able to move the mouse into monitor two, or do anthing with it.

In short, while EQ is running, you can only look at what monitor 2 is doing, you cant interact with it.

Basically you should use EQW...run that window on one screen and just Alt tab out of EQW to what ever program you want on the other screen. This will only work for Duel headed Display cards properly tho.

With a system with 2 Display cards you can do a work around but performance is slow. DirectX only works with the primary display card so run EQW in that. Agian running 2 different graphics cards is not as good as 1 graphics card that supports 2 monitors.

For other games like Counterstrike just run them in a window also
