Ever wondered why big sites as EQ Stratics or EQ Vault dont have spells listings over lvl30?
Because it takes time to make a character with such a high level! :)

So for all you people out there, who are curious what spells you are going to get at higher levels or which secret spells GM's and NPC's got, here is the program for you.

The EQ Spell Sniffer!

This little program, that has full source code included (for you bo/netbus paranoid types), does nothing but read a datafile in the EQ directory, parse it and give you the information.

For you RPG'ing people, this program might be a spoiler and you should not download it, as it reveals alot of information about spells.

Install notes:
Just copy either the eqss.exe to the EQ directory or copy the 'spdat.eff' file from EQ directory to the same directory as eqss.exe. I have no idea how it will react without, so dont try it ;)

Since this is still a beta version, working or not, I still have to add some functions:
 - Search button does nothing.
 - Go! button doesnt do anything either.
 - I havent added such things as Mana cost (it is there), price (dont know about this one) etc. etc.
 - No checking for the spdat.eff file what so ever. If its not there, program will prolly nag at you or crash. Be sure to have either, this exe in the EQ dir or copy the spdat.eff to the same directory as Spell Sniffer.
 - It doesnt parse empty entries correct. Just ignore it for the time being.
 - The sourcecode is VERY messy. Don't change it, unless you know what you are doing ;).

NOTE: It should work without any extra files, but I have only tested it on my own machine, with Delphi installed and on Fool's machine, which doesnt have Delphi installed. It worked flawlessy on both.
