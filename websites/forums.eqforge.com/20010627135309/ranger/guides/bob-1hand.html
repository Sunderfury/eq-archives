ASSUMPTIONS AND FORMULAE
These numbers are calculated using a level 50 ranger with max skills and 140 strength, supposedly fighting level 50 mobs with 100-120 dex.


1st premise: main hand damage is different from off hand damage in that a LevelBonus is applied to each main hand strike.

LevelBonus = (Level - 25)/3 for 1H weapons
LevelBonus = (Level - 25)/2 for 2H weapons
(some speculation exists as to these formulae after level 50. it is possible that they change then.)

2nd premise: weapon delay can be translated into real time(ie. seconds) in this way
RealSeconds = (Delay/10)
EXAMPLE: SSOY 8dam/24delay, or swings every 2.4 seconds.

3rd premise: Max damage is a function of the Strength multiplier. MAX damage is calculated by using:
MaxDamage = ((STR + Weaponskill)/100)*WeaponBaseDamage</code>

or for me with a SSOY:
((134str + 200skill)/100)* 8 = 26.72 (27)
Main hand adds the level bonus for a total of 33.32 (33) MAX damage.

4th premise: Based on data collected with parsers, it looks like there is a MagicNumber, or ModalDamage amount that you strike for an extra large percentage of the time. This number is found with this formula:
ModalDamage = 2*WeaponBaseDamage + (LevelBonus, if main hand)

or for me with SSOY: 2*8+6.6 = 22.6(23)

5th premise:
MinDamage = 1 + (LevelBonus, if main hand)

6th premise: Based on a large amount of parsed logs of fights, you will hit for ModalDamage number around 40% of the time. You will also hit for Minimum Damage 40% of the time, and hit for the other amounts of damage 20% of the time, which I approximate by using Max damage 20% of the time. You can then derive an "AverageDamage" amount by using the following hit rate data.

AverageDamageAmount = (40%*MinDamage)+ (40%*ModalDamage) + (20%*MaxDamage)
I used 20% for my data, which is fine because it is merely a relative ranking.

7th premise:
Using parsed data, the following assumptions about Dual Wield and Double Attack success ratios have arisen.
DualWieldPercentage = (Level + DWSkill)/400
and
DoubleAttackPercentage = (Level + DASkill)/500

8th premise: A weapon that procs can be calculated a little more accurately if we assume that against a white mob, with Dex 100-120ish weapon will proc 2% of swings, and be unresisted 75% of the time.
So proc damage, to add to average, is:
ProcDamage = (.02 * .75 * ProcAmount)

FINALLY, if we then assume all of the above formulae are correct, and we assign an arbitrary constant value for accuracy (60%), we can approximate DamagePerSecond per weapons and derive a relative ranking for each. A separate ranking will exist for each Main hand and Off hand usage, as each are calculated VERY differently.

MainHandDamagePerSecond = AccuracyPercent *((AverageDamage+ProcDamage)/(Delay/10))*(1+DoubleAttackPercent)

or for me with SSOY: 60% * ((16.96 + 1.125) / 2.4 seconds) * (1 + 61%) = 6.839 DamagePerSecond.

OffHandDamagePerSecond = AccuracyPercent *((AverageDamage+ProcDamage)/(Delay/10))*(1+DoubleAttackPercent)* DualWieldPercentage

or for me with SSOY: 60% * ((9.95 + 1.125) / 2.4 seconds) * (1 + .61) * .49 = 1.96 DamagePerSecond
