Bow Making

Intermediate Bow Making:

The next stage in bow making is using tools to help make a slightly less powerful bow that is a faster bow. The two available tools are the Whitling Knife and the Planing Tool. If you want a faster bow you can use one of these two tools. A whitling knife produces a Carved Bow and decreases the damage by 1 and decreases the delay by 4. Using a Planing Tool produces a Shaped Bow and decreases the damage by 2 and decreases the delay by 9.

To use one of these tools simply purchase the tool and put it in the fletching kit along with the stave and string. If you succeed in making a bow you will lose the tool but on the bright side, if you fail at making the bow, and you will fail on making bows just as you fail on making arrows, you will get the tool back.

The one important thing to be aware of is that you should never, I repeat NEVER, use a tool with a stave that is cheaper than the tool. For example, never use any tools with a Hickory stave. If you do, you will lose all items in the fletching kit including the tool. Also never use more than one tool of any type. For instance do not use 2 Knives or 2 Planes or 1 Knife and 1 Plane to make a bow.

So now you should be able to make Carved Bows and Shaped Bows. But remember, only use the knife on Elm bows and above and only use the Planing Tool on Ashwood bows and above.
