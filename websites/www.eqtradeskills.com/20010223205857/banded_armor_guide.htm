Banded Armor Guide

Okay, here we go.  This is a smith's bread and butter.  No more metal bits, no more broken wrists.  However the trade off is that banded armor is much more expensive to make and when you fail, it hurts a lot more.  The first and foremost thing you have to remember as an armorer is that there a 3 sizes of armor you cab make.  Small, medium and large so be sure to check what size mold you are using when you start making it.  The following recipes are going to be listed in the order of how many sheets it takes to make each piece. Boots, cloaks, mantles and gauntlets are MUCH tougher to make than other pieces that use the same amount of sheets.  All of these recipes can be found in the Armor and Weapons Guide here.
