Skinning Knife (used to lower the quality of a pelt) = 1 small brick of ore + 1 dagger mold + 1 hilt mold + 1 water

Here is a copy of the Tailoring handbook:

Pelt sizes for Armor

Large Sizes = Bear pelt   

Medium Sizes = Cat pelts

Small Sizes = Wolf pelt   

You will need a large sewing kit before you begin.  This is what it looks like

    

Patchwork Armor -- Trivial at 26
Patchwork Armor = Ruined Pelt (per size needed) + Pattern for type of Armor 

 

Raw Silk Armor -- Trivial at 37

To create a silk swatch you need to combine 2 SPIDER silks (not spiderling) in your sewing kit.  Silk Swatches become trivial at 17
Raw Silk Armor = 1 Silk Swatch (two for leggings and tunic) + Pattern

 

Cured Silk Armor -- Trivial at 82  (Heady Kiola)

Cap = 1 silk swatch + 3 Heady Kiolas + Pattern 
Cloak = 2 Silk swatches + 2 Heady Kiolas  + Pattern 
 Gi = 3 silk swatches + 2 Heady Kiolas + Pattern 
Gloves = 2 silk swatches + 2 Kiolas + Pattern 
Gorget = 1 Silk swatch + 2 Heady Kiolas + Pattern 
 Mask = 1 Silk swatch + 1 Heady Kiola + Pattern 
Pants = 2 silk swatches + 3 Heady kiolas + Pattern 
Mantle = 2 Silk swatches + 2 Heady Kiolas + Pattern 
 Sandals = 1 silk swatches + 2 Heady Kiolas + Pattern 
 Sash = 1 Silk swatch + 2 Heady Kiolas + Pattern 
 Sleeves = 2 Silk swatches + 2 Heady Kiolas + Pattern 
 Wristbands = 1 Silk swatch + 2 Heady Kiolas + Pattern 

 

Studded Leather Armor -- Trivial at 57 

 Belt = Medium Quality Pelt (per size needed) + 2 studs + Pattern 
 Boots = Medium Quality Pelt (per size needed) + 4 studs + Pattern 
 Bracer = Medium Quality Pelt (per size needed) + 2 studs + Pattern 
 Cap = Medium Quality Pelt (per size needed) + 3 studs + Pattern 
 Cloak = Medium Quality Pelt (per size needed) + 4 studs + Pattern 
 Gloves = Medium Quality Pelt (per size needed) + 4 studs + Pattern 
Gorget = Medium Quality Pelt (per size needed) + 2 studs + Pattern 
 Mask = Medium Quality Pelt (per size needed) + 1 stud + Pattern 
Pants = Medium Quality Pelt (per size needed) + 4 studs + Pattern 
 Shoulderpad = Medium Quality Pelt (per size needed) + 3 studs + Pattern 
Sleeves = Medium Quality Pelt (per size needed) + 3 studs + Pattern 
Tunic = Medium Quality Pelt (per size needed) + 5 studs + Pattern  

 

Reinforced Leather Armor -- Trivial at 108

 Belt = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern 
 Boots = High Quality Pelt (per size needed) + 3 Steel Bones + Pattern 
 Bracers = High Quality Pelt (per size needed) + 1 Steel Bone + Pattern 
 Cap = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern 
 Cloak = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern 
 Gloves = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern 
 Gorget = High Quality Pelt (per size needed) + 1 Steel Bone + Pattern 
 Mask = High Quality Pelt (per size needed) + 1 Steel Bone + Pattern  
 Pants = High Quality Pelt (per size needed) + 3 Steel Bones + Pattern 
 Shoulderpads = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern 
 Sleeves = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern 
 Tunic = High Quality Pelt (per size needed) + 4 Steel Bones + Pattern

Animal Skins

Wolf = small size

Cat = medium size

Bear = large size

a skinning knife when combined with a quality skin or pelt will lower quality to the next lowest

e.g. a HQ bear skin and a skinning knife in a sewing kit would result in an MQ bear skin and the knife back,

an MQ bear skin and a skinning knife in a sewing kit would result in an LQ bear skin and the knife back,

and an LQ bear skin and a skinning knife in a sewing kit would result in a ruined skin and the knife back.
