    Re: Bizarre Website Issue

    See if you can reach the WoW site by going here: http://12.129.242.22/

    That's the IP address for the site.
    Code:
    > host www.worldofwarcraft.com
www.worldofwarcraft.com is an alias for worldofwarcraft.com.
worldofwarcraft.com has address 12.129.242.22
www.worldofwarcraft.com is an alias for worldofwarcraft.com.
www.worldofwarcraft.com is an alias for worldofwarcraft.com.
worldofwarcraft.com mail is handled by 10 mail.worldofwarcraft.com.
    Now, check to see that that's the IP address you're trying to connect to from your PC. Go to Start -> Run -> cmd (just type in cmd and hit enter.. a DOS window pops up)

    Now type "nslookup www.worldofwarcraft.com" and hit enter. What IP address is showing?

    Should look like this:
    Code:
    C:\Documents and Settings\YOUR_USERNAME>nslookup www.worldofwarcraft.com
Server:  YOURPCNAME
Address:  192.168.1.1

Non-authoritative answer:
Name:    worldofwarcraft.com
Address:  12.129.242.22
Aliases:  www.worldofwarcraft.com
    Now, if you show something different then you might want to check your hosts file as a virus or something might have changed added a line in there that redirects your connection to a different IP to steal your login/password for WoW. Could also be a program running that intercepts and steals it too.
