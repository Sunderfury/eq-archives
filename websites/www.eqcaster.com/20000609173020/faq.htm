Frequently Asked Questions

Q) Can I get banned for using this utility?

A) I do not believe so, no.

I am very public about my character and my server, and I have not been banned nor warned. I have also let many people at Verant know about this site and what my intentions are.

This program is very benign and does not 'packet sniff' like ShowEq and other utilities. It only displays the current spell data that you could get off of at least twenty other sites, but in a more convenient off-line environment. The only purpose of this program is to let you know specifically what effect does a particular spell have, and some of the minor differences between each.

Q) How can I see what has changed since the last patch?

A) Make sure you make a copy of spdat.eff into a different file (such as 'Jan1500.eff'). Then after a patch, load up both files in the same program and do a 'Window-Tile' command. This will split the screen and allow simulaneous viewing of both spell files.

Q) How come there are no files when I download EqCaster?
Q) Where do I get the eff files?

A) I do not create the spell information, but rather decypher them directly from the EverQuest directory itself. If EverQuest is loaded in the default directories, then the spell files should be opened automatically. Otherwise, you must guide EqCaster to the correct location.

When the 'File Open' dialog is displayed, change the current directory to the EverQuest directory, and open up the file named spdat.eff. The next time you run EqCaster, it will automatically load up this file.

Q) How do I view all the spells

A) Double click on 'All Classes' to view every spell.

Q) How do I sort the fields.

A) Just click on the heading of the spell that you would like to sort. Click a second time, and the list will be sorted in reverse.

The best way to sort class specific data is to click on the 'Spell Name' column, then the 'Level' column.
