December 30, 2000
Well, it's come to my attention that a lot of people aren't testing EQSim because they don't want to supply their EQ Station Name and password. That's perfectly understandable. But the thing is, YOU DON'T HAVE TO USE YOUR STATION NAME OR PASSWORD!!!! The login screen is used for one reason - to set your character's name in-game. So if you want your player's name to be "Super Mario", you would sign in with the login name "Super", ANY PASSWORD YOU WANT (passwords mean nothing.. you can use a different one for the same name every single time), and use the /surname command in-game for "Mario". Once again, passwords mean absolutely nothing. I only kept it in to emulate the game more realistically. If you have any questions about this or anything else, either leave a message on the EQSim board (link to the left) or email me (also a link to the left). Happy New Year!!!!!!!!!
