I couldn't sleep tonight so I made a little utility that a few of you may be able to use with some games. It's called MiniQ. Basically, MiniQ can minimize and restore any window it's assigned to. MiniQ lets you customize which keys will minimize and maximize the window and what window caption to look for. Yes, MiniQ works by searching for the caption of the window so it's easy to apply to different windows. MiniQ does work with EverQuest, and I'm sure it will work with many other games / applications. Best of all, it's only a 6KB download!

June 10, 2001
AdminEQ has been updated after finding a fault with the custom file triggers. When you selected the file to open in-game, AdminEQ would then write it to the INI file. When it detected the string, it would open the file it had loaded from STARTUP. Thus meaning AdminEQ would be running the file you last told it to, or nothing at all. Restarting AdminEQ would fix the problem. Anyway, I've fixed it so that AdminEQ will read the INI file immediately after writing to it to update the settings. Sorry about the inconvenience.
