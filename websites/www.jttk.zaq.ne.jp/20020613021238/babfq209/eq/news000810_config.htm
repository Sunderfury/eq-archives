◇ eqclient.ini ファイルを開く

EQをインストールしたフォルダ内にある、eqclient.ini ファイルをテキストエディタで開きます。　
このファイルがない〜という場合は、一回EQを起動してみてください。

ファイルがどこにあるか分からない場合は、

【スタート】 → 【検索】 → 【ファイルやフォルダ】 で探しましょう。

テキストエディタで開くには、eqclient.iniファイルを、

【Shift+右クリック】 → 【アプリケーションから開く 】 → 出てくるリストのなかから 【 NOTEPAD 】を選び OKを押す。

( 注 ) 設定をいじる前に、初期設定にすぐ戻せるよう、あらかじめ eqclient.iniのコピーを取っておくのが吉。

◇ EQで表示されるテキスト文の色を変更する

eqclient.iniファイルの[TextColor]以下にある、

User_X_Color=Y

X …対応するテキスト。 1なら say 、２なら tell といった感じ。 この数字は変更しちゃダメ。

Color … Red / Green / Blue 。　ここもいじる必要なし。

Y … 0 〜 255。　RGB各色の強さ? 明るさ? なんていうんだろ。ｗ) 　 0に近いほど暗い色。 自分が望む色になるように、ここの数字を変更する。

例 ) 他人の Shoutメッセージを明るい水色に変更する。

Shoutに対応するUser_X は 7 、
明るい水色 にあたるRGB各色の値は、Red=0 、Green=255 、Blue=255 なので、
User_7の各行の値を、以下のように書き直す。

User_7_Red=0
User_7_Green=255
User_7_Blue=255

◇ RUN / WALK デフォルト設定

eqclient.ini ファイルの [Defaults] 項目、

RunMode=True でRUN
RunMode=False だとWALK。

◇ EQ起動時のスクリーンモード（F10で切り替えるやつ）選択

eqclient.ini ファイルの [Defaults] 項目、

ScreenMode=NORMAL でメイン画面が真中に小さく表示されてるモード。　
ScreenMode=FULLで全画面のモード。
