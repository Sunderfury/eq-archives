〜 1st Impression 〜
Written Date: 2001/ 6/11
ついに始まったAnachy Online Beta 4。
発売日を考えれば、実質FinalBetaです。
最近EQに食傷気味で、他の何をやっても楽しい状態で(反面1週間持ちませんが;;´д｀)
とりあえずやってみることにしてみます。
あ、画像は大きいサイズのものにLinkしています。

○さいしょ
キャラクタ作成は、メニュー等ではなくてキャラを操作して装置を使うイメージで行われます。
詳しい説明は、もっと凄いところを見ていただくとして。

最初は通称NewbieZoneから開始になります。正式名称はよく判りません。
(キャラ選択ですとUnknownになります)
そうそう、私のキャラは全部Omniの管轄区域から始めています。
Clanの地域だと勝手が違うかもしれないので注意。

Logonしてみると・・・結構処理が重いかも・・・。特に、
Window → 地面 → キャラクタ → 建物の配置 → Texture
の順で描画されていくのですがこれが十数秒。これが重く感じさせる一因の気が。
それに、画面が表示される前にObjectの配置をした方が格好付くと思うのですが・・・。
まぁ、ここで言っても仕方ないし、たぶんもう手遅れ（笑

さて、これがGame中の画面(File Size : 89,676 Byte)。
中央に体育座りしているハゲが私のMainキャラ「Buchi」。
BreedがHomo-Nanoだと、まともな外見のものができないと
覚悟しておくべきです(^^;
後でまた出てきますが、この顔が無難だと思います。

Nanoゲージを見てもらえば判ると思いますが
ちょうど回復待ちをしているところです。

中央で何かしている肌色が悪いのは敵ではありません。
Homo-OpifexのFemaleです。何かCast中ですね。

ちなみに、Opifexの隣にある花は敵ではありません。
ちゃんとしたオブジェクトです(でも、敵だと思って最初はClickしてました（爆)。
他には、オブジェクト扱いの鳥とかもいるので紛らわしいです。


○たたかう
さて、戦闘中の画面(File Size : 92,270 Byte)。
画面したでバット(違う)を構えているのが2ndキャラ「Kiryu」。
Solitus FemaleのEnforcerです。
Enforcerの初期装備はバットメイスになります。

TargetしてQ KeyかCombatのボタンを押すことにより
武器のRange内ならば、あとは勝手に追尾して殴ります。
あ、Nano Tech.の初期装備は小銃なので「撃つ」ですね。

武器によっては、Special AttackのSkillが使えますので
それをやっても自動的にCombatModeになります。

このメイスは「Brawl」が使えるもので、当然Kiryuもそれを使って戦闘します。
ChatMes見てもらうと判りますが、このBrawlは普通の殴りより大きいDMGを叩き出します。
加えて初動無しです(普通にCombatModeにすると、殴り始めるまでに時間がかかります)。
これが使えるなら1st Attackに使ったほうが良いと思います。
特殊攻撃は「チャージタイム」が長いので、使いどころを良く考えたほうがいいみたいです。


○いどう
さて、移動中の画面(File Size : 129,819 Byte)。
これは、Rome-GreenとOmni-1 Entを繋ぐAreaです。
画像サイズが大きいのは、ここで1280x1024の
画面サイズに変更したからですね。

MAPが無いのは、インストール方法が
全然判らなかったからです（苦笑

ちなみに、NewbieAreaには番号が振ってあり
各都市毎に1から20以上のAreaに分かれています。
つまり、同じAreaNumberであろうとも、別な都市で
開始している可能性があるわけです。
LV-2にしないとそのAreaから脱出できないのですが
出てみないとどこの都市から開始なのかは判りません。


ちなみに、NanoTechを2つ作ったところ両方ともRomeにいました。
Enforcerは両方ともOmni-1でしたが、一方はTrader、もう一方はEntertainmentからの開始でした。


○ちーむ
さて、お次は・・・？(File Size : 145,277 Byte)。
AOのBeta4では、1つのアカウントで複数同時Logonが
できるようになっています(製品版では無理でしょう)。
これはMyキャラのうち3つを同時にLogonさせて
Teamを組ませたときの画面です。

右上のTeamと書かれたWindowがTeam Memberの
表示Windowです。
入れたいPlayerをTargetしてRECRUITボタンを押すと
Teamへの組み入れができます。
相手側でYes・Noが出て、Yesを押す必要がありますが。
LeaveでTeamから抜けます。Leaderが抜けるとどうなるか
は試していません。


MemberのTargetは、Team WindowのNameをクリックするか、F2〜F6を押すことによってできます。
F1はSelf、つまり自分です。

あ、そうそう。上のSnapは、左から順に
Kiryu(Enforcer)、Buchi(Nano-Technician)、Debu(Doctor)
です。

なおExpの分配ですが、LVが高い人に多く行くようになっているようです。
一人で倒したときとExpの量はあまり変わらないような気がしました。
つまり、大勢でタコ殴りにしたほうが効率良いということになります。
しかしその一方で、Missionで得ることができるExpはしっかり人数で除算されていました。
あと、強いキャラだけで敵を倒しても、Partyにさえ入っていればExpは入ります。
このとき、BuchiとKiryuはLV7でDebuが2だったのですが、問題無くExpを獲得できました。
LV差がどのくらいまでOKなのかは不明ですが、ちょっと離れているくらいなら
弱い敵をガンガン倒して底上げするという事ができそうです。
