開始：タイマーの開始
停止：タイマーの停止
リセット：タイマーを動作させたまま経過時間を 0 に戻します。
経過時間をセット：現在の「経過時間」を「セット時間」にセットします。

タイマーは割り当てられたファンクションキーを押す事によって開始されます。
Buff をかけたらファンクションキー、
更新したらファンクションキーというように使います。
つまり、ボタンを押す事によって、Start→リセット→リセット・・・・
と、動作します。

上部に配置してあるコンボボックスにより、複数の設定が保存可能です。
コンボボックスに適当な名前を入れ、保存ボタンを押す事により設定が保存されます。
なお、起動直後には「Default」の設定が読み込まれますので、
よく使う設定にしておくと便利です。

セット時間は、分単位で入力してください。
１時間３０分は「90」、４０分３０秒は「40.30」と入力できます。

- 動作環境 -
Windows95・98・2000
NT4 では正常動作しません。
VB6 Runtime が必要です。
