Problems Logging In?
 14:07 EST - jcsmith

If you are having problems logging in to Everquest this tidbit from the Developer's Corner may help...

** For those having problems logging in **

If you are having a problem that causes you to be kicked out of the game after selecting "Play on line", there is a solution to this problem.


Delete the file "bmpwad6.s3d" from your EverQuest directory.


This should solve the problem.

However, if you find that you are still having the problem and you have previously edited your eqclient.ini file in order to test out the DX8 DLL, then you should be able to fix the problem by:

Edit the line in your EQClient.ini in your EQ directory from "GraphicsDLL=EQGfx_Dx8.DLL" to "GraphicsDLL=EQGfx_Dx7.DLL".

We apologize for any trouble this may have caused some of our customers.

The Dark Age of Online Gaming?
 13:37 EST - jcsmith

Is this the Dark Age of Online Gaming? A recent article at Gamer's Voice asks that question. Here's a text bite...

The reality is that five companies are putting out almost the exact same game with only minor differences in packaging and game play -- and making millions of dollars doing it. How is this, then, a renaissance in gaming? It is more like a dark age -- creativity and innovation are replaced by poor customer service and the recycling of the same basic ideas over and over again. The old triumvirate of power HAS been destroyed, but instead of increased variety, we're seeing a new trend of "jump from one game to the newest release, stay for a few months, then repeat the cycle when another MMORPG hits the market." With so few unique features in each game, there is no real reason for most players to stay. Sadly, Wolfpack Studio's soon to be released Shadowbane and Blizzard's World of Warcraft, while both promising new innovations, appear to be on the same track -- same basic game play, similar leveling systems, and just generally the same game we have seen before -- with a few new advances. While, as a player, I hope someone does come up with an interesting NEW MMORPG that doesn't borrow so heavily from the old ones, I'm not holding my breath.

DirectX8 Update
 13:31 EST - jcsmith

This comes from the EQ Developer's Corner...

The DX8 DLL is on the patcher now. One thing not mentioned above is that you may set VideoModeBitsPerPixel to 32 instead of 16 in your EQClient.ini if your card supports it. If you have any problems with this configuration you should set it back to 16.

The difference between the two settings is one of aesthetic quality versus memory usage on the card. Some people using 16 meg video cards may have texture memory issues and may prefer to go with 16 bit color. Those with 32 meg cards will probably prefer the 32 bit setting.
