EQ Headers Parser Added
Posted by Xylor on Monday April 16, @02:42AM
from the eq-protocol-development dept.
A new page has been added to the web site. It allows you to upload tcpdump files, which will be parsed, stripped, and displayed in a colorized html table. This is mostly for use in the development of OpenQuest, but you are welcome to add your own tcpdump files and see how the EQ protocol works. This new page is available at http://openquest.dhs.org/EQHeaders
