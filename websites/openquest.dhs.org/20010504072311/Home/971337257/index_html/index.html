Welcome to the OpenQuest Home Page
Posted by Xylor on Thursday October 12, @02:54AM
from the welcome home dept.

This page is still under construction. Most of my free time is being spent working on OpenQuest itself, so I don't have much time to work on a web page yet.

Many thanks to casey, syn and everyone at trifocus.net for hosting this web site, and to Aaearon for help on the web site. There is also a CVS repository on trifocus. export CVSROOT=:pserver:anonymous@openquest.dhs.org:/cvs. Then run cvs login (enter for password), and cvs -z3 co openquest.

Please feel free to check out the API documentation, or download the current in-development tarball version of OpenQuest. If you have any questions, or would like to help on this project, please contact me at xylor@openquest.dhs.org.
