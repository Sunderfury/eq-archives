To simulate the Patch Server is not really a great job. 

1. You need only to open the exe file with an HEX-Editor and seek for "patch.everquest.com" than replace it with your own IP-Adress, normally 127.0.0.1 the rest of the remaining bytes you fill up with "00".

2. Than you seek in the exe file for "581B" (The Port) its in decimal 7000 to hex converted it is 1B58 than you have to reverse the HEX value and get the "581B". If you found it than change it to 571B (this would be 6999). 

3. Now you need a programm, like mine that listens for a TCP/IP connection at 127.0.0.1:6999. Than start the modified patch programm and it would connect to you server and send a command.

4. Now i write my programm so that it connects now to the real patch server and forwards the command to it. If the patch server answers to my programm it will forward it to the EQ-Patch programm and bum you can record the packets and the patch server works.

5. After you know the answers from the real server you can easily write your own patch server!
