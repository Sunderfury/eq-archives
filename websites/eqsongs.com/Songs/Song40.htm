SYVELIAN's ANTI-MAGIC ARIA

Level: 40

Effect: Cancel magic

Skill: Singing

Target: Single

Location: Kelethin

Does not stack with: N/A

Taunt: ?

Quantified: Removes one buff per pulse. May now have a stronger effect (per 2/3 patch), testing has now shown any extra effectiveness.

Analysis by Rokenn Swiftsong, Karana server

Comments: This is the same as the Cancel Magic spell.

Strategies

Great for taking down those annoying damage shields.
Since this is a straight debuff you can use it on mesmerized NPCs without waking them up.
[Sartori] When dueling any DoT-using class ALWAYS have this song memmed! Just don't tell them about it. =) It will strip the DoT causing minimal damage if you are quick enough (and if it doesn't repeatedly fail).
[Aaeandar Vie] I've found that this song comes in handy when you are facing big mobs, like dragons and such, where you WILL be dispelled, and you need to have a bunch of garbage buffs set up first. IF you got good buffs on, you can dispel yourself to get the bad ones on first.  I had to do it, and you should consider it an option. I usually use this song on myself (losing crap buffs, misplaced buffs, dots).

Cautions:

[Elwyn] This song will not effect player characters unless you are in a PvP zone or a duel.
[Patch Message] NPCs will now treat the "Cancel Magic" series of spells as hostile. Previously, NPCs did not mind if players dispelled all of their buffs before attacking. Now they will.
