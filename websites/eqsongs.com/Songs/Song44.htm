CASSINDRA's ELEGY

Level: 44

Effect: Increase wisdom and intelligence.

Skill: Singing

Target: Party

Location: Erudin

Does not stack with: ?

Taunt: Nil

Quantified: ?

Analysis by Rokenn Swiftsong, Karana server

Comments: This song grants a temporary increase to INT & WIS. 

Strategies:

This can be handy before a large battle, since it will give the casters in the party a temporary boast to their mana reserves.  You will need to keep this song going till they have burned the extra mana as it will disappear if song stops.
Can be useful will doing trade skills; INT helps with skill increases, WIS helps recover from failures.
