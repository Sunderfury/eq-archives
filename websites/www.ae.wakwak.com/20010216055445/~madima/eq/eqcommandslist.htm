新着コマンド
/lootnodrop - NODROPアイテムを間違ってlootしてしまう問題を回避するために、/lootnodrop
コマンドが追加されました。これには３つのオプションがあります。
　●/lootnodrop ALWAYS(DEFAULT) - NODROPアイテムをlootする時に、yes/noの確認が行われます。
　●/lootnodrop SOMETIMES - そのNODROPアイテムが、あなたのrace/classで使用できない物だった場合に、確認のboxが現れます。
　●/lootnodrop NEVER - 確認なしに全てのNODROPアイテムをlootします。

/wincolor（SoVレジスト以降使用可能になります）　フルスクリーンのウィンドウ変更コマンド
そのまま/wincolorと打つと
Format: /wincolor <window name> <red> <green> <blue> <trans>
<window name> -> CHAT, SPELL, PLAYER, PARTY, TARGET, BUFF, DATA, MAIN, INV, or HOTBOX
<red> <green> <blue> -> 0 to 255
<trans> -> 0 = off, 1 = 25, 2 = 50, 3 = 75, 4 = 100 percent
To reset a window color use /wincolor <window name> reset
と出ます。
<window name>でウィンドウの指定
<red> <green> <blue>でウィンドウの色を指定（いらないっぽいですが）
<trans> でウィンドウの透過光率変更
リセットするときは/wincolor <window name> とウィンドウだけ指定すればリセットされます。

　%- (percent sign) ターゲット対象（つまりClickして名前が出てる状態の相手）をエモートやチャットチャンネル(/Say, /Shout, /OOC, /Auction, /Group,/Guild)に使用可能。
現在はすべての対象で可能ではないようです。また対象によっては使えないものもあります。

  • %T - ターゲット対象名を表示
    例, "/emote kicks %" …がSakuranomiko kicks Savchenko.
  • %S -ターゲットの主観的な代名詞に変換します。 (He, She, It)
    例, "/say %S kicks me!!" がSavchenko says, "She kicks me!!"
  • %O -ターゲットの目的格代名詞に変換します。(Him, Her, It)
    例, "/shout Ahhh！％Ｔ！, I resurrected %O!" がYoisho shouts, "Ahhh！Sakuranomiko !, I resurrect her!"
  • %P -ターゲットの 所有格代名詞に変換します (His, Her, Its)
    例, "/emote pokes %P head" returns Sakuranomiko pokes his head.
  • %R -ターゲットのRace（種族名）に変換---NPC不可
    例, "/emote thinks that %R is happy." returns Sakuranomiko thinks that Human is happy.
  • %C -ターゲットのクラス名に変換---ＮＰＣ不可
    例, "/ooc I love %C!" ----Sukuranomiko says out of character, "I love Monk!"

Format: /wincolor <window name> <red> <green> <blue> <trans>
<window name> -> CHAT, SPELL, PLAYER, PARTY, TARGET, BUFF, DATA, MAIN, INV, or HOTBOX
<red> <green> <blue> -> 0 to 255
<trans> -> 0 = off, 1 = 25, 2 = 50, 3 = 75, 4 = 100 percent

<window name>でウィンドウの指定

<trans>でウィンドウの透過光率変更
