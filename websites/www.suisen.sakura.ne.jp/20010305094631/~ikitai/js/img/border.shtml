電　飾
IE4以上　NN6

説明とサンプルはソースの下です。

以下のソースをHEAD内の入れる。

太文字の所を設定する。
水色が色の変わる順番とカラーコード。数は何個でもOKです。
緑色が電飾する枠の太さです。
黄色が電飾が終わったときの色です。通常は背景と同じにしてください。

<SCRIPT language="JavaScript">
<!--
// 枠の色を変えたい順に色を設定する。何個でも良いです。
var clr = new Array("#FFFFFF","#FFF000","#FF00FF","#00FF00","#0000FF","#FF0000","#FFFFFF",
                    "#FFF000","#FFA000","#FF00FF","#00FF00","#FFA000","#0000FF","#FF0000");

var i = 0;
function borderOn(name) {
        obj = "";
        if (document.all) obj = document.all(name);
        else if (document.getElementById) obj = document.getElementById(name);
        if (obj) {
                obj.style.border = "5px solid " + clr[i];
                i++;
                if (i >= clr.length) i = 0;
                tim = setTimeout("borderOn('"+name+"')",100);
        }
}

function borderOff(name,back) {
        obj = "";
        if (document.all) obj = document.all(name);
        else if (document.getElementById) obj = document.getElementById(name);
        if (obj) {
                clearTimeout(tim);
                obj.style.border = "5px solid black";
                if (back) obj.style.borderStyle = "none";
        }
}

// -->
</SCRIPT>

-->
</STYLE>

以下はサンプルとタグの書き方。

共通。

壁紙がない場合は class="〜" を挿入。〜にはwaku または waku2 が入る。
id は他のと重ならなければ何でも良いです。
onmouseover="borderOn('img1')" onmouseout="borderOff('img1')" 太文字の部分が id名です。
例えば下のサンプルですと２つ目の画像の( )内に入れるid名を img2 から img1 にするとimgが電飾します。
要するに括弧の中には電飾したい所のid名を入れればいいのです。

Backに壁紙が無い場合

<IMG src="uwo19.jpg" width="200" height="150" class="waku" id="img1" onmouseover="borderOn('img1')" onmouseout="borderOff('img1')">
<IMG src="ao07.jpg" width="200" height="150" class="waku" id="img2" onmouseover="borderOn('img2')" onmouseout="borderOff('img2')">

壁紙がある場合

次のようにテーブルに入れてTDの幅と高さをイメージの大きさ+枠にしてください。サンプルの場合は枠が5pxなので+10してます。
onmouseoutの括弧の中はborderOff('img3',1)のように ,1 が追加します。(壁紙がない場合は ,1 は付かない。)


<TABLE border="0" cellspacing="0" cellpadding="0">
　<TR>
　　<TD width="210" height="160" align="center">
　　　<IMG src="uwo19.jpg" width="200" height="150" id="img3" onmouseover="borderOn('img3')" onmouseout="borderOff('img3',1)">
　　</TD>
　　<TD width="210" height="160" align="center">
　　　<IMG src="ao07.jpg" width="200" height="150" id="img4" onmouseover="borderOn('img4')" onmouseout="borderOff('img4',1)">
　　</TD>
　</TR>
</TABLE>

リンクで他を電飾する場合

( )の中に電飾したい要素のID名を入れてやればよい。
(IMG3と4は壁紙有りなのでonmouseoutの方に ,1 が入ってます）

<A href="#" onclick="return false" onmouseover="borderOn('img3')" onmouseout="borderOff('img3',1)">IMG3</A>
<A href="#" onclick="return false" onmouseover="borderOn('img4')" onmouseout="borderOff('img4',1)">IMG4</A>

リンクを電飾する場合

壁紙がある場合に使うと電飾した部分の背景が消えます。
どうしても消したくない場合はテーブルなどに入れて枠の分の余白を充分取り class="waku2" を削除。画像の場合同様onmouseoutの（ ）の中に ,1 を追加します。

<A href="#" class="waku2" id="lnk1" onclick="return false" onmouseover="borderOn('lnk1')" onmouseout="borderOff('lnk1')">りんく１</A>

サンプル 　　りんく１ りんく２

普通の文字の所を電飾

リンクと同じく壁紙がある場合に使うと電飾した部分の背景が消えます。
消したくない場合の対処はリンクに同じ。
また下のサンプルのように枠の分余白が出来ます。

ここが<SPAN class="waku2" id="moji1" onmouseover="borderOn('moji1')" onmouseout="borderOff('moji1')">電 飾</SPAN>するぞ
