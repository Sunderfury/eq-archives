雪降りTIPS

「雪を降らせる」と各改造版のTIPSです。
画像の枚数、縦長のPAGEへの合わせ方、Speedの調整の仕方、揺れの大きさの調整の仕方です。

各TIPS共通の説明
同じ行が何カ所かあるのは１度画面から消えたのを次に現れるときに以前と違う動きにするためです。その為、数カ所ある場合は全て同じ変更をしてください。

動く画像の数

ソースの上の方の var no = 15; // snow number の数字を変えれば良い。
但し余り数を多くしますと動きが遅くなります。遅くなる度合いはパソコンの性能に左右されます。
多くても２０ぐらいに納めた方が良いと思います。

var no = 5; にした場合のサンプル


横長・縦長のPAGEへの合わせ方1

１．
ソースの var i, doc_width = 800, doc_height = 1800; をPAGEの長さに合わせて設定してください。widthが横の長さ、heightが縦の長さです。変更するのは合わせたい側のみでも構いません。

２．
doc_width = self.innerWidth;
doc_height = self.innerHeight;
doc_width = document.body.clientWidth;
doc_height = document.body.clientHeight;
の行が各2箇所あります。合わせたい方向の先頭に // を入れてください(2箇所全てに)
例えば縦長のPAGEに合わせるなら doc_height の行の先頭に // を入れてやる。
例）
doc_width = self.innerWidth;
//doc_height = self.innerHeight;
doc_width = document.body.clientWidth;
//doc_height = document.body.clientHeight;

横長・縦長のPAGEへの合わせ方2

こちらのやり方はスクロールに合わせて画像も一緒にスクロールするやり方です。
上の１番とは一緒に使わないでください。

１．
document.layers["dot"+i].top = ??;
document.layers["dot"+i].left = ??;
の行を探して次のようにしてください。(??はScriptによって入る文字が違ってるためです)
document.layers["dot"+i].top = ?? + pageYOffset;
document.layers["dot"+i].left = ?? + pageXOffset;

２．
document.all["dot"+i].style.pixelTop = ??;
document.all["dot"+i].style.pixelLeft = ??;
の行を探して次のようにしてください。(??はScriptによって入る文字が違ってるためです)
document.all["dot"+i].style.pixelTop = ?? + document.body.scrollTop;
document.all["dot"+i].style.pixelLeft = ?? + document.body.scrollLeft;

Speedの調整の仕方

sty[i] = 〜〜〜〜〜; の行が３箇所あります。
Speedはこの行の〜〜〜に入ってる計算式で変わります。
例えば計算式が 0.7 + Math.random() なら Math.random()は0-1の間の数をランダムで返すのでSpeedが0.7-0.8の間でランダムで決まります。
式が 5 + Math.random()*3 なら 5-8の間でランダムに決まります。

3 + Math.random()*2 (speedが3-5)のサンプル


揺れの調整の仕方

am[i] = Math.random()*20 が１箇所あります。最後の数字が揺れる幅です。
20なら0-20ピクセルの間でランダムに揺れ幅が変わります。
am[i] = 0; にすれば揺れなくなります。

揺れの周期の調整は
stx[i] = 0.02 + Math.random()/10; の行が３箇所ありその計算式で変わります。
数字が小さいほど揺れの周期が大きくなります。この辺はSpeedによっても変わるので動きを見ながら適当に調整してください。

am[i] = 50+Math.random()*50; , stx[i] = 0.02 + Math.random()/50; のサンプル


降る位置(幅)を指定する

xp[i] = 〜〜 の行が３箇所あります。
この右辺の式で変わります。

中央６００の幅に降らす場合は３箇所全部以下のようにしてください。
xp[i] = (doc_width-600) / 2 + Math.random() * 600;

左端から６００なら
xp[i] = Math.random() * 600;

右端６００なら
xp[i] = (doc_width-600) + Math.random() * 600;

注) テーブルの中だけとかは出来ません。

HTMLメールで使いたい1

説明の前にまず注意点。
送る相手の使ってるメーラーを確認してください。Outlook ExpressならOKでしょう。
他のメーラーの場合はHTMLメール表示にIEの機能を使ってるなら動くと思いますが余り送らない方が良いでしょう。
HTMLメールを送られるとPCがハングアップする人もいると聞いたことがありますので使用には充分気を付けて使ってください。
ちなみに私のデフォで使ってるメーラーはHTMLメールは表示しない設定にしてますので送らないで下さい。
１．まずスクリプトより上に以下を必要な数だけ書いてください。
　　　　(下のソースは改行してますが１行で書いても構いません）

<DIV id="dot0" style="POSITION: absolute; TOP: -15px; LEFT: -15px; width:1;">
<IMG src="snow.gif" border="0"></DIV>

id="dot0" の数字0は0から始まる連番でvar no = 15;に設定されてる数だけ用意します。
デフォなら15なので上のソースを番号を変えて0から14まで書きます。（0から始まるので15なら14まで）
画像 snow.gif はお好みに合わせて変えてください。

２．スクリプトの以下の行を全て削除してください。

　　と

　　から
function snowNS()の２行上の } まで
（sty[i] = 0.7 + Math.random();の次の行に ｝ が１つ残る）　

これで完成です。

HTMLメールで使いたい2

こちらは殆どのJavaScriptで使えるやり方です。

１．HEAD内へ入れるとなってるSCRIPTも全てBODY内へ入れる。

２．SCRIPTより上に使いたい画像を各１枚ずつ入れてやります。
　　<IMG src="画像のパス" width="幅" height="高さ" border="0">
　　次に挿入した画像のタグに ID="〜〜" style="display:none"　を追加する。
　　<IMG src="画像のパス" width="幅" height="高さ" border="0" ID="〜〜" style="display:none">
　　IDは画像事に違う名前を任意に付ける。　　例）　img0、img1、img2・・・

３．JavaScript内のイメージを指定する箇所を document.all("ID名").src に変更する。
　　雪を降らせるを例にすると
　　２で挿入した画像が
　　<IMG src="snow.gif" width="10" height="10" border="0" ID="img0" style="display:none">
　　なら
　　var snowflake = "snow.gif";
　　は
　　var snowflake = document.all("img0").src;
　　になる。
