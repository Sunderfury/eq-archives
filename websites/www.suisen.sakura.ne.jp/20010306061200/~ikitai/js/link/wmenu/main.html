リモートリンク

まずフレームセットを作ります。
ファイル名は何でも良いです。ここでは frame.html としておきます。
最終的にここで作成するPAGEにアクセスすることになりますのでそれを踏まえてファイル名を付けてください。
リモートするページをmain.htmlとしますと以下のようにします。
nameの値は何でも良いです。あなたが分かりやすいように付けてください。

frame.htmlのソース

<HTML>
<HEAD>
<TITLE>リモートリンク</TITLE>
</HEAD>
<FRAMESET ROWS="100%,*" border=0 framespacing=0>
        <FRAME SRC="main.html" name="main">
</FRAMESET>
</HTML>

メニューのHTMLを作ります。
これもファイル名は何でも良いです。ここではmenu.html としておきます。

menu.htmlのソース

<HTML>
<HEAD>
<META http-equiv="Content-Type" content="text/html; charset=Shift_JIS">
<TITLE>menu</TITLE>
</HEAD>
<BODY bgColor="#FFFFFF">
<BR><BR>
<A href="main.html" target="main">戻る</A><BR>
<BR>
<A href="http://www.yahoo.co.jp/" target="main">Yahoo!Japan</A><BR>
<BR>
<A href="http://www.goo.ne.jp/" target="main">Goo</A><BR>
<BR>
<A href="http://www.excite.co.jp/" target="main">Excite Japan</A><BR>
<BR>
<A href="http://www.infoseek.co.jp/" target="main">Infoseek</A><BR>
<BR>
<A href="http://www.joyjoy.com/" target="main">J.O.Y</A><BR>
<BR>
<A href="http://www.fresheye.com/" target="main">フレッシュアイ</A><BR>
</BODY>
</HTML>

上記のように普通のリンクに target属性でfreme.htmlで指定したnameを指定します。

リモート操作するPAGEにメニューを開くボタンを付ける。
ここですとmain.htmlに書きます。

<FORM>
<INPUT TYPE="button" VALUE="Open Menu" onClick='window.open("menu.html","menu_window","width=200,height=400")'>
</FORM>

これで完成です。
アクセスは最初に作成したファイルにアクセスします。
ここですと frame.html に。
直接 リモートするHTMLにアクセスしますと操作が出来ませんのでご注意。
