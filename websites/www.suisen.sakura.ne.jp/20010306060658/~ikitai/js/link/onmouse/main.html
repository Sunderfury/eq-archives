オンマウス
クリックした後、画像を変えたままにしておく

○が<IMG>要素のname属性。△がイメージ番号。☆がonMouseOutの△と同じ番号

onMouseOver="chgimg('○',△);" マウスが乗ったとき
onMouseOut="chgimg('○',△);" 　マウスが出たとき
onClick="showCnt('○',☆);" 　クリックしたとき

私のサイトではこのScriptを応用した物を使ってます。

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN">
<HTML>
<HEAD>
<META http-equiv="Content-Type" content="text/html; charset=Shift_JIS">

<SCRIPT language="JavaScript">

function chgimg(name,num){
  if (name != showName) {
    document.images[name].src = img[num].src;
  }
}

function showCnt(name,num){
  if (name != showName) {
    document.images[showName].src = img[showNo].src;
    showName = name;
    showNo = num;
  }
}
