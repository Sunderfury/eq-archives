雪降りTIPS

「雪を降らせる」と各改造版のTIPSです。
画像の枚数、縦長のPAGEへの合わせ方、Speedの調整の仕方、揺れの大きさの調整の仕方です。

各TIPS共通の説明
同じ行が何カ所かあるのは１度画面から消えたのを次に現れるときに以前と違う動きにするためです。その為、数カ所ある場合は全て同じ変更をしてください。

動く画像の数

ソースの上の方の var no = 15; の数字を変えれば良い。
但し余り数を多くしますと動きが遅くなります。遅くなる度合いはパソコンの性能に左右されます。
多くても２０ぐらいに納めた方が良いと思います。

横長・縦長のPAGEへの合わせ方1

function getDocumentSize() {
  if(_dom == 1 || _dom == 2) {
    doc_width = self.innerWidth;
    doc_height = self.innerHeight;
  }
  if (_dom == 3) {
    doc_width = document.body.clientWidth;
    doc_height = document.body.clientHeight;
  }
}

のdoc_width または doc_height の右辺へPAGEの長さを直接入れてやる。
doc_width が幅、doc_height が高さです。

横長・縦長のPAGEへの合わせ方2

こちらのやり方はスクロールに合わせて画像も一緒にスクロールするやり方です。
上の１番とは一緒に使わないでください。

function snowMove(lay,t,l) {
  if (_dom == 1) {
    document.getElementById(lay).style.top = t;
    document.getElementById(lay).style.left = l;
  }
  if (_dom == 2) {
    document.layers[lay].top = t;
    document.layers[lay].left = l;
  }
  if (_dom == 3) {
    document.all[lay].style.pixelTop = t;
    document.all[lay].style.pixelLeft = l;
  }
}

を
function snowMove(lay,t,l) {
  if (_dom == 1) {
    document.getElementById(lay).style.top = t + scrollY;
    document.getElementById(lay).style.left = l + scrollX;
  }
  if (_dom == 2) {
    document.layers[lay].top = t + pageYOffset;
    document.layers[lay].left = l + pageXOffset;
  }
  if (_dom == 3) {
    document.all[lay].style.pixelTop = t + document.body.scrollTop;
    document.all[lay].style.pixelLeft = l + document.body.scrollLeft;
  }

Speedの調整の仕方

sty[i] = 〜〜〜〜〜; の行が２箇所あります。
Speedはこの行の右辺に入ってる計算式で変わります。
例えば計算式が 0.7 + Math.random() なら Math.random()は0-1の間の数をランダムで返すのでSpeedが0.7-1.7の間でランダムで決まります。
式が 5 + Math.random()*3 なら 5-8の間でランダムに決まります。

3 + Math.random()*2 (speedが3-5)のサンプル


揺れの調整の仕方

am[i] = Math.random()*20 が１箇所あります。最後の数字が揺れる幅です。
20なら0-20ピクセルの間でランダムに揺れ幅が変わります。
am[i] = 0; にすれば揺れなくなります。

揺れの周期の調整は
stx[i] = 0.02 + Math.random()/10; の行が２箇所ありその計算式で変わります。
数字が小さいほど揺れの周期が大きくなります。この辺はSpeedによっても変わるので動きを見ながら適当に調整してください。

am[i] = 50+Math.random()*50; , stx[i] = 0.02 + Math.random()/50; のサンプル


降る位置(幅)を指定する

xp[i] = 〜〜 の行が２箇所あります。
この右辺の式で変わります。

中央６００の幅に降らす場合は２箇所全部以下のようにしてください。
xp[i] = (doc_width-600) / 2 + Math.random() * 600;

左端から６００なら
xp[i] = Math.random() * 600;

右端６００なら
xp[i] = (doc_width-600) + Math.random() * 600;

注) テーブルの中だけとかは出来ません。
